# 基本時制 - 説明

## 現在形

今の習慣や事実を表す。

- I **play** games.（私はゲームをします）
- 一般動詞はそのまま原形を使う
- be動詞: I **am** / You **are** / He **is**

## 過去形

過去の出来事を表す。

- I **played** games.（私はゲームをしました）
- 規則動詞: 語尾に **-ed** をつける（play → played）
- 不規則動詞: 形が変わる（go → went, eat → ate）
- be動詞: I **was** / You **were**

## 未来形

これからの予定や予測を表す。

- I **will play** games.（私はゲームをするでしょう）
- **will + 動詞の原形**
- be動詞: I **will be**

## 不規則動詞の変化表

| 原形 | 過去形 | 意味 |
|------|--------|------|
| go | went | 行く |
| eat | ate | 食べる |
| write | wrote | 書く |
| run | ran | 走る |
| have | had | 持つ |
| read | read | 読む |
| see | saw | 見る |
| take | took | 取る |
