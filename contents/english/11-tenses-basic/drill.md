# 基本時制ドリル

## 現在形・過去形・未来形を使い分けましょう

下の文を日本語に訳しましょう。

| 英文 | 日本語訳 |
|------|----------|
| I play games. | 　　　　　　　　　　　　 |
| You play games. | 　　　　　　　　　　　　 |
| He plays games. | 　　　　　　　　　　　　 |
| She plays games. | 　　　　　　　　　　　　 |
| We play games. | 　　　　　　　　　　　　 |
| They play games. | 　　　　　　　　　　　　 |
| I played games. | 　　　　　　　　　　　　 |
| You played games. | 　　　　　　　　　　　　 |
| He played games. | 　　　　　　　　　　　　 |
| We played games. | 　　　　　　　　　　　　 |
| I will play games. | 　　　　　　　　　　　　 |
| You will play games. | 　　　　　　　　　　　　 |
| He will play games. | 　　　　　　　　　　　　 |
| They will play games. | 　　　　　　　　　　　　 |
