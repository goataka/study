# 条件文 - 解説

現在形:
I `have` time. I `play` games.（時間がある。ゲームをする。）

第1条件文:
If I `have` time, I `will play` games.（時間があれば、する）

- 現在形の事実を「もし～なら」の条件にすると条件文になる

## 条件文の3パターン

| パターン | if節 | 帰結節 | 意味 |
|----------|------|--------|------|
| 第1条件文 | if + 現在形 | will + 原形 | 現実的な条件 |
| 第2条件文 | if + 過去形 | would + 原形 | 現在の事実に反する仮定 |
| 第3条件文 | if + had + 過去分詞 | would have + 過去分詞 | 過去の事実に反する仮定 |

## 例文で比較

第1条件文:
If I have time, I `will` play.（時間があれば、する — ありえる）

第2条件文:
If I had time, I `would` play.（時間があれば、するのに — 今はない）

第3条件文:
If I had had time, I `would have` played.（時間があったら、したのに — 過去になかった）
