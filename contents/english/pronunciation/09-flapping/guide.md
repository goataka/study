# フラッピング（音の変化） - 解説

## フラッピングとは

- `t` または `d` が「母音 + t/d + 母音（または r）」のパターンに挟まれると、ラ行の音になる
- アメリカ英語で特によく起きる変化

## フラッピングが起きる条件

```text
母音 + [t / d] + 母音（強勢のない）
     ↓
  ラ行の音
```

## 例題

👅 `water` → ワ **ラ** ー

```text
wa - t - er
     ↓
    ラ
→ ワラー
```

👅 `better` → ベ **ラ** ー（tt もフラッピングする）

👅 `little` → リ **ロ** ー（tt → ロ）

## フラッピングの例一覧

| 単語/フレーズ | 変化 | 発音 |
|-------------|------|------|
| `water` | t → ラ | ワ ラー |
| `better` | tt → ラ | ベ ラー |
| `city` | t → リ | シ リー |
| `party` | t → リ | パー リー |
| `get it` | t → リ | ゲ リッ ト |
| `what about` | t → ラ | ワ ラ バウ ト |

## ポイント

- 💡 `t` の前の母音に強勢（アクセント）があるときに起きやすい
- 💡 `butter`・`better`・`bitter` はすべてラ行に変化するので聞き分けが難しい
- 💡 日本語の「ら行」に近いが、舌先の当て方が少し異なる

## 解答

t/dがラ行になる部分の音をカタカナで書きましょう。

| フレーズ | 音 | 変化した部分 |
|----------|------|--------------|
| water | ワ ラー | t → ラ |
| better | ベ ラー | tt → ラ |
| little | リ ロー | tt → ロ |
| get it | ゲ リッ ト | t → リ |
| what about | ワ ラ バウ ト | t → ラ |
| put it | プ リッ ト | t → リ |
| city | シ リー | t → リ |
| party | パー リー | t → リ |
