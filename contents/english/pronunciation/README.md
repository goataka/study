# 発音学習ガイド

英語の発音を基礎から学習します。「読み書きの基礎」→「フォニックス」→「音声変化」の順に進みます。

## 学習の流れ

### 1. 読み書きの基礎

- [アルファベット](01-alphabet/guide.md) — 大文字・小文字と読み仮名
- [50音](02-50on/guide.md) — ひらがなとローマ字の対応

### 2. フォニックス（音と文字の関係）

- [フォニックス（1文字）](03-phonics-1letter/guide.md) — 1文字の基本的な音
- [フォニックス（2文字・マジックE）](04-phonics-2letter/guide.md) — 2文字の組み合わせの音
- [フォニックス（3文字）](05-phonics-3letter/guide.md) — 3文字の組み合わせの音

### 3. 音声変化（リスニング・スピーキング）

- [リンキング](07-linking/guide.md) — 音がつながる現象
- [リダクション](08-reduction/guide.md) — 音が弱くなる・消える現象
- [フラッピング](09-flapping/guide.md) — t/d がラ行の音に変わる現象
- [アシミレーション](10-assimilation/guide.md) — 隣り合う音が影響し合う現象
