# アシミレーション（音の同化） - 解説

## アシミレーションとは

- となりの音に引っ張られて、音が変化する現象
- 会話を速く・自然に話すために起きる

## t / d + you のパターン

語末の `t` または `d` に `you` が続くと音が変わる。

| 変化ルール | 例 | 発音 |
|-----------|-----|------|
| `t` + `y` → `チュ` | `meet you` | ミー **チュ** ー |
| `t` + `y` → `チュ` | `can't you` | キャン **チュ** ー |
| `d` + `y` → `ジュ` | `would you` | ウッ **ジュ** ー |
| `d` + `y` → `ジュ` | `could you` | クッ **ジュ** ー |

## s + you のパターン

| 変化ルール | 例 | 発音 |
|-----------|-----|------|
| `s` + `y` → `シュ` | `miss you` | ミッ **シュ** ー |
| `s` + `y` → `シュ` | `bless you` | ブレッ **シュ** ー |

## n → m のパターン（唇音の前）

`p`・`b`・`m` など唇を使う音（唇音）の前では `n` が `m` に変化する。

| 例 | 変化 | 発音 |
|----|------|------|
| `ten people` | n → m | テム ピー プル |
| `in Paris` | n → m | イム パ リ |

## 例題

🔄 `meet you` → ミーチュー

```text
meet  you
    ↘↗
   チュ
→ ミーチュー
```

## ポイント

- 💡 `t/d/s` + `y` の組み合わせが最もよく出る
- 💡 `Would you like...?`・`Could you please...?` は日常会話で頻出
- 💡 `Bless you`（くしゃみの後）も `ブレッシュー` と聞こえる
