# フォニックス練習ドリル（2文字） - 解答

## 2文字の音を書きましょう

下線部の音をカタカナで書きましょう。

### 2文字の組み合わせ

| 単語 | 単語全体 | 意味 |
|------|----------|------|
| `sh`oe | `シュ` ウー → シュウー | くつ |
| fi`sh` | フ イ `シュ` → フイシュ | さかな |
| `ch`air | `チ` エ アー → チエアー | いす |
| mu`ch` | ム ア `チ` → ムアチ | たくさん |
| `th`at | `ザ` ア ト → ザアト | あれ・それ |
| ba`th` | ブ ア `ス` → ブアス | おふろ |
| `th`ank | `ス` ア ン ク → スアンク | ありがとう |
| `ph`one | `フ` オウ ヌ → フオウヌ | でんわ |
| `wh`ale | `ホウ` エイ ル → ホウエイル | くじら |
| g`oo`d | グ `ウ` ドゥ → グウドゥ | よい |
| b`oo`k | ブ `ウ` ク → ブウク | ほん |
| m`oo`n | ム `ウー` ヌ → ムウーヌ | つき |
| d`ee`p | ドゥ `イー` プ → ドゥイープ | ふかい |
| gr`ee`n | グ ル `イー` ヌ → グルイーヌ | みどり |
| r`ai`n | ル `エイ` ヌ → ルエイヌ | あめ（雨） |
| tr`ai`n | トゥ ル `エイ` ヌ → トゥルエイヌ | でんしゃ |
| b`oa`t | ブ `オウ` ト → ブオウト | ふね |
| c`oa`t | ク `オウ` ト → クオウト | コート |
