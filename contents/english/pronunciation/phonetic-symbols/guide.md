# 発音記号（IPA）

## 発音記号とは

- 辞書に載っている `/æ/` や `/iː/` などの記号のこと
- 世界共通の「音の表記法」（国際音声記号 IPA）
- カタカナでは表せない微妙な音の違いを表せる

## 母音（ぼいん）

### 主な母音

| 発音記号 | 音のイメージ | 例 |
|----------|-------------|-----|
| `/æ/` | 「ア」と「エ」の間 | `cat`（キャット） |
| `/ʌ/` | 口を軽く開けた「ア」 | `cup`（カップ） |
| `/ɑː/` | 口を大きく開けた「アー」 | `car`（カー） |
| `/ɒ/` | 丸く口を開けた「オ」 | `hot`（ホット） |
| `/ɔː/` | 口を丸めた「オー」 | `law`（ロー） |
| `/ə/` | 力を抜いた「ア」（シュワー） | `about`（アバウト） |
| `/ɜː/` | 「アー」と「エー」の間 | `bird`（バード） |
| `/ɪ/` | 短くゆるい「イ」 | `hit`（ヒット） |
| `/iː/` | 長くはっきりした「イー」 | `see`（スィー） |
| `/ʊ/` | 短くゆるい「ウ」 | `book`（ブック） |
| `/uː/` | 長くはっきりした「ウー」 | `too`（トゥー） |
| `/e/` | 日本語の「エ」に近い | `bed`（ベッド） |

### 二重母音（ダイフトング）

| 発音記号 | 音のイメージ | 例 |
|----------|-------------|-----|
| `/eɪ/` | 「エイ」 | `day`（デイ） |
| `/aɪ/` | 「アイ」 | `fly`（フライ） |
| `/ɔɪ/` | 「オイ」 | `boy`（ボイ） |
| `/aʊ/` | 「アウ」 | `now`（ナウ） |
| `/əʊ/` | 「オウ」 | `go`（ゴウ） |
| `/ɪə/` | 「イア」 | `ear`（イア） |
| `/eə/` | 「エア」 | `air`（エア） |

## 子音（しいん）のポイント

日本語にない子音を中心に覚えよう。

| 発音記号 | 音のイメージ | 例 |
|----------|-------------|-----|
| `/θ/` | 舌先を軽く噛んだ「ス」 | `think`（シンク） |
| `/ð/` | 舌先を軽く噛んだ濁った音 | `this`（ジス） |
| `/ʃ/` | 「シュ」 | `shoe`（シュー） |
| `/ʒ/` | 「ジュ」（濁った `/ʃ/`） | `vision`（ヴィジョン） |
| `/tʃ/` | 「チュ」 | `chair`（チェア） |
| `/dʒ/` | 「ジュ」（破裂あり） | `jump`（ジャンプ） |
| `/ŋ/` | 鼻にかかった「ング」 | `song`（ソング） |
| `/r/` | 舌を丸めた「ル」 | `red`（ウレッド） |
| `/j/` | 「イ」から滑る音 | `yes`（イェス） |

## 例題

🔤 `cat` → `/kæt/` → `ク ア ト`（「ア」は口を横に大きく開ける）

- 💡 `/æ/` と `/ʌ/` は似ているが、`/æ/` は口を横に開ける、`/ʌ/` は軽く縦に開ける
- 💡 `/θ/` と `/ð/` は同じ「舌を噛む動作」で、`/θ/` は無声、`/ð/` は有声
- 💡 `/iː/` と `/ɪ/` は長さだけでなく口の形も違う：`/iː/` は口を横に引く

## 解答

発音記号と対応する音・単語を答えましょう。

| 発音記号 | 音のイメージ | 代表的な単語 |
|----------|-------------|-------------|
| `/æ/` | ア（口を横に開ける） | cat, apple, hat |
| `/ʌ/` | ア（軽く縦に開ける） | cup, sun, up |
| `/ɑː/` | アー（口を大きく開ける） | car, far, bar |
| `/ə/` | ア（力を抜く・シュワー） | about, banana, sofa |
| `/ɜː/` | アー（中間の音） | bird, word, nurse |
| `/iː/` | イー（長い） | see, feel, team |
| `/ɪ/` | イ（短い・ゆるい） | hit, sit, big |
| `/uː/` | ウー（長い） | too, food, blue |
| `/ʊ/` | ウ（短い・ゆるい） | book, cook, put |
| `/eɪ/` | エイ（二重母音） | day, say, make |
| `/aɪ/` | アイ（二重母音） | fly, night, time |
| `/θ/` | ス（舌先を噛む・無声） | think, three, tooth |
| `/ð/` | ズ（舌先を噛む・有声） | this, that, the |
| `/ʃ/` | シュ | shoe, ship, wish |
| `/ʒ/` | ジュ（濁り） | vision, measure |
| `/tʃ/` | チュ | chair, watch, teach |
| `/dʒ/` | ジュ（破裂） | jump, age, bridge |
| `/ŋ/` | ング（鼻音） | song, ring, long |
