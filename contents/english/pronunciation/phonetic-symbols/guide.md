# 発音記号（IPA）

## 📚 発音記号とは

- 辞書に載っている `/æ/` や `/iː/` などの記号のこと
- 世界共通の「音の表記法」（国際音声記号 IPA）
- カタカナでは表せない微妙な音の違いを表せる

## 📌 母音（ぼいん）

### 短母音

| 発音記号 | 音のイメージ | 例 |
|----------|-------------|-----|
| `/ɪ/` | 短くゆるい「イ」 | `sit`（シット） |
| `/e/` | 日本語の「エ」に近い | `bed`（ベッド） |
| `/æ/` | 「ア」と「エ」の間（口を横に大きく開ける） | `cat`（キャット） |
| `/ɑ/` | 口を大きく開けた短い「ア」（米英語） | `hot`（ハット） |
| `/ʌ/` | 口を軽く縦に開けた短い「ア」 | `cut`（カット） |
| `/ʊ/` | 短くゆるい「ウ」 | `book`（ブック） |
| `/ɒ/` | 口を丸く開けた短い「オ」（英英語） | `not`（ノット） |
| `/ə/` | 力を抜いた最も弱い「ア」（シュワー） | `about`（アバウト） |

### 長母音・R音性母音

| 発音記号 | 音のイメージ | 例 |
|----------|-------------|-----|
| `/iː/` | 長くはっきりした「イー」 | `see`（スィー） |
| `/uː/` | 長くはっきりした「ウー」（唇を丸める） | `too`（トゥー） |
| `/ɑː/` | 口を大きく開けた長い「アー」 | `far`（ファー） |
| `/ɔː/` | 口を丸めた長い「オー」 | `law`（ロー） |
| `/ɜːr/` | 「アー」と「エー」の中間（舌を丸める） | `bird`（バード） |
| `/ɚ/` | 舌を丸めた弱いR音性「ア」 | `butter`（バター） |

### 二重母音（ダイフトング）

| 発音記号 | 音のイメージ | 例 |
|----------|-------------|-----|
| `/aɪ/` | 「アイ」 | `my`（マイ） |
| `/aʊ/` | 「アウ」 | `cow`（カウ） |
| `/eɪ/` | 「エイ」 | `say`（セイ） |
| `/oʊ/` | 「オウ」（主に米英語） | `go`（ゴウ） |
| `/ɔɪ/` | 「オイ」 | `boy`（ボイ） |

## 📌 子音（しいん）

### 破裂音

| 発音記号 | 音のイメージ | 例 |
|----------|-------------|-----|
| `/p/` | 唇を閉じて開く無声音 | `pen`（ペン） |
| `/b/` | 唇を閉じて開く有声音 | `bad`（バッド） |
| `/t/` | 舌先を歯茎に当てる無声音 | `tea`（ティー） |
| `/d/` | 舌先を歯茎に当てる有声音 | `did`（ディッド） |
| `/k/` | 舌の奥を使う無声音 | `cat`（キャット） |
| `/ɡ/` | 舌の奥を使う有声音 | `go`（ゴー） |
| `/ʔ/` | 声門閉鎖音（のどを閉じて開く） | `bottle`（ボ`ʔ`ル） |

### 摩擦音

| 発音記号 | 音のイメージ | 例 |
|----------|-------------|-----|
| `/f/` | 上の歯を下唇に当てる無声音 | `fan`（ファン） |
| `/v/` | 上の歯を下唇に当てる有声音 | `van`（ヴァン） |
| `/θ/` | 舌先を上の歯に当てる無声音（「ス」に近い） | `think`（シンク） |
| `/ð/` | 舌先を上の歯に当てる有声音（「ズ」に近い） | `this`（ジス） |
| `/s/` | 「ス」 | `see`（スィー） |
| `/z/` | 「ズ」 | `zoo`（ズー） |
| `/ʃ/` | 「シュ」 | `she`（シー） |
| `/ʒ/` | 「ジュ」（濁った `/ʃ/`） | `vision`（ヴィジョン） |
| `/h/` | 息だけを出す「ハ」行 | `he`（ヒー） |

### 破擦音

| 発音記号 | 音のイメージ | 例 |
|----------|-------------|-----|
| `/tʃ/` | 「チュ」 | `chat`（チャット） |
| `/dʒ/` | 「ジュ」（破裂あり） | `jam`（ジャム） |

### 鼻音

| 発音記号 | 音のイメージ | 例 |
|----------|-------------|-----|
| `/m/` | 唇を閉じた鼻音 | `man`（マン） |
| `/n/` | 舌先を歯茎に当てた鼻音 | `not`（ノット） |
| `/ŋ/` | 鼻にかかった「ング」 | `sing`（スィング） |

### 接近音・流音

| 発音記号 | 音のイメージ | 例 |
|----------|-------------|-----|
| `/l/` | 舌先を歯茎に当てた「ル」 | `let`（レット） |
| `/r/` | 舌を丸めた「ル」（舌先を歯茎に当てない） | `run`（ラン） |
| `/w/` | 唇を丸めた「ウ」から次の音へ滑る | `wet`（ウェット） |
| `/j/` | 「イ」から次の音へ滑る | `yes`（イェス） |

## 📝 例題

🔤 `cat` → `/kæt/` → `ク ア ト`（「ア」は口を横に大きく開ける）

- 💡 `/æ/` と `/ʌ/` は似ているが、`/æ/` は口を横に開ける、`/ʌ/` は軽く縦に開ける
- 💡 `/θ/` と `/ð/` は同じ「舌を噛む動作」で、`/θ/` は無声、`/ð/` は有声
- 💡 `/iː/` と `/ɪ/` は長さだけでなく口の形も違う：`/iː/` は口を横に引く
- 💡 `/l/` と `/r/` はどちらも「ル」に聞こえるが、`/l/` は舌先を歯茎にしっかり当てる
