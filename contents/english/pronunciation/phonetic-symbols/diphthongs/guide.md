# 発音記号 — 二重母音

## 📚 二重母音（ダイフトング）とは

- 2つの母音音が滑らかにつながる音
- 最初の音から次の音へ「滑る」ように発音する

## 📌 二重母音一覧

| 発音記号 | 音のイメージ | 例 |
|----------|-------------|-----|
| `/aɪ/` | 「アイ」 | `my`（マイ） |
| `/aʊ/` | 「アウ」 | `cow`（カウ） |
| `/eɪ/` | 「エイ」 | `say`（セイ） |
| `/oʊ/` | 「オウ」（主に米英語） | `go`（ゴウ） |
| `/ɔɪ/` | 「オイ」 | `boy`（ボイ） |

## 📝 例題

🔤 `my` → `/maɪ/` → `マイ`（「ア」から「イ」へ滑らかに）

- 💡 二重母音は前の音を強く、後の音を弱くする
- 💡 `/oʊ/` は主にアメリカ英語で使われる（イギリス英語では `/əʊ/`）

## 📝 例

| 発音記号 | 音のイメージ | 代表的な単語 |
|----------|-------------|-------------|
| `/aɪ/` | アイ（二重母音） | my, fly, night |
| `/aʊ/` | アウ（二重母音） | cow, now, out |
| `/eɪ/` | エイ（二重母音） | say, day, make |
| `/oʊ/` | オウ（二重母音・米） | go, home, stone |
| `/ɔɪ/` | オイ（二重母音） | boy, coin, voice |
