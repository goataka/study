# 五十音ローマ字 - 解説

## ローマ字とは

- ひらがな（日本語の音）を英字（ローマ字）で表す方法
- 英語とは発音が異なるので注意！

## 基本ルール

- 子音 + 母音（a/i/u/e/o）で1つの音になる
- 例: `k` + `a` = `ka`（か）

## 特殊な書き方

| ひらがな | ローマ字 | ポイント |
|----------|----------|---------|
| し | `shi` | `si` ではない |
| ち | `chi` | `ti` ではない |
| つ | `tsu` | `tu` ではない |
| ふ | `fu` | `hu` ではない |
| を | `wo` | `o` ではない |
| ん | `n` | 単独の `n` |

## 例題

🇯🇵 `さくら` → `s`a`k`u`r`a

🇯🇵 `ありがとう` → `a`ri`g`a`t`o`u`

- 💡 `n` の後に母音が続く場合は `n'` と区切ることがある（例: `ho-n'ya`）
