# フォニックス（3文字） - 解説

## 3文字の特殊パターン

### `tch` / `dge`（語末の「チ」「ジ」）

短母音の直後に来るパターン。

| 文字 | 音 | 例 |
|------|----|----|
| `tch` | チ | ma`tch`, ca`tch`, ki`tch`en |
| `dge` | ジ | e`dge`, bri`dge`, ju`dge` |

- 💡 長母音・2文字母音の後は `ch` / `ge` になる（例: teach, huge）

### `igh`（アイの音）

| 文字 | 音 | 例 |
|------|----|----|
| `igh` | アイ | f`igh`t, n`igh`t, l`igh`t |

- 💡 `gh` は発音しない（サイレントレター）

### 子音クラスター（3文字子音の組み合わせ）

3つの子音が連なって1つの音のかたまりになる。

| 文字 | 音 | 例 |
|------|----|----|
| `sch` | スク | `sch`ool |
| `thr` | スル | `thr`ee, `thr`ow |
| `shr` | シュル | `shr`imp |
| `str` | ストゥル | `str`eet, `str`ong |
| `spr` | スプル | `spr`ing |
| `spl` | スプル | `spl`ash |
| `scr` | スクル | `scr`een |

## 例題

🔤 ma`tch` → `チ` / f`igh`t → `アイ` / `str`eet → `ストゥル`

- 💡 子音クラスターは素早く続けて発音する（各文字の音をくっつけるイメージ）

## 解答

下線部の音をカタカナで書きましょう。

| 単語 | 単語全体 | 意味 |
|------|----------|------|
| ki`tch`en | ク イ `チ` エ ヌ → クイチエヌ | だいどころ |
| ma`tch` | ム ア `チ` → ムアチ | マッチ |
| ca`tch` | ク ア `チ` → クアチ | つかまえる |
| e`dge` | エ `ジ` → エジ | ふち |
| bri`dge` | ブ ル イ `ジ` → ブルイジ | はし（橋） |
| ju`dge` | ジュ ア `ジ` → ジュアジ | さいばんかん |
| f`igh`t | フ `アイ` ト → フアイト | けんか |
| n`igh`t | ヌ `アイ` ト → ヌアイト | よる |
| l`igh`t | ル `アイ` ト → ルアイト | ひかり |
| `sch`ool | `スク` ウー ル → スクウール | がっこう |
| `thr`ee | `スル` イー → スルイー | みっつ |
| `thr`ow | `スル` オウ → スルオウ | なげる |
| `shr`imp | `シュル` イ ム プ → シュルイムプ | えび |
| `str`eet | `ストゥル` イー ト → ストゥルイート | みち |
| `str`ong | `ストゥル` オ ン グ → ストゥルオング | つよい |
| `spr`ing | `スプル` イ ン グ → スプルイング | はる |
| `spl`ash | `スプル` ア シュ → スプルアシュ | はねる |
| `scr`een | `スクル` イー ヌ → スクルイーヌ | がめん |
