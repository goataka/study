# リダクション（音の脱落） - 解説

## リダクションとは

- 自然な会話では、一部の音が弱くなったり消えたりする
- 特に `to`・`of`・助動詞などで起きやすい

## よくあるリダクションパターン

### `to` のリダクション

| 元の形 | 変化後 | 変化のしかた |
|--------|--------|------------|
| `want to` | ワナ | to → na |
| `going to` | ゴナ / ガナ | going to → gonna |
| `have to` | ハフタ | ve が弱化 |
| `got to` | ガタ | t が弱化 |

### `of` のリダクション

| 元の形 | 変化後 | 変化のしかた |
|--------|--------|------------|
| `kind of` | カインダ | of → a（弱化） |
| `sort of` | ソータ | t が消失 |
| `a lot of` | アロダ / アロラ | t が弱化・ラ行化 |
| `out of` | アウタ / アウラ | t が弱化・ラ行化 |

## 例題

💨 `want to` go → `ワナ` ゴウ

💨 `going to` rain → `ゴナ` レイン

## ポイント

- 💡 書き言葉では `wanna`・`gonna` とスペリングされることもある
- 💡 リダクションは「省略」ではなく「自然な発音の変化」
- 💡 聞き取りが難しいのはリダクションが原因のことが多い

## 練習問題

音が弱くなる・消える部分の音をカタカナで書きましょう。

| フレーズ | 音 |
|----------|------|
| want to | 　　　　　　　　　　　　 |
| going to | 　　　　　　　　　　　　 |
| have to | 　　　　　　　　　　　　 |
| got to | 　　　　　　　　　　　　 |
| kind of | 　　　　　　　　　　　　 |
| sort of | 　　　　　　　　　　　　 |
| a lot of | 　　　　　　　　　　　　 |
| out of | 　　　　　　　　　　　　 |

## 解答

音が弱くなる・消える部分の音をカタカナで書きましょう。

| フレーズ | 音 | 変化した部分 |
|----------|------|--------------|
| want to | ワナ | t が消失 |
| going to | ゴナ / ガナ | ing to → na |
| have to | ハフタ | ve が弱化 |
| got to | ガタ | t が弱化 |
| kind of | カインダ | d が弱化 |
| sort of | ソータ | t が消失 |
| a lot of | アロダ / アロラ | t が弱化・ラ行化 |
| out of | アウタ / アウラ | t が弱化・ラ行化 |
