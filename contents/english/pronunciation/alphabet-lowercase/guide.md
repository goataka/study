# アルファベット小文字の読み

## 📚 小文字の読み方

- 大文字・小文字は形が違っても、読み方（名前）は同じ
- 例: A/a → 両方とも「えい」

## 📌 母音と子音

- 🔵 **母音（ぼいん）**: a・e・i・o・u の5文字
- 🔴 **子音（しいん）**: 残りの21文字

## 📝 例題

🔤 a → `えい` / b → `びー` / z → `ぜっと`

- 💡 w（ダブリュー）・x（エックス）・y（ワイ）は特に注意！
