# 音声変化練習ドリル - 解答

## リンキング、リダクション、フラッピング、アシミレーションの音を書きましょう

音声変化した部分の音をカタカナで書きましょう。

### リンキング（Linking）

音がつながる

| フレーズ | 音 |
|----------|------|
| check it out | チェキットアウト |
| turn it off | ターニットオフ |
| put it on | プティットオン |
| pick it up | ピキットアップ |
| take a look | テイカルック |
| not at all | ノタットール |
| an apple | アナップル |
| turn on | ターノン |

### リダクション（Reduction）

音が弱くなる・消える

| フレーズ | 音 |
|----------|------|
| want to | ワナ |
| going to | ゴナ / ガナ |
| have to | ハフタ |
| got to | ガタ |
| kind of | カインダ |
| sort of | ソータ |
| a lot of | アロダ / アロラ |
| out of | アウタ / アウラ |

### フラッピング（Flapping）

t/dがラ行になる

| フレーズ | 音 |
|----------|------|
| water | ワラー |
| better | ベラー |
| little | リロー |
| get it | ゲリット |
| what about | ワラバウト |
| put it | プリット |
| city | シリー |
| party | パーリー |

### アシミレーション（Assimilation）

となりの音に引っ張られて音が変わる

| フレーズ | 変化 | 音 |
|----------|------|------|
| meet you | t+y→チュ | ミーチュー |
| can't you | t+y→チュ | キャンチュー |
| would you | d+y→ジュ | ウッジュー |
| could you | d+y→ジュ | クッジュー |
| miss you | s+y→シュ | ミッシュー |
| bless you | s+y→シュ | ブレッシュー |
| ten people | n→m（唇音の前） | テムピープル |
| in Paris | n→m（唇音の前） | イムパリ |
