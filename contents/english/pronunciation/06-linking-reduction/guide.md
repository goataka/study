# 音声変化まとめ - 解説

## 音声変化とは

- ネイティブスピーカーが自然に話すとき、音がつながったり変化したりする
- 4つの主な変化を覚えよう

## 4つの音声変化

| 種類 | 変化のしかた | 例 |
|------|------------|-----|
| 🔗 **リンキング** | 音がつながる | check `it` out → チェキットアウト |
| 💨 **リダクション** | 音が弱くなる・消える | `want to` → ワナ |
| 👅 **フラッピング** | t/d がラ行になる | `water` → ワラー |
| 🔄 **アシミレーション** | 隣の音に変わる | `meet you` → ミーチュー |

## リンキング（Linking）🔗

子音で終わる語 + 母音で始まる語 → 音がつながる

- `check it out` → チェ **キッ** ト アウ ト（k + i → キ）

## リダクション（Reduction）💨

弱く発音される、または消える音

- `want to` → ワナ（t が消え、to → na）
- `going to` → ゴナ

## フラッピング（Flapping）👅

t / d が「母音 + t/d + 母音」のパターンでラ行になる

- `water` → ワ **ラ** ー（t → ラ）

## アシミレーション（Assimilation）🔄

となりの音に引っ張られて音が変わる

- `meet you` → ミー **チュ** ー（t + y → チュ）
