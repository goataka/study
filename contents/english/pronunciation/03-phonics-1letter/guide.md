# フォニックス（1文字） - 解説

## フォニックスとは

- 文字の「音（おと）」を学ぶ方法
- アルファベットの「名前」とは違う！

| 文字 | 名前（読み方） | フォニックスの音 |
|------|--------------|----------------|
| A | えー | `ア`（apple の最初の音） |
| B | びー | `ブ`（ball の最初の音） |

## 母音（ぼいん）の短母音

| 文字 | 音 | 例 |
|------|----|----|
| `a` | ア | `a`pple |
| `e` | エ | `e`gg |
| `i` | イ | `i`nk |
| `o` | ア/オ | `o`range |
| `u` | ア | `u`p |

## 例題

🔤 `a`pple → `ア` / `b`all → `ブ` / `c`at → `ク`

- 💡 `q` は単独では使わず `qu`（クウ）の形が多い
- 💡 `o` は「オ」ではなく「ア」に近い音が多い

## 解答

下線部の音をカタカナで書きましょう。

| 単語 | 単語全体 | 意味 |
|------|----------|------|
| `a`pple | `ア` プ ル → アプル | りんご |
| `b`all | `ブ` オール → ブオール | ボール |
| `c`at | `ク` ア ト → クアト | ねこ |
| `d`og | `ドゥ` オ グ → ドゥオグ | いぬ |
| `e`gg | `エ` グ → エグ | たまご |
| `f`ish | `フ` イ シュ → フイシュ | さかな |
| `g`irl | `グ` アール → グアール | おんなのこ |
| `h`at | `ハ` ア ト → ハアト | ぼうし |
| `i`nk | `イ` ン ク → インク | インク |
| `j`am | `ジュ` ア ム → ジュアム | ジャム |
| `k`ite | `ク` アイ ト → クアイト | たこ（凧） |
| `l`ion | `ル` アイ オン → ルアイオン | ライオン |
| `m`ap | `ム` ア プ → ムアプ | ちず |
| `n`et | `ヌ` エ ト → ヌエト | あみ |
| `o`range | `ア` リン ジュ → アリンジュ | オレンジ |
| `p`en | `プ` エ ヌ → プエヌ | ペン |
| `q`ueen | `クウ` イー ヌ → クウイーヌ | じょおう |
| `r`ed | `ル` エ ドゥ → ルエドゥ | あか |
| `s`un | `ス` ア ヌ → スアヌ | たいよう |
| `t`iger | `トゥ` アイ ガー → トゥアイガー | とら |
| `u`p | `ア` プ → アプ | うえ |
| `v`an | `ヴ` ア ヌ → ヴアヌ | バン（車） |
| `w`ater | `ウ` オー ラー → ウオーラー | みず |
| bo`x` | ブ ア `クス` → ブアクス | はこ |
| `y`es | `イ` エ ス → イエス | はい |
| `z`oo | `ズ` ウー → ズウー | どうぶつえん |
