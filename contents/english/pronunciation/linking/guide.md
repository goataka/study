# リンキング（音のつながり）

## リンキングとは

- 子音で終わる語 + 母音で始まる語が続くと、音がつながって聞こえる
- 単語の境界がなくなり、まるで1つの単語のように発音される

## リンキングのパターン

| パターン | 例 | つながり方 |
|---------|-----|-----------|
| 子音 + 母音 | `check it` out | k + i → キ |
| `n` + 母音 | `turn it` | n + i → ニ |
| `t` + 母音 | `put it` on | t + i → ティ |
| `k` + 母音 | `pick it` up | k + i → キ |

## 例題

🔗 `check it out`

```text
check  it  out
     ↘↗   ↘↗
 チェ キッ ト アウ ト
→ チェキットアウト
```

🔗 `not at all`

```text
not  at  all
   ↘↗  ↘↗
ノ タッ ト オー ル
→ ノタットール
```

## ポイント

- 💡 前の語の語末子音と、次の語の語頭母音がくっつく
- 💡 聞き取るときは「単語の区切り」より「音のかたまり」で捉える
- 💡 `an apple` → `アナップル`（n + a → ナ）

## 解答

音がつながる部分の音をカタカナで書きましょう。

| フレーズ | 音 | 変化した部分 |
|----------|------|--------------|
| check it out | チェキットアウト | k + i → キ |
| turn it off | ターニットオフ | n + i → ニ |
| put it on | プティットオン | t + i → ティ |
| pick it up | ピキットアップ | k + i → キ |
| take a look | テイカルック | k + ə → カ |
| not at all | ノタットール | t + a → タ |
| an apple | アナップル | n + a → ナ |
| turn on | ターノン | n + o → ノ |
