# 時制の英作文ドリル

## 英作文

日本語を英語にしましょう。

### 一般動詞

| 日本語 | 英語 |
|--------|------|
| 私はゲームをします。 | 　　　　　　　　　　　　 |
| 私はゲームをしました。 | 　　　　　　　　　　　　 |
| 私はゲームをするでしょう。 | 　　　　　　　　　　　　 |
| 私は英語を勉強します。 | 　　　　　　　　　　　　 |
| 私は英語を勉強しました。 | 　　　　　　　　　　　　 |
| 私は英語を勉強するでしょう。 | 　　　　　　　　　　　　 |

### be動詞

| 日本語 | 英語 |
|--------|------|
| 私は幸せです。 | 　　　　　　　　　　　　 |
| 私は幸せでした。 | 　　　　　　　　　　　　 |
| 私は幸せになるでしょう。 | 　　　　　　　　　　　　 |
| 私は学生です。 | 　　　　　　　　　　　　 |
| 私は学生でした。 | 　　　　　　　　　　　　 |

### 不規則動詞

| 日本語 | 英語 |
|--------|------|
| 私は学校に行きます。 | 　　　　　　　　　　　　 |
| 私は学校に行きました。 | 　　　　　　　　　　　　 |
| 私は手紙を書きます。 | 　　　　　　　　　　　　 |
| 私は手紙を書きました。 | 　　　　　　　　　　　　 |
| 私は朝食を食べます。 | 　　　　　　　　　　　　 |
| 私は朝食を食べました。 | 　　　　　　　　　　　　 |
