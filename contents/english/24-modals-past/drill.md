# 助動詞の過去形ドリル

## 穴埋め問題

( ) に正しい過去形を入れましょう。

| 現在 | 過去 |
|------|------|
| I can swim. | I (　　　　　　) swim. |
| I will help. | I (　　　　　　) help. |
| I must go. | I (　　　　　　) go. |
| I may come. | I (　　　　　　) come. |

## 英作文

日本語を英語にしましょう。

| 日本語 | 英語 |
|--------|------|
| 私は泳ぐことができた。 | 　　　　　　 |
| 私は手伝うつもりだった。 | 　　　　　　 |
| 私は行かなければならなかった。 | 　　　　　　 |
