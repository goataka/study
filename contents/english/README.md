# 英語学習ガイド

## 学習の流れ

英語学習は「発音」と「文法」の2つのカテゴリに分けて学習します。

### 📢 発音

発音学習では「読み書きの基礎」→「フォニックス」→「音声変化」の順に進みます。

[発音学習ガイドを見る](pronunciation/README.md)

### 📝 文法

文法学習では基本的な文の形から、応用的な表現へと段階的に学習します。

[文法学習ガイドを見る](grammar/README.md)
