# 一般動詞の未来形ドリル

## 穴埋め問題

( ) に正しい語を入れましょう。

| 現在形 | 未来形 |
|--------|--------|
| I play games. | I (　　　　　　) games. |
| I study English. | I (　　　　　　) English. |
| I watch TV. | I (　　　　　　) TV. |
| I like music. | I (　　　　　　) music. |
| I walk to school. | I (　　　　　　) to school. |

## 英作文

日本語を英語にしましょう。

| 日本語 | 英語 |
|--------|------|
| 私はゲームをするでしょう。 | 　　　　　　 |
| 私は英語を勉強するでしょう。 | 　　　　　　 |
| 私はテレビを見るでしょう。 | 　　　　　　 |
