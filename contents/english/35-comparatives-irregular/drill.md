# 不規則な比較変化ドリル

## 穴埋め問題

( ) に正しい語を入れましょう。

| 原級 | 比較級 | 最上級 |
|------|--------|--------|
| good | (　　　　　　) | (　　　　　　) |
| bad | (　　　　　　) | (　　　　　　) |
| many / much | (　　　　　　) | (　　　　　　) |
| little | (　　　　　　) | (　　　　　　) |

## 英作文

日本語を英語にしましょう。

| 日本語 | 英語 |
|--------|------|
| この本はあれよりよい。 | 　　　　　　 |
| これは一番よい本です。 | 　　　　　　 |
| 彼は私よりたくさん持っている。 | 　　　　　　 |
