# 不規則な比較変化 - 解説

原級:
This is a `good` book.

比較級:
This book is `better` than that one.

最上級:
This is `the best` book.

- 一部の形容詞は不規則に変化する

## 不規則変化一覧

| 原級 | 比較級 | 最上級 |
|------|--------|--------|
| `good` | `better` | `best` |
| `bad` | `worse` | `worst` |
| `many` / `much` | `more` | `most` |
| `little` | `less` | `least` |
