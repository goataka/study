# 所有格の関係代名詞ドリル

## 穴埋め問題

( ) に正しい関係代名詞を入れましょう。

2文を1文にする例:
I have a friend. His father is a doctor.
→ I have a friend `whose` father is a doctor.

| 文 |
|------|
| I have a friend (　　　　　　) father is a doctor. |
| I have a book (　　　　　　) cover is red. |
| I have a friend (　　　　　　) mother is a teacher. |
