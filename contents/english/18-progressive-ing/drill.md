# -ing形の作り方ドリル

## 穴埋め問題

原形を参考に、-ing形の ( ) を埋めましょう。

| 原形 | -ing形 |
|------|--------|
| play → playing | I am playing games. |
| study | I am (　　　　　　) English. |
| run | I am (　　　　　　) fast. |
| swim | I am (　　　　　　) in the pool. |
| write | I am (　　　　　　) a letter. |
| make | I am (　　　　　　) a cake. |
| sit | I am (　　　　　　) on the chair. |
| dance | I am (　　　　　　) now. |
| come | I am (　　　　　　) home. |
| have | I am (　　　　　　) lunch. |
