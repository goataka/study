# -ing形の作り方 - 解説

動詞の原形から -ing 形を作るルールです。

## -ing 形の変化ルール

| ルール | 変化 | 例 |
|--------|------|------|
| 通常 | そのまま + ing | play → `playing` |
| e で終わる | e を取って + ing | make → `making` |
| 短母音 + 子音 | 子音を重ねて + ing | run → `running` |
| ie で終わる | ie → y + ing | die → `dying` |
