| meet you | t+y→チュ | 　　　　　　　　　　　　 |
| can't you | t+y→チュ | 　　　　　　　　　　　　 |
| would you | d+y→ジュ | 　　　　　　　　　　　　 |
| could you | d+y→ジュ | 　　　　　　　　　　　　 |
| miss you | s+y→シュ | 　　　　　　　　　　　　 |
| bless you | s+y→シュ | 　　　　　　　　　　　　 |
| ten people | n→m（唇音の前） | 　　　　　　　　　　　　 |
