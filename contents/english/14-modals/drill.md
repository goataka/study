# 助動詞ドリル

## 穴埋め問題

( ) に正しい助動詞を入れましょう。

### 助動詞の使い分け

| 意味 | 文 |
|------|------|
| ～できる | I (　　　　　　) play the piano. |
| ～してもよい | I (　　　　　　) go now. |
| ～しなければならない | I (　　　　　　) study hard. |
| ～すべきだ | I (　　　　　　) go to bed early. |
| ～するでしょう | I (　　　　　　) help you. |
| ～かもしれない | I (　　　　　　) be late. |

### 助動詞の否定形

| 肯定文 | 否定文 |
|--------|--------|
| I can swim. | I (　　　　　　) swim. |
| I must go. | I (　　　　　　) go. |
| I will come. | I (　　　　　　) come. |
| I should stay. | I (　　　　　　) stay. |
| I may leave. | I (　　　　　　) leave. |

### 助動詞の過去形

| 現在 | 過去 |
|------|------|
| I can swim. | I (　　　　　　) swim. |
| I will help. | I (　　　　　　) help. |
| I must go. | I (　　　　　　) go. |

### must と have to の違い

| 意味 | 文 |
|------|------|
| ～しなければならない（義務） | I (　　　　　　) study hard. |
| ～しなければならない（必要性） | I (　　　　　　) study hard. |
| ～する必要はない | I (　　　　　　) study today. |
| ～してはいけない | I (　　　　　　) cheat. |

| 日本語 | 英語 |
