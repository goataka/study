# 比較級・最上級ドリル

## 穴埋め問題

( ) に正しい語を入れましょう。

### 形容詞の比較変化

基本文（現在形）: I am tall.

| パターン | 文 |
|----------|------|
| 原級 | I am as tall as him. |
| 比較級 | I am (　　　　　　) than him. |
| 最上級 | I am the (　　　　　　) in the class. |

### 短い形容詞の変化

| 原級 | 比較級 | 最上級 |
|------|--------|--------|
| tall | taller | (　　　　　　) |
| big | (　　　　　　) | biggest |
| easy | (　　　　　　) | (　　　　　　) |
| hot | (　　　　　　) | (　　　　　　) |

### 長い形容詞の変化

| 原級 | 比較級 | 最上級 |
|------|--------|--------|
| interesting | (　　　　　　) | (　　　　　　) |
| difficult | (　　　　　　) | (　　　　　　) |
| beautiful | (　　　　　　) | (　　　　　　) |

### 不規則変化

| 原級 | 比較級 | 最上級 |
|------|--------|--------|
| good | (　　　　　　) | (　　　　　　) |
| bad | (　　　　　　) | (　　　　　　) |

## 英作文

日本語を英語にしましょう。

| 日本語 | 英語 |
|--------|------|
| 私は彼と同じくらい背が高い。 | 　　　　　　　　　　　　 |
| 私は彼より背が高い。 | 　　　　　　　　　　　　 |
| 私はクラスで一番背が高い。 | 　　　　　　　　　　　　 |
| このゲームはあれより面白い。 | 　　　　　　　　　　　　 |
| このゲームは全部の中で一番面白い。 | 　　　　　　　　　　　　 |
