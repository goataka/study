# 疑問文ドリル

## 穴埋め問題

( ) に正しい語を入れましょう。

### 一般動詞の疑問文

| 肯定文 | 疑問文 |
|--------|--------|
| I play games. | (　　　　　　) I play games? |
| I played games. | (　　　　　　) I play games? |
| I will play games. | (　　　　　　) I play games? |

### be動詞の疑問文

| 肯定文 | 疑問文 |
|--------|--------|
| I am happy. | (　　　　　　) I happy? |
| I was happy. | (　　　　　　) I happy? |
| I am playing games. | (　　　　　　) I playing games? |

## 英作文

日本語を英語にしましょう。

| 日本語 | 英語 |
|--------|------|
| 私はゲームをしますか？ | 　　　　　　 |
| 私はゲームをしましたか？ | 　　　　　　 |
| 私は幸せですか？ | 　　　　　　 |
