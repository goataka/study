# 関係代名詞ドリル

## Who を使った関係代名詞

下の文を日本語に訳しましょう。

| 英文 | 日本語訳 |
|------|----------|
| I know a man who speaks five languages. | 　　　　　　　　　　　　 |
| She is the girl who won the prize. | 　　　　　　　　　　　　 |
| He is the teacher who taught me English. | 　　　　　　　　　　　　 |
| The boy who is playing soccer is my brother. | 　　　　　　　　　　　　 |
| People who live in Tokyo are very busy. | 　　　　　　　　　　　　 |
| The woman who called you is my mother. | 　　　　　　　　　　　　 |

## Which を使った関係代名詞

|------|----------|

| The book which is on the table is mine. | 　　　　　　　　　　　　 |
| He likes movies which are funny. | 　　　　　　　　　　　　 |

## 関係代名詞の省略

省略できる関係代名詞を見つけましょう。

| 英文 | 省略形 | 日本語訳 |
|------|--------|----------|
| This is the book which I bought. | 　　　　　　　　　　　　 | 　　　　　　　　　　　　 |
| The man whom I met is my teacher. | 　　　　　　　　　　　　 | 　　　　　　　　　　　　 |
| I like the car that he drives. | 　　　　　　　　　　　　 | 　　　　　　　　　　　　 |
| The movie which we saw was great. | 　　　　　　　　　　　　 | 　　　　　　　　　　　　 |
