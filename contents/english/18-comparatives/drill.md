# 比較級・最上級ドリル

## 穴埋め問題

( ) に正しい語を入れましょう。

### 形容詞の比較級

基本文: This game is fun.

| パターン | 文 |
|----------|------|
| 原級 | This game is as (　　　　　　) as that one. |
| 比較級 | This game is (　　　　　　) than that one. |
| 最上級 | This game is the (　　　　　　) of all. |

### 短い形容詞の変化

| 原級 | 比較級 | 最上級 |
|------|--------|--------|
| tall | (　　　　　　) | (　　　　　　) |
| big | (　　　　　　) | (　　　　　　) |
| easy | (　　　　　　) | (　　　　　　) |
| hot | (　　　　　　) | (　　　　　　) |

### 長い形容詞の変化

| 原級 | 比較級 | 最上級 |

## 英作文

日本語を英語にしましょう。

| 日本語 | 英語 |
|--------|------|
| このゲームはあれと同じくらい面白い。 | 　　　　　　　　　　　　 |
| このゲームはあれより面白い。 | 　　　　　　　　　　　　 |
| このゲームは全部の中で一番面白い。 | 　　　　　　　　　　　　 |
| 私は背が高い。 | 　　　　　　　　　　　　 |
| 私は彼より背が高い。 | 　　　　　　　　　　　　 |
| 私はクラスで一番背が高い。 | 　　　　　　　　　　　　 |
