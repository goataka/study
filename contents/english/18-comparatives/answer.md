# 比較級・最上級ドリル - 解答

## 穴埋め問題

### 形容詞の比較級

| パターン | 文 |
|----------|------|
| 原級 | This game is as (fun) as that one. |
| 比較級 | This game is (more fun) than that one. |
| 最上級 | This game is the (most fun) of all. |

### 短い形容詞の変化

| 原級 | 比較級 | 最上級 |
|------|--------|--------|
| tall | (taller) | (tallest) |
| big | (bigger) | (biggest) |
| easy | (easier) | (easiest) |
| hot | (hotter) | (hottest) |

### 長い形容詞の変化

| 原級 | 比較級 | 最上級 |
|------|--------|--------|
| interesting | (more interesting) | (most interesting) |
| difficult | (more difficult) | (most difficult) |
| beautiful | (more beautiful) | (most beautiful) |

### 不規則変化

| 原級 | 比較級 | 最上級 |
|------|--------|--------|
| good | (better) | (best) |
| bad | (worse) | (worst) |
| many/much | (more) | (most) |
| little | (less) | (least) |

## 英作文

| 日本語 | 英語 |
|--------|------|
| このゲームはあれと同じくらい面白い。 | This game is as fun as that one. |
| このゲームはあれより面白い。 | This game is more fun than that one. |
| このゲームは全部の中で一番面白い。 | This game is the most fun of all. |
| 私は背が高い。 | I am tall. |
| 私は彼より背が高い。 | I am taller than him. |
| 私はクラスで一番背が高い。 | I am the tallest in the class. |
