# 比較級・最上級 - 説明

## 3つの比較パターン

| パターン | 形 | 例 |
|----------|------|------|
| 原級（同等） | as ～ as | This game is **as fun as** that one. |
| 比較級（～より） | ～er / more ～ + than | This game is **more fun than** that one. |
| 最上級（最も～） | the ～est / the most ～ | This game is **the most fun** of all. |

## 比較級・最上級の作り方

### 短い形容詞（1〜2音節）

| ルール | 原級 → 比較級 → 最上級 |
|--------|------------------------|
| そのまま -er/-est | tall → taller → tallest |
| 子音を重ねる | big → bigger → biggest, hot → hotter → hottest |
| yをiに変える | easy → easier → easiest |

### 長い形容詞（3音節以上）

**more / most** をつける

- interesting → more interesting → most interesting
- beautiful → more beautiful → most beautiful

## 不規則変化

| 原級 | 比較級 | 最上級 |
|------|--------|--------|
| good | better | best |
| bad | worse | worst |
| many/much | more | most |
| little | less | least |
