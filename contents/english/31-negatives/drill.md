# 否定文ドリル

## 穴埋め問題

( ) に正しい語を入れましょう。

### 一般動詞の否定文

| 肯定文 | 否定文 |
|--------|--------|
| I play games. | I (　　　　　　) play games. |
| I played games. | I (　　　　　　) play games. |
| I will play games. | I (　　　　　　) play games. |

### be動詞の否定文

| 肯定文 | 否定文 |
|--------|--------|
| I am happy. | I (　　　　　　) happy. |
| I was happy. | I (　　　　　　) happy. |

## 英作文

日本語を英語にしましょう。

| 日本語 | 英語 |
|--------|------|
| 私はゲームをしません。 | 　　　　　　 |
| 私はゲームをしませんでした。 | 　　　　　　 |
| 私は幸せではありません。 | 　　　　　　 |
