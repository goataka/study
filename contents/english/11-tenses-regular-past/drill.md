# 一般動詞の過去形ドリル

## 穴埋め問題

( ) に正しい語を入れましょう。

| 現在形 | 過去形 |
|--------|--------|
| I play games. | I (　　　　　　) games. |
| I study English. | I (　　　　　　) English. |
| I watch TV. | I (　　　　　　) TV. |
| I like music. | I (　　　　　　) music. |
| I walk to school. | I (　　　　　　) to school. |

## 英作文

日本語を英語にしましょう。

| 日本語 | 英語 |
|--------|------|
| 私はゲームをしました。 | 　　　　　　 |
| 私は英語を勉強しました。 | 　　　　　　 |
| 私はテレビを見ました。 | 　　　　　　 |
