## 穴埋め問題

( ) に正しい否定形を入れましょう。

| 肯定文 | 否定文 |
|--------|--------|
| I can swim. | I (　　　　　　) swim. |
| I must go. | I (　　　　　　) go. |
| I will come. | I (　　　　　　) come. |
| I should stay. | I (　　　　　　) stay. |
| I may leave. | I (　　　　　　) leave. |
