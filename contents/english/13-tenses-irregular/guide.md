# 不規則動詞の過去形 - 解説

## 現在形との比較

規則動詞は -ed をつけるが、不規則動詞は形が変わる。

現在形: I **go** to school.
過去形: I **went** to school.

## 不規則動詞の変化表

| 原形 | 過去形 | 意味 |
|------|--------|------|
| go | went | 行く |
| eat | ate | 食べる |
| write | wrote | 書く |
| run | ran | 走る |
| have | had | 持つ |
| read | read | 読む |
| see | saw | 見る |
| take | took | 取る |
| come | came | 来る |
| make | made | 作る |
| get | got | 得る |
| know | knew | 知る |
| speak | spoke | 話す |
| think | thought | 思う |
| buy | bought | 買う |

> **注意**: read は綴りが同じだが、発音が変わる（現在: リード → 過去: レッド）
