# 受動態 - 解説

能動態（現在形）:
I `play` games.

## 受動態

Games `are played` by me.

- `be動詞` + 過去分詞で受動態を作る
- 「～される」「～された」

## 受動態の時制パターン

| 時制 | be動詞 | 例 |
|------|--------|------|
| 現在形 | `are` | Games `are played` by me. |
| 過去形 | `were` | Games `were played` by me. |
| 未来形 | `will be` | Games `will be played` by me. |
| 現在完了 | `have been` | Games `have been played` by me. |
