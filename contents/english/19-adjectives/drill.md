# 形容詞ドリル

## 穴埋め問題

( ) に正しい形容詞を入れましょう。

| パターン | 文 |
|----------|------|
| 名詞の前 | I play fun games. |
| 名詞の前 | I read a (　　　　　　) book. |
| be動詞の後 | The game is (　　　　　　). |
| 名詞の前 | She has (　　　　　　) hair. |
| be動詞の後 | The book is (　　　　　　). |
| 名詞の前 | He is a (　　　　　　) boy. |

ヒント: interesting, fun, long, new, tall
