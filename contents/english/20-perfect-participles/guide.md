# 過去分詞の変化 - 解説

現在形:
I `play` games.

完了形:
I have `played` games.

- 規則動詞の過去分詞は過去形と同じ（-ed）
- 不規則動詞は別の形になることがある

## 不規則動詞の過去分詞

| 原形 | 過去形 | 過去分詞 |
|------|--------|----------|
| go | went | `gone` |
| eat | ate | `eaten` |
| write | wrote | `written` |
| see | saw | `seen` |
| take | took | `taken` |
| make | made | `made` |
| read | read | `read` |
| speak | spoke | `spoken` |
| know | knew | `known` |
