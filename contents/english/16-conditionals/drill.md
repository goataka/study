# 条件文ドリル

## 第1条件文（現在の習慣・未来の可能性）

下の文を日本語に訳しましょう。

| 英文 | 日本語訳 |
|------|----------|
| If it rains, I will stay home. | 　　　　　　　　　　　　 |
| If you study hard, you will pass the exam. | 　　　　　　　　　　　　 |
| If he comes, I will be happy. | 　　　　　　　　　　　　 |
| If she calls, tell her I'm busy. | 　　　　　　　　　　　　 |
| If we have time, we will visit you. | 　　　　　　　　　　　　 |
| If they arrive early, we can start. | 　　　　　　　　　　　　 |
| If you are tired, you should rest. | 　　　　　　　　　　　　 |
| If it snows, the roads will be dangerous. | 　　　　　　　　　　　　 |

## 条件文のパターン比較

同じ内容の3つの条件文を比べましょう。

| 第1条件文 | 第2条件文 | 第3条件文 |
|-----------|-----------|-----------|
| If I have time, I will help you. | If I had time, I would help you. | If I had had time, I would have helped you. |
| 　　　　　　　　　　　　 | 　　　　　　　　　　　　 | 　　　　　　　　　　　　 |

| 第1条件文 | 第2条件文 | 第3条件文 |
|-----------|-----------|-----------|
| If he studies, he will pass. | If he studied, he would pass. | If he had studied, he would have passed. |
| 　　　　　　　　　　　　 | 　　　　　　　　　　　　 | 　　　　　　　　　　　　 |

| 第1条件文 | 第2条件文 | 第3条件文 |
|-----------|-----------|-----------|
| If it rains, we will stay home. | If it rained, we would stay home. | If it had rained, we would have stayed home. |
| 　　　　　　　　　　　　 | 　　　　　　　　　　　　 | 　　　　　　　　　　　　 |
