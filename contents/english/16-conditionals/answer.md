# 条件文ドリル - 解答

## 第1条件文（現在の習慣・未来の可能性）

下の文を日本語に訳しましょう。

| 英文 | 日本語訳 |
|------|----------|
| If it rains, I will stay home. | もし雨が降れば、私は家にいます。 |
| If you study hard, you will pass the exam. | もしあなたが一生懸命勉強すれば、試験に合格するでしょう。 |
| If he comes, I will be happy. | もし彼が来れば、私は幸せです。 |
| If she calls, tell her I'm busy. | もし彼女が電話したら、忙しいと伝えてください。 |
| If we have time, we will visit you. | もし時間があれば、あなたを訪ねます。 |
| If they arrive early, we can start. | もし彼らが早く着けば、始められます。 |
| If you are tired, you should rest. | もしあなたが疲れているなら、休むべきです。 |
| If it snows, the roads will be dangerous. | もし雪が降れば、道路は危険になるでしょう。 |

## 第2条件文（現在の事実に反する仮定）

| 英文 | 日本語訳 |
|------|----------|

| If he studied harder, he would pass. | もし彼がもっと勉強すれば、合格するのですが。 |
| If she were here, she would be happy. | もし彼女がここにいれば、幸せなのですが。 |
| If we had money, we would buy a car. | もしお金があれば、車を買うのですが。 |
| If they knew the truth, they would be angry. | もし彼らが真実を知っていれば、怒るでしょう。 |
| If I were you, I would apologize. | もし私があなたなら、謝ります。 |
| If it were sunny, we could go to the beach. | もし晴れていれば、ビーチに行けるのですが。 |
| If you lived near me, we could meet often. | もしあなたが近くに住んでいれば、よく会えるのですが。 |

## 第3条件文（過去の事実に反する仮定）

| 英文 | 日本語訳 |
|------|----------|
| If I had known, I would have told you. | もし知っていたら、あなたに言ったのですが。 |
| If he had studied, he would have passed. | もし彼が勉強していたら、合格していたでしょう。 |
| If she had come, she would have met him. | もし彼女が来ていたら、彼に会えたでしょう。 |
| If we had left earlier, we would have arrived on time. | もし私たちがもっと早く出発していたら、時間通りに着いていたでしょう。 |
| If they had known the truth, they would have been angry. | もし彼らが真実を知っていたら、怒っていたでしょう。 |
| If it had rained, we would have stayed home. | もし雨が降っていたら、家にいたでしょう。 |
| If you had asked me, I would have helped you. | もしあなたが私に頼んでいたら、手伝ったのですが。 |

## 条件文のパターン比較

同じ内容の3つの条件文を比べましょう。

| 第1条件文 | 第2条件文 | 第3条件文 |
|-----------|-----------|-----------|
| If I have time, I will help you. | If I had time, I would help you. | If I had had time, I would have helped you. |
| もし時間があれば、手伝います。 | もし時間があれば、手伝うのですが。（実際にはない） | もし時間があったら、手伝ったのですが。（過去） |

| 第1条件文 | 第2条件文 | 第3条件文 |
|-----------|-----------|-----------|
| If he studies, he will pass. | If he studied, he would pass. | If he had studied, he would have passed. |
| もし彼が勉強すれば、合格します。 | もし彼が勉強すれば、合格するのですが。（実際にはしない） | もし彼が勉強していたら、合格していたでしょう。 |

| 第1条件文 | 第2条件文 | 第3条件文 |
|-----------|-----------|-----------|
| If it rains, we will stay home. | If it rained, we would stay home. | If it had rained, we would have stayed home. |
| もし雨が降れば、家にいます。 | もし雨が降れば、家にいるのですが。 | もし雨が降っていたら、家にいたでしょう。 |

## Unless（～でない限り）

| If 文 | Unless 文 | 日本語訳 |
|-------|-----------|----------|
| If you don't hurry, you will be late. | Unless you hurry, you will be late. | 急がないと遅れますよ。 |
| If he doesn't study, he will fail. | Unless he studies, he will fail. | 勉強しないと落第します。 |
| If it doesn't rain, we will go out. | Unless it rains, we will go out. | 雨が降らなければ、出かけます。 |
| If you don't ask, you won't know. | Unless you ask, you won't know. | 尋ねなければ、わかりません。 |

下の文を日本語に訳しましょう。

| 英文 | 日本語訳 |

| If we had left on time, we would be there now. | もし時間通りに出発していたら、今頃そこにいるでしょう。 |
