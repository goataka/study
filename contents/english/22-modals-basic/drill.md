# 助動詞の使い分けドリル

## 穴埋め問題

( ) に正しい助動詞を入れましょう。

基本文: I play the piano.（現在形）

| 意味 | 文 |
|------|------|
| ～できる | I (　　　　　　) play the piano. |
| ～してもよい | I (　　　　　　) go now. |
| ～しなければならない | I (　　　　　　) study hard. |
| ～すべきだ | I (　　　　　　) go to bed early. |
| ～するでしょう | I (　　　　　　) help you. |
| ～かもしれない | I (　　　　　　) be late. |

## 英作文

日本語を英語にしましょう。

| 日本語 | 英語 |
|--------|------|
| 私はピアノを弾くことができる。 | 　　　　　　 |
| 私は行かなければならない。 | 　　　　　　 |
| 私は早く寝るべきだ。 | 　　　　　　 |
| 私は遅れるかもしれない。 | 　　　　　　 |
