# 受動態 - 説明

## 受動態の作り方

**be動詞 + 過去分詞（+ by ～）**

能動態: I play games. → 受動態: Games **are played** by me.

## 時制ごとの受動態

| 時制 | 形 | 例 |
|------|------|------|
| 現在形 | am/is/are + 過去分詞 | Games **are played**. |
| 過去形 | was/were + 過去分詞 | Games **were played**. |
| 未来形 | will be + 過去分詞 | Games **will be played**. |
| 現在完了 | have been + 過去分詞 | Games **have been played**. |

## 助動詞の受動態

**助動詞 + be + 過去分詞**

| 助動詞 | 例 |
|--------|------|
| can | The piano **can be played**. |
| must | The work **must be finished**. |
| should | The room **should be cleaned**. |

## by の省略

行為者が不明・不要な場合、by ～ は省略できる。

- English **is spoken** in many countries.（多くの国で英語が話されている）
- This book **was written** 100 years ago.（この本は100年前に書かれた）
