# 命令文の間接話法ドリル

## 穴埋め問題

( ) に正しい語を入れましょう。

基本文: He said, "Play games."
→ He told me `to play` games.

| 直接話法 | 間接話法 |
|----------|----------|
| He said, "Play games." | He told me (　　　　　　) games. |
| He said, "Don't play games." | He told me (　　　　　　) games. |
| He said, "Study hard." | He told me (　　　　　　) hard. |
| He said, "Don't be late." | He told me (　　　　　　) late. |

## 英作文

日本語を英語にしましょう。

| 日本語 | 英語 |
|--------|------|
| 彼は私にゲームをするように言った。 | 　　　　　　 |
| 彼は私にゲームをするなと言った。 | 　　　　　　 |
