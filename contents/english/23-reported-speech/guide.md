# 間接話法 - 解説

## 現在形との比較

現在形: I **play** games.
間接話法: He said that he **played** games.

- 直接話法を間接話法に変えるとき、**時制を1つ過去にずらす**

## 時制の一致

| 直接話法 | → | 間接話法 |
|----------|---|----------|
| 現在形 (play) | → | 過去形 (played) |
| 現在進行形 (am playing) | → | 過去進行形 (was playing) |
| 過去形 (played) | → | 過去完了 (had played) |
| 現在完了 (have played) | → | 過去完了 (had played) |
| will | → | would |
| can | → | could |

## 疑問文の変換

### Yes/No疑問文

**if / whether** を使い、語順は平叙文にする。

- "Do you play?" → He asked **if** I played.

### 疑問詞付き疑問文

疑問詞はそのまま、語順は平叙文にする。

- "What do you play?" → He asked **what** I played.

## 命令文の変換

**tell + 人 + to + 動詞の原形**

- "Play games." → He **told** me **to play** games.
- "Don't play." → He told me **not to play** games.

## 指示語・時の表現の変化

| 直接話法 | → | 間接話法 |
|----------|---|----------|
| now | → | then |
| today | → | that day |
| tomorrow | → | the next day |
| yesterday | → | the day before |
| here | → | there |
| this | → | that |
