# 助動詞の否定形ドリル

## 穴埋め問題

( ) に正しい否定形を入れましょう。

| 肯定文 | 否定文 |
|--------|--------|
| I can swim. | I (　　　　　　) swim. |
| I must go. | I (　　　　　　) go. |
| I will come. | I (　　　　　　) come. |
| I should stay. | I (　　　　　　) stay. |
| I may leave. | I (　　　　　　) leave. |

## 英作文

日本語を英語にしましょう。

| 日本語 | 英語 |
|--------|------|
| 私は泳ぐことができない。 | 　　　　　　 |
| 私は行ってはいけない。 | 　　　　　　 |
| 私は今日勉強する必要はない。 | 　　　　　　 |
