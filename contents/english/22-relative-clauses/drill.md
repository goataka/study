# 関係代名詞ドリル

## 穴埋め問題

( ) に正しい関係代名詞を入れましょう。

### 主格の関係代名詞

基本文（現在形）: I have a friend. He speaks English.
→ I have a friend **who** speaks English.

| 先行詞 | 文 |
|--------|------|
| 人 | I have a friend (　　　　　　) speaks English. |
| 物 | I have a book (　　　　　　) is very interesting. |
| 人・物 | I have a friend (　　　　　　) lives in Tokyo. |

### 目的格の関係代名詞

基本文: I have a friend. I met him yesterday.
→ I have a friend **whom** I met yesterday.

| 先行詞 | 文 |
|--------|------|
| 人 | I have a friend (　　　　　　) I met yesterday. |
| 物 | I have a book (　　　　　　) I bought yesterday. |

### 所有格の関係代名詞

基本文: I have a friend. His father is a doctor.
→ I have a friend **whose** father is a doctor.

| 文 |
|------|
| I have a friend (　　　　　　) father is a doctor. |
| I have a book (　　　　　　) cover is red. |

### 関係副詞

| 意味 | 文 |
|------|------|
| 場所 | This is the place (　　　　　　) I was born. |
| 時 | I remember the day (　　　　　　) I met you. |
| 理由 | Tell me the reason (　　　　　　) you were late. |
