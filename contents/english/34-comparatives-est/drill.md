# 最上級ドリル

## 穴埋め問題

( ) に正しい最上級を入れましょう。

基本文（原級）: I am tall.

### 短い形容詞

| 原級 | 最上級 |
|------|--------|
| I am tall. | I am the (　　　　　　) in the class. |
| I am big. | I am the (　　　　　　) in the class. |
| I am easy to talk to. | I am the (　　　　　　) to talk to in the class. |

### 長い形容詞

| 原級 | 最上級 |
|------|--------|
| This game is interesting. | This game is the (　　　　　　) of all. |
| This problem is difficult. | This problem is the (　　　　　　) of all. |
| This flower is beautiful. | This flower is the (　　　　　　) of all. |

## 英作文

日本語を英語にしましょう。

| 日本語 | 英語 |
|--------|------|
| 私はクラスで一番背が高い。 | 　　　　　　 |
| このゲームは全部の中で一番面白い。 | 　　　　　　 |
