# I wish ドリル

## 穴埋め問題

( ) に正しい語を入れましょう。

基本文（現在形）: I have time. I speak English.

| パターン | 文 |
|----------|------|
| 現在の願望 | I wish I (　　　　　　) time. |
| 過去の願望 | I wish I (　　　　　　) time. |
| 能力の願望 | I wish I (　　　　　　) speak English. |

| パターン | 文 |
|----------|------|
| 現在の事実 | I am busy. |
| 願望 | I wish I (　　　　　　) busy. |

## 英作文

日本語を英語にしましょう。

| 日本語 | 英語 |
|--------|------|
| 時間があればいいのに。 | 　　　　　　 |
| 英語が話せたらいいのに。 | 　　　　　　 |
| あの時勉強していればよかった。 | 　　　　　　 |
