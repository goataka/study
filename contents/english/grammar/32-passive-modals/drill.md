# 助動詞の受動態ドリル

## 穴埋め問題

( ) に正しい語を入れましょう。

| 能動態 | 受動態 |
|--------|--------|
| I can play games. | Games can be played by me. |
| I must finish the work. | The work (　　　　　　) by me. |
| I should clean the room. | The room (　　　　　　) by me. |
| I will repair the car. | The car (　　　　　　) by me. |
| I may use this room. | This room (　　　　　　) by me. |
