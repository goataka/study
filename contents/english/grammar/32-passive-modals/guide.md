# 助動詞の受動態 - 解説

受動態（現在形）:
Games `are played` by me.

## 助動詞の受動態

Games `can be played` by anyone.

- `助動詞` + `be` + 過去分詞

## 助動詞の受動態パターン

| 助動詞 | 受動態 | 例 |
|--------|--------|------|
| `can` | `can be + 過去分詞` | This `can be done` easily. |
| `must` | `must be + 過去分詞` | This `must be finished` today. |
| `should` | `should be + 過去分詞` | This `should be checked` again. |
| `will` | `will be + 過去分詞` | This `will be sent` tomorrow. |

## 練習問題

( ) に正しい語を入れましょう。

| 能動態 | 受動態 |
|--------|--------|
| I can play games. | Games can be played by me. |
| I must finish the work. | The work (　　　　　　) by me. |
| I should clean the room. | The room (　　　　　　) by me. |
| I will repair the car. | The car (　　　　　　) by me. |
| I may use this room. | This room (　　　　　　) by me. |

## 解答

| 能動態 | 受動態 |
|--------|--------|
| I can play games. | Games can be played by me. |
| I must finish the work. | The work (must be finished) by me. |
| I should clean the room. | The room (should be cleaned) by me. |
| I will repair the car. | The car (will be repaired) by me. |
| I may use this room. | This room (may be used) by me. |

## 英作文

日本語を英語にしましょう。

| 日本語 | 英語 |
|--------|------|
| その部屋は掃除されなければならない。 | 　　　　　　　　　　　　 |
| この部屋は使ってもよい。 | 　　　　　　　　　　　　 |
| その車は修理されるべきだ。 | 　　　　　　　　　　　　 |

## 英作文解答

| 日本語 | 英語 |
|--------|------|
| その部屋は掃除されなければならない。 | The room must be cleaned. |
| この部屋は使ってもよい。 | This room may be used. |
| その車は修理されるべきだ。 | The car should be repaired. |
