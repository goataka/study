# 助動詞の過去形 - 解説

現在形:
I `can play` games.

## 過去形

I `could play` games.

- 一部の助動詞には過去形がある
- 過去の能力・許可・推量を表す

## 過去形パターン

| 助動詞 | 過去形 | 例 |
|--------|--------|------|
| `can` | `could` | I `could play` games when I was young. |
| `will` | `would` | I `would play` games every day. |
| `may` | `might` | I `might play` games tomorrow. |
| `must` | `had to` | I `had to play` this game. |
| `should` | `should have + 過去分詞` | I `should have played` games earlier. |

## 練習問題

( ) に正しい過去形を入れましょう。

| 現在 | 過去 |
|------|------|
| I can play games. | I (　　　　　　) play games. |
| I will help. | I (　　　　　　) help. |
| I must go. | I (　　　　　　) go. |
| I may come. | I (　　　　　　) come. |

## 解答

| 現在 | 過去 |
|------|------|
| I can play games. | I (could) play games. |
| I will help. | I (would) help. |
| I must go. | I (had to) go. |
| I may come. | I (might) come. |

## 英作文

日本語を英語にしましょう。

| 日本語 | 英語 |
|--------|------|
| 私は泳ぐことができた。 | 　　　　　　　　　　　　 |
| 私は手伝うつもりだった。 | 　　　　　　　　　　　　 |
| 私は行かなければならなかった。 | 　　　　　　　　　　　　 |

## 英作文解答

| 日本語 | 英語 |
|--------|------|
| 私は泳ぐことができた。 | I could swim. |
| 私は手伝うつもりだった。 | I would help. |
| 私は行かなければならなかった。 | I had to go. |
