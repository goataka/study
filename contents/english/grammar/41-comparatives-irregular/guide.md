# 不規則な比較変化 - 解説

原級:
This is `good`.

## 比較級・最上級

This is `better` than that.
This is `the best`.

- 一部の形容詞は不規則に変化する

## 不規則変化の一覧

| 原級 | 比較級 | 最上級 |
|------|--------|--------|
| `good` / `well` | `better` | `best` |
| `bad` / `badly` | `worse` | `worst` |
| `many` / `much` | `more` | `most` |
| `little` | `less` | `least` |
| `far` | `farther` / `further` | `farthest` / `furthest` |
