# 最上級 - 解説

原級:
I play games `well`.

## 最上級

I play games `the best` in my class.

- 形容詞・副詞に `-est` をつける（短い語）
- 長い語は `the most` + 原級

## 最上級のパターン

| ルール | 変化 | 例 |
|--------|------|------|
| 短い語 | + est | fast → `fastest` |
| e で終わる | + st | large → `largest` |
| 子音 + y | y → iest | early → `earliest` |
| 短母音 + 子音 | 子音を重ねて + est | big → `biggest` |
| 長い語 | most + 原級 | often → `most often` |

## 解答

### 短い形容詞

| 原級 | 最上級 |
|------|--------|
| This game is hard. | This game is the (hardest) of all. |
| This game is big. | This game is the (biggest) of all. |
| This game is easy. | This game is the (easiest) of all. |

### 長い形容詞

| 原級 | 最上級 |
|------|--------|
| This game is interesting. | This game is the (most interesting) of all. |
| This problem is difficult. | This problem is the (most difficult) of all. |
| This flower is beautiful. | This flower is the (most beautiful) of all. |

## 英作文解答

| 日本語 | 英語 |
|--------|------|
| 私はクラスで一番背が高い。 | I am the tallest in the class. |
| このゲームは全部の中で一番面白い。 | This game is the most interesting of all. |
