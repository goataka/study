# 副詞ドリル

## 穴埋め問題

( ) に正しい副詞を入れましょう。

| パターン | 文 |
|----------|------|
| 頻度 | I always play games. |
| 頻度 | I (　　　　　　) play games. |
| 様態 | I play games (　　　　　　). |
| 頻度 | I (　　　　　　) study English. |
| 様態 | She sings (　　　　　　). |
| 時 | I played games (　　　　　　). |

ヒント: sometimes, well, usually, beautifully, yesterday
