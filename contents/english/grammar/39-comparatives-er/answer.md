# 比較級ドリル - 解答

## 穴埋め問題

### 短い形容詞

| 原級 | 比較級 |
|------|--------|
| This game is hard. | This game is (harder) than that one. |
| This game is big. | This game is (bigger) than that one. |
| This game is easy. | This game is (easier) than that one. |
| It is hot today. | It is (hotter) than yesterday. |

### 長い形容詞

| 原級 | 比較級 |
|------|--------|
| This game is interesting. | This game is (more interesting) than that one. |
| This problem is difficult. | This problem is (more difficult) than that one. |
| This flower is beautiful. | This flower is (more beautiful) than that one. |
