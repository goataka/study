# be動詞の過去形 - 解説

現在形:
The game `is` fun.

## 過去形

The game `was` fun.

- I / he / she / it → `was`
- you / we / they → `were`

## 活用パターン

| 時制 | 主語 | be動詞 | 例 |
|------|------|--------|------|
| 現在形 | I | `am` | I `am` a player. |
| 現在形 | he/she | `is` | The game `is` fun. |
| 過去形 | I/he/she | `was` | The game `was` fun. |
| 過去形 | you/we/they | `were` | Games `were` fun. |

## 練習問題

( ) に正しい語を入れましょう。

| 現在形 | 過去形 |
|--------|--------|
| The game is fun. | The game (　　　　　　) fun. |
| I am a student. | I (　　　　　　) a student. |
| I am tired. | I (　　　　　　) tired. |
| I am at home. | I (　　　　　　) at home. |
| I am hungry. | I (　　　　　　) hungry. |

## 解答

| 現在形 | 過去形 |
|--------|--------|
| The game is fun. | The game (was) fun. |
| I am a student. | I (was) a student. |
| I am tired. | I (was) tired. |
| I am at home. | I (was) at home. |
| I am hungry. | I (was) hungry. |

## 英作文

日本語を英語にしましょう。

| 日本語 | 英語 |
|--------|------|
| 私は幸せでした。 | 　　　　　　　　　　　　 |
| 私は学生でした。 | 　　　　　　　　　　　　 |
| 私は疲れていました。 | 　　　　　　　　　　　　 |

## 英作文解答

| 日本語 | 英語 |
|--------|------|
| 私は幸せでした。 | I was happy. |
| 私は学生でした。 | I was a student. |
| 私は疲れていました。 | I was tired. |
