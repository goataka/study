# 冠詞ドリル

## 穴埋め問題

( ) に `a`、`an`、`the` のいずれかを入れましょう。

| パターン | 文 |
|----------|------|
| 不特定・単数 | I have a book. |
| 母音の前 | I eat (　　　　　　) apple. |
| 特定のもの | I read (　　　　　　) book on the desk. |
| 不特定・単数 | She is (　　　　　　) teacher. |
| 母音の前 | He is (　　　　　　) old man. |
| 特定のもの | (　　　　　　) sun is bright. |
