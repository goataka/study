# 冠詞ドリル

## 穴埋め問題

( ) に `a`、`an`、`the` のいずれかを入れましょう。

| パターン | 文 |
|----------|------|
| 不特定・単数 | I play a game. |
| 母音の前 | I play (　　　　　　) exciting game. |
| 特定のもの | I play (　　　　　　) game every day. |
| 不特定・単数 | She plays (　　　　　　) game. |
| 母音の前 | He plays (　　　　　　) old game. |
| 特定のもの | (　　　　　　) game is fun. |
