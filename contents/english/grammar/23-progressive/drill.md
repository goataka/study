# 進行形ドリル

## 穴埋め問題

( ) に正しい語を入れましょう。

| パターン | 文 |
|----------|------|
| 現在進行形 | I am playing games. |
| 過去進行形 | I (　　　　　　) playing games. |
| 未来進行形 | I (　　　　　　) playing games. |

| パターン | 文 |

| 過去進行形 | I (　　　　　　) studying English. |
| 未来進行形 | I (　　　　　　) studying English. |
