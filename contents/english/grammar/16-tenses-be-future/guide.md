# be動詞の未来形 - 解説

現在形:
The game `is` fun.

## 未来形

The game `will be` fun.

- `will be` + 形容詞/名詞で未来を表す
- 主語に関係なく `will be` の形

## 活用パターン

| 時制 | be動詞 | 例 |
|------|--------|------|
| 現在形 | am/is/are | The game `is` fun. |
| 過去形 | was/were | The game `was` fun. |
| 未来形 | will be | The game `will be` fun. |
