# 前置詞ドリル

## 穴埋め問題

( ) に正しい前置詞を入れましょう。

| パターン | 文 |
|----------|------|
| 場所 | I play games at home. |
| 場所 | I play games (　　　　　　) the park. |
| 時 | I play games (　　　　　　) Sunday. |
| 時 | I play games (　　　　　　) the morning. |
| 方向 | I go (　　　　　　) school. |
| 手段 | I go to school (　　　　　　) bus. |
