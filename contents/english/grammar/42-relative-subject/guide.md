# 主格の関係代名詞 - 解説

2つの文:
I know a boy. He plays games.

## 主格の関係代名詞

I know a boy `who` plays games.

- 先行詞（人）→ `who`
- 先行詞（物）→ `which`
- 先行詞（人・物）→ `that`

## パターン

| 先行詞 | 関係代名詞 | 例 |
|--------|-----------|------|
| 人 | `who` / `that` | a boy `who` plays games |
| 物 | `which` / `that` | a game `which` is fun |

## 練習問題

( ) に正しい関係代名詞を入れましょう。

2文を1文にする例:
I have a friend. He speaks English.
→ I have a friend `who` speaks English.

| 先行詞 | 文 |
|--------|------|
| 人 | I have a friend (　　　　　　) speaks English. |
| 物 | I have a book (　　　　　　) is very interesting. |
| 人・物 | I have a friend (　　　　　　) lives in Tokyo. |
| 物 | I have a cat (　　　　　　) is very cute. |

## 解答

| 先行詞 | 文 |
|--------|------|
| 人 | I have a friend (who) speaks English. |
| 物 | I have a book (which) is very interesting. |
| 人・物 | I have a friend (that) lives in Tokyo. |
| 物 | I have a cat (which / that) is very cute. |

## 英作文

日本語を英語にしましょう。

| 日本語 | 英語 |
|--------|------|
| 私には英語を話す友達がいる。 | 　　　　　　　　　　　　 |
| 私にはとても面白い本がある。 | 　　　　　　　　　　　　 |

## 英作文解答

| 日本語 | 英語 |
|--------|------|
| 私には英語を話す友達がいる。 | I have a friend who speaks English. |
| 私にはとても面白い本がある。 | I have a book which is very interesting. |
