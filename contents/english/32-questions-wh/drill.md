# 疑問詞ドリル

## 穴埋め問題

( ) に正しい疑問詞を入れましょう。

基本文: I play games at home every day because it's fun.

| 疑問詞 | 疑問文 |
|--------|--------|
| 何を？ | (　　　　　　) do I play? |
| どこで？ | (　　　　　　) do I play games? |
| いつ？ | (　　　　　　) do I play games? |
| なぜ？ | (　　　　　　) do I play games? |
| どのように？ | (　　　　　　) do I play games? |
