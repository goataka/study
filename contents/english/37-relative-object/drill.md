# 目的格の関係代名詞ドリル

## 穴埋め問題

( ) に正しい関係代名詞を入れましょう。

2文を1文にする例:
I have a friend. I met him yesterday.
→ I have a friend `whom` I met yesterday.

| 先行詞 | 文 |
|--------|------|
| 人 | I have a friend (　　　　　　) I met yesterday. |
| 物 | I have a book (　　　　　　) I bought yesterday. |
| 物 | I have a pen (　　　　　　) I use every day. |

## 英作文

日本語を英語にしましょう。

| 日本語 | 英語 |
|--------|------|
| 私は昨日買った本を持っている。 | 　　　　　　 |
| 私は毎日使うペンを持っている。 | 　　　　　　 |
