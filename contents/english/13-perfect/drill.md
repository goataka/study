# 完了形ドリル

## 穴埋め問題

( ) に正しい語を入れましょう。

### have/hadの変化

| パターン | 文 |
|----------|------|
| 現在完了 | I (　　　　　　) played games. |
| 過去完了 | I (　　　　　　) played games. |
| 未来完了 | I (　　　　　　) played games. |

### 過去分詞の変化

| 原形 | 文 |
|------|------|
| play | I have (　　　　　　) games. |
| go | I have (　　　　　　) to school. |
| eat | I have (　　　　　　) breakfast. |
| write | I have (　　　　　　) a letter. |
| see | I have (　　　　　　) the movie. |
| read | I have (　　　　　　) the book. |
| take | I have (　　　　　　) the bus. |
| make | I have (　　　　　　) a cake. |

### 完了形の用法

| 用法 | 文 |
|------|------|
| 経験 | I have (　　　　　　) to Tokyo before. |
| 継続 | I have (　　　　　　) here for three years. |
| 完了 | I have just (　　　　　　) my homework. |
| 結果 | I have (　　　　　　) my key. |
