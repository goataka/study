# 受動態ドリル

## 穴埋め問題

( ) に正しい語を入れましょう。

### 受動態の時制変化

| パターン | 能動態 | 受動態 |
|----------|--------|--------|
| 現在形 | I play games. | Games are played by me. |
| 過去形 | I played games. | Games (　　　　　　) by me. |
| 未来形 | I will play games. | Games (　　　　　　) by me. |
| 現在完了 | I have played games. | Games (　　　　　　) by me. |

### 能動態→受動態

| 能動態 | 受動態 |
|--------|--------|
| I play games. | Games are played by me. |
| I wrote a letter. | A letter (　　　　　　) by me. |
| I will make a cake. | A cake (　　　　　　) by me. |
| I have read the book. | The book (　　　　　　) by me. |
