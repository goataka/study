# 進行形ドリル

## 穴埋め問題

( ) に正しい語を入れましょう。

### be動詞の変化

| パターン | 文 |
|----------|------|
| 現在進行形 | I am playing games. |
| 過去進行形 | I (　　　　　　) playing games. |
| 未来進行形 | I (　　　　　　) playing games. |

### -ing形の作り方

| 原形 | 文 |
|------|------|
| play → playing | I am playing games. |
| study | I am (　　　　　　) English. |
| run | I am (　　　　　　) fast. |
| swim | I am (　　　　　　) in the pool. |
| write | I am (　　　　　　) a letter. |
| make | I am (　　　　　　) a cake. |
| sit | I am (　　　　　　) on the chair. |
| dance | I am (　　　　　　) now. |

## 英作文

日本語を英語にしましょう。

| 日本語 | 英語 |
|--------|------|
| 私はゲームをしています。 | 　　　　　　　　　　　　 |
| 私はゲームをしていました。 | 　　　　　　　　　　　　 |
| 私は英語を勉強しています。 | 　　　　　　　　　　　　 |
| 私は手紙を書いています。 | 　　　　　　　　　　　　 |
| 私はプールで泳いでいます。 | 　　　　　　　　　　　　 |
