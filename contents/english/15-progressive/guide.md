# 進行形 - 解説

## 現在形との比較

現在形: I **play** games.（習慣）
現在進行形: I **am playing** games.（今まさにしている）

- 現在形は「いつもする」、進行形は「今している」

## 進行形の作り方

**be動詞 + 動詞の-ing形**

| 時制 | be動詞 | 例 |
|------|--------|------|
| 現在進行形 | am | I **am playing** games. |
| 過去進行形 | was | I **was playing** games. |
| 未来進行形 | will be | I **will be playing** games. |

## -ing形の作り方ルール

| ルール | 例 |
|--------|------|
| そのまま -ing | play → playing, study → studying |
| eを取って -ing | write → writing, make → making, dance → dancing |
| 子音を重ねて -ing | run → running, swim → swimming, sit → sitting |
