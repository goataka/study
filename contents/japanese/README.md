# 国語ドリル

[ドリル一覧へ戻る](../README.md)

## 学習の流れ

国語学習は「漢字」→「ことわざ・四字熟語」の順に進みます。

### 📖 漢字

小学1年生から高校生までの漢字を学年別に扱います。各漢字の読み方（訓読み・音読み）を練習します。

[漢字学習ガイドを見る](kanji/README.md)
