# ベクトル - 解答

**参考学年**: 高校2年

## ベクトルの計算

| 問題 | 答え |
|------|------|
| →a + →b = | (6, 2) |
| →a - →b = | (-2, 4) |
| 3→a = | (6, 9) |
| →a · →b（内積）= | 2×4 + 3×(-1) = 5 |
| \|→a\|（大きさ）= | √(4+9) = √13 |

## 応用

| 問題 | 答え |
|------|------|
| →a = (1, 2), →b = (3, 4) のとき →a · →b = | 1×3 + 2×4 = 11 |
| →a = (3, 4) の大きさ \|→a\| = | √(9+16) = 5 |
| →a = (1, 0), →b = (0, 1) のとき →a · →b = | 0（直交） |

💡 **ポイント**: 内積が0のとき、2つのベクトルは直交（垂直）です。
