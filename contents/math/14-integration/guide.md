# 積分

> **参考学年**: 高校3年

## 学習目的

不定積分の概念を理解し、多項式関数の積分と定積分の計算ができるようになる。

## 解説

### 積分とは

微分の逆の操作です。導関数から元の関数を求めます。

### 不定積分の公式

- **∫xⁿdx = (1/(n+1))xⁿ⁺¹ + C**（n ≠ -1）
- C は積分定数（任意の定数）
- ∫{f(x) + g(x)}dx = ∫f(x)dx + ∫g(x)dx

### 計算例

- ∫(3x² + 2x)dx
- = 3 × (1/3)x³ + 2 × (1/2)x² + C
- = x³ + x² + C

### 定積分

定積分は「面積」を求める計算です。

**∫[a→b] f(x)dx = F(b) - F(a)**

- F(x) は f(x) の不定積分（原始関数）
- 例：∫[0→2] (3x² + 2x)dx = [x³ + x²]₀² = (8 + 4) - (0 + 0) = 12
