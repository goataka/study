# 積分 - 解答

> **参考学年**: 高校3年

## 不定積分

| 問題 | 答え |
|------|------|
| ∫(3x² + 2x)dx = | x³ + x² + C |
| ∫(2x³ - 4x)dx = | (1/2)x⁴ - 2x² + C |
| ∫(x² + 3x + 2)dx = | (1/3)x³ + (3/2)x² + 2x + C |
| ∫(4x³ + 6x²)dx = | x⁴ + 2x³ + C |

> **解説**: ∫xⁿdx = (1/(n+1))xⁿ⁺¹ + C を各項に適用します。例えば ∫3x²dx = 3 × (1/3)x³ = x³。積分定数 C を忘れずにつけましょう。

## 定積分

| 問題 | 答え |
|------|------|
| ∫[0→2] (3x² + 2x)dx = | 12 |
| ∫[1→3] (2x + 1)dx = | 10 |
| ∫[0→1] (x² + x)dx = | 5/6 |

> **解説**: まず不定積分を求め、上端と下端の値の差を計算します。∫[0→2] (3x² + 2x)dx = [x³ + x²]₀² = (8 + 4) - (0 + 0) = 12。∫[0→1] (x² + x)dx = [(1/3)x³ + (1/2)x²]₀¹ = (1/3 + 1/2) - 0 = 5/6 です。
