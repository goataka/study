# 中学1年生 - 正負の数（解答）

## 正負の数の計算

| 問題 | 答え |
|------|------|
| (+5) + (+3) = | +8 |
| (+7) + (-4) = | +3 |
| (-6) + (+2) = | -4 |
| (-5) + (-3) = | -8 |
| (+8) - (+5) = | +3 |
| (+6) - (-2) = | +8 |
| (-4) - (+7) = | -11 |
| (-9) - (-3) = | -6 |

## 正負の数のかけ算・わり算

| 問題 | 答え |
|------|------|
| (+3) × (+4) = | +12 |
| (+6) × (-2) = | -12 |
| (-5) × (+3) = | -15 |
| (-4) × (-7) = | +28 |
| (+12) ÷ (+3) = | +4 |
| (+15) ÷ (-5) = | -3 |
| (-18) ÷ (+6) = | -3 |
| (-20) ÷ (-4) = | +5 |
