# 分数のわり算 - 解説

**参考学年**: 小学6年 | **学習目的**: 分数のわり算を逆数を使って正確に計算できるようにする

## ✏️ 分数のわり算

- わる数を **ひっくり返す**（逆数にする）
- そのあと **かけ算** をする
- 公式: **a/b ÷ c/d = a/b × d/c**

## 📖 例題と解説

### 例① 1/2 ÷ 1/3 = ?

```
ステップ１: わる数（1/3）をひっくり返す → 3/1
ステップ２: かけ算にする
  1/2 × 3/1 = 3/2 = 1と1/2 ✅
```

### 例② 5/6 ÷ 5/9 = ?

```
ステップ１: 5/9 → 9/5
ステップ２: 5/6 × 9/5
  約分: 5と5 → 1と1
  1/6 × 9/1 = 9/6 = 3/2 ✅
```

## 📌 逆数とは

- 分子と分母を入れかえた数
- 2/3 の逆数 → **3/2**
- 4 の逆数 → **1/4**（4 = 4/1 だから）
- ある数 × その逆数 = **1** になる
