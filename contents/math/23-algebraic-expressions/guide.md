# 文字式 - 解説

**参考学年**: 中学1年 | **学習目的**: 文字を使った式の表し方と計算のルールを身につける

## ✏️ 文字式のルール

- かけ算の記号 × は **省略** する → a × 3 = **3a**
- **数字を文字の前** に書く → a × 3 = 3a（a3 ではない）
- 1は省略 → 1 × x = **x**
- わり算は **分数** で書く → x ÷ 3 = **x/3**
- 同じ文字のかけ算 → a × a = **a²**

## 📖 例題と解説

### 例① 3a + 5a = ?

```
🔵🔵🔵 ＋ 🔵🔵🔵🔵🔵 ＝ 🔵🔵🔵🔵🔵🔵🔵🔵
  3a   ＋    5a     ＝     8a
```

🎯 同じ文字「a」→ 係数（前の数字）をたす → 3 + 5 = 8 → **8a**

### 例② 4a + 3b + 2a - b = ?

```
aの仲間: 4a + 2a = 6a
bの仲間: 3b - b = 2b
答え: 6a + 2b ✅
```

## 📌 おぼえておこう

- **同類項**（同じ文字の項）をまとめる
- 違う文字はまとめられない（3a + 2b はそのまま）
- 分配法則: a(b + c) = ab + ac
