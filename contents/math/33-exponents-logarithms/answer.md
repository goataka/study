# 指数・対数 - 解答

**参考学年**: 高校2年

## 指数の計算

| 問題 | 答え |
|------|------|
| 2³ × 2⁴ = | 2⁷ = 128 |
| 3⁵ ÷ 3² = | 3³ = 27 |
| (2³)² = | 2⁶ = 64 |
| 4^(1/2) = | √4 = 2 |
| 8^(2/3) = | (∛8)² = 2² = 4 |

## 対数の計算

| 問題 | 答え |
|------|------|
| log₂8 = | 3 |
| log₃27 = | 3 |
| log₁₀100 = | 2 |
| log₂4 + log₂8 = | 2 + 3 = 5 |
| log₃9 - log₃3 = | 2 - 1 = 1 |

💡 **ポイント**: log_a(b) は「aを何乗したらbになるか」。対数の性質を使うと計算がラクに。
