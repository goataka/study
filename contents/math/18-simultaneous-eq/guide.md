# 連立方程式 - 解説

**参考学年**: 中学2年 | **学習目的**: 加減法や代入法を使って連立方程式を解けるようにする

## ✏️ 連立方程式とは

- 2つの方程式を **同時に** 満たす解を求める
- 未知数が2つ → 方程式も **2つ必要**

## 📖 例題と解説

### 例① 加減法

```
  x + y = 5  …①
  x - y = 1  …②

① + ②:  2x = 6  →  x = 3
x=3を①に代入: 3 + y = 5  →  y = 2
```

🎯 **x = 3, y = 2**

### 例② 代入法

```
  x + y = 5  →  y = 5 - x  …①'
  ①'を②に代入:
  x - (5 - x) = 1
  2x - 5 = 1
  x = 3, y = 2
```

🎯 同じ答え！方法が違うだけ

## 📌 文章題の手順

1. 求めたいものを **x, y** とおく
2. 条件から **2つの式** をつくる
3. 連立方程式を **解く**
