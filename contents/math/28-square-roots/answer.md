# 平方根 - 解答

**参考学年**: 中学3年

## 平方根を求めましょう

| 問題 | 答え |
|------|------|
| √4 = | 2 |
| √9 = | 3 |
| √25 = | 5 |
| √49 = | 7 |
| √100 = | 10 |

## 平方根の計算

| 問題 | 答え |
|------|------|
| √2 × √3 = | √6 |
| √8 × √2 = | √16 = 4 |
| √12 ÷ √3 = | √4 = 2 |
| √18 を簡単にする | 3√2 |
| √50 を簡単にする | 5√2 |
| 2√3 + 5√3 = | 7√3 |
| √12 + √27 = | 2√3 + 3√3 = 5√3 |

💡 **ポイント**: √の中を素因数分解して、ペアになった数を外に出します。
