# 連立方程式 - 解答

**参考学年**: 中学2年

## 連立方程式を解く

| 問題 | x = | y = |
|------|------|------|
| x + y = 5, x - y = 1 | 3 | 2 |
| 2x + y = 7, x + y = 4 | 3 | 1 |
| x + 2y = 8, x - y = 2 | 4 | 2 |
| 3x + y = 11, x + y = 5 | 3 | 2 |
| 2x + 3y = 13, x + 2y = 8 | 2 | 3 |

💡 加減法: 2式をたす/ひくことで1つの文字を消す。`x+y=5` と `x-y=1` を足す → `2x=6` → `x=3`、`y=2`

## 文章題

| 問題 | 答え |
|------|------|
| りんご1個とみかん1個で180円、りんご2個とみかん1個で260円。りんご1個は？ | 80円 |
| 同じ条件で、みかん1個は？ | 100円 |

💡 りんご=x、みかん=y → `x+y=180`, `2x+y=260`。②-① → `x=80`、`y=100`
