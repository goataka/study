# 正負の数 - 解答

> **参考学年**: 中学1年

## たし算・ひき算

| 問題 | 答え |
|------|------|
| (+5) + (+3) = | +8 |
| (+7) + (-4) = | +3 |
| (-6) + (+2) = | -4 |
| (-5) + (-3) = | -8 |
| (+8) - (+5) = | +3 |
| (+6) - (-2) = | +8 |
| (-4) - (+7) = | -11 |
| (-9) - (-3) = | -6 |

> 💡 ひき算は「符号を変えてたし算」に。`(+6) - (-2) = (+6) + (+2) = +8`

## かけ算・わり算

| 問題 | 答え |
|------|------|
| (+3) × (+4) = | +12 |
| (+6) × (-2) = | -12 |
| (-5) × (+3) = | -15 |
| (-4) × (-7) = | +28 |
| (+12) ÷ (+3) = | +4 |
| (+15) ÷ (-5) = | -3 |
| (-18) ÷ (+6) = | -3 |
| (-20) ÷ (-4) = | +5 |

> 💡 同符号 → ＋（正）、異符号 → −（負）。`(-4) × (-7)` → マイナス×マイナス = **＋28**
