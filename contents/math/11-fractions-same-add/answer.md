# 分数のたし算（同分母） - 解答

**参考学年**: 小学4年

| 問題 | 答え |
|------|------|
| 1/4 + 2/4 = | 3/4 |
| 1/3 + 1/3 = | 2/3 |
| 2/5 + 1/5 = | 3/5 |
| 3/8 + 2/8 = | 5/8 |
| 1/6 + 4/6 = | 5/6 |
| 2/7 + 3/7 = | 5/7 |
| 1/9 + 5/9 = | 6/9 = 2/3 |
| 3/10 + 4/10 = | 7/10 |
| 2/5 + 2/5 = | 4/5 |
| 1/8 + 6/8 = | 7/8 |

💡 **ポイント**: 分母はそのまま、分子だけをたします。約分できるときは約分しましょう
