# 分数のたし算（同分母） - 解説

**参考学年**: 小学4年 | **学習目的**: 分母が同じ分数のたし算を正確に計算できるようにする

## ✏️ 同じ分母のたし算

- 分母が同じなら、**分子だけをたす**
- 分母はそのまま変えない
- 答えが約分できるときは約分する

## 📖 例題と解説

### 例① 1/4 + 2/4 = ?

```
🍕  ＋  🍕🍕  ＝  🍕🍕🍕
1/4    2/4      3/4
```

🎯 分子をたす → 1 + 2 = 3 → **3/4**

### 例② 1/9 + 5/9 = ?

- 分子をたす → 1 + 5 = 6 → 6/9
- 約分 → 6÷3 / 9÷3 = **2/3** ✅

## 📌 約分のやり方

- 分子と分母を **同じ数でわる**
- 例: 6/9 → 6÷3 / 9÷3 = **2/3**
