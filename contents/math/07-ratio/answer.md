# 比と割合 - 解答

> **参考学年**: 小学6年

## 比の計算

| 問題 | 答え |
|------|------|
| 4:6を最も簡単な整数の比にしましょう | 2:3 |
| 9:12を最も簡単な整数の比にしましょう | 3:4 |
| 15:20を最も簡単な整数の比にしましょう | 3:4 |
| 8:12を最も簡単な整数の比にしましょう | 2:3 |

> **解説**: 比を簡単にするには、両方の数を最大公約数でわります。例えば 4:6 は両方を2でわって 2:3 です。

## 割合の計算

| 問題 | 答え |
|------|------|
| 200の30%はいくつですか | 60 |
| 150の20%はいくつですか | 30 |
| 80の25%はいくつですか | 20 |
| 300の15%はいくつですか | 45 |

> **解説**: 「もとの数 × 割合」で求めます。30%は0.3なので、200 × 0.3 = 60 です。25%は1/4なので、80 ÷ 4 = 20 とも計算できます。

## 速さの計算

| 問題 | 答え |
|------|------|
| 時速60kmで2時間走ったときの距離は？ | 120km |
| 120kmの道のりを3時間で走ったときの速さは？ | 時速40km |
| 時速40kmで100km走るのにかかる時間は？ | 2.5時間 |

> **解説**: 「距離 = 速さ × 時間」「速さ = 距離 ÷ 時間」「時間 = 距離 ÷ 速さ」の3つの公式を使い分けます。
