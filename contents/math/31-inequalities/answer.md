# 不等式 - 解答

**参考学年**: 高校1年

| 問題 | 答え |
|------|------|
| 2x + 3 > 7 | x > 2 |
| 3x - 1 ≤ 8 | x ≤ 3 |
| -2x + 4 > 0 | x < 2 |
| 5x + 2 ≥ 3x + 10 | x ≥ 4 |
| 4(x - 1) < 2x + 6 | x < 5 |
| -3x + 9 ≤ 0 | x ≥ 3 |
| 1 < 2x + 3 ≤ 9 | -1 < x ≤ 3 |
| x/2 + 1 > 3 | x > 4 |

💡 **ポイント**: 方程式と同じ解き方。ただし負の数でかける・わるときは不等号の向きが逆転します。
