# 一次方程式 - 解説

**参考学年**: 中学1年 | **学習目的**: 一次方程式を移項を使って正確に解けるようにする

## ✏️ 方程式とは

- **=（イコール）** でつながった式
- 未知の数 **x** の値を求める

## 📖 例題と解説

### 例① x + 3 = 7

```
x + 3 = 7
x = 7 - 3  ← +3を移項すると-3
x = 4 ✅
```

### 例② 3x + 4 = 13

```
3x + 4 = 13
3x = 13 - 4  ← +4を移項
3x = 9
x = 9 ÷ 3   ← ×3を移項すると÷3
x = 3 ✅
```

### 例③ 2x + 1 = x + 5

```
2x + 1 = x + 5
2x - x = 5 - 1  ← xを左、数字を右に移項
x = 4 ✅
```

## 📌 移項のルール

- 左辺から右辺（またはその逆）に項を移すこと
- 移項すると **符号が逆** になる
  - +3 → **-3**
  - -5 → **+5**
