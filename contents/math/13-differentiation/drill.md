# 微分 - 練習問題

> **参考学年**: 高校3年

> **学習目的**: 多項式関数の導関数を求め、極値を計算できるようにする。

## 導関数を求める

次の関数を微分しましょう。

| 問題 | 導関数 f'(x) |
|------|------|
| f(x) = 3x² + 2x + 1 | 　　　　　　　　　　　　 |
| f(x) = x³ - 4x² + 5x | 　　　　　　　　　　　　 |
| f(x) = 2x⁴ - 3x³ + x | 　　　　　　　　　　　　 |
| f(x) = 5x³ + 2x² - 7 | 　　　　　　　　　　　　 |

## 極値を求める

次の関数の極値を求めましょう。

| 問題 | 極大値 | 極小値 |
|------|------|------|
| f(x) = x³ - 3x² + 4 | 　　　　　　　　　　　　 | 　　　　　　　　　　　　 |
| f(x) = -x³ + 3x² + 1 | 　　　　　　　　　　　　 | 　　　　　　　　　　　　 |
