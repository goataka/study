# 微分 - 解答

> **参考学年**: 高校3年

## 導関数を求める

| 問題 | 導関数 f'(x) |
|------|------|
| f(x) = 3x² + 2x + 1 | 6x + 2 |
| f(x) = x³ - 4x² + 5x | 3x² - 8x + 5 |
| f(x) = 2x⁴ - 3x³ + x | 8x³ - 9x² + 1 |
| f(x) = 5x³ + 2x² - 7 | 15x² + 4x |

> **解説**: (xⁿ)' = n・xⁿ⁻¹ を各項に適用します。例えば 3x² → 3×2×x¹ = 6x、2x → 2×1×x⁰ = 2、定数1 → 0 です。

## 極値を求める

| 問題 | 極大値 | 極小値 |
|------|------|------|
| f(x) = x³ - 3x² + 4 | 4 (x=0) | 0 (x=2) |
| f(x) = -x³ + 3x² + 1 | 5 (x=2) | 1 (x=0) |

> **解説**: f(x) = x³ - 3x² + 4 の場合、f'(x) = 3x² - 6x = 3x(x-2) = 0 より x = 0, 2。f'(x) の符号変化を調べると、x=0 で + → -（極大）、x=2 で - → +（極小）。f(0) = 4（極大値）、f(2) = 8 - 12 + 4 = 0（極小値）です。
