# 微分

> **参考学年**: 高校3年

## 学習目的

微分の概念を理解し、多項式関数の導関数を求めて極値を計算できるようになる。

## 解説

### 微分とは

関数の変化の割合（傾き）を求めることです。f(x) の導関数を f'(x) と書きます。

### 微分の公式

- **(xⁿ)' = n・xⁿ⁻¹**
- (定数)' = 0
- {f(x) + g(x)}' = f'(x) + g'(x)
- {a・f(x)}' = a・f'(x)

### 計算例

- f(x) = 3x² + 2x + 1
- f'(x) = 6x + 2（各項を個別に微分）

### 極値

f'(x) = 0 となる x での f(x) の値が極値です。

- f'(x) の符号が + → - に変わる点 → **極大値**
- f'(x) の符号が - → + に変わる点 → **極小値**

### 極値の求め方

1. f'(x) を求める
2. f'(x) = 0 を解く
3. f'(x) の符号の変化を調べる
4. 元の関数に代入して極値を求める
