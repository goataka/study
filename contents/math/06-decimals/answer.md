# 小数 - 解答

> **参考学年**: 小学5年

## 小数のたし算

| 問題 | 答え |
|------|------|
| 0.3 + 0.4 = | 0.7 |
| 1.5 + 2.3 = | 3.8 |
| 2.6 + 1.8 = | 4.4 |
| 3.25 + 1.5 = | 4.75 |
| 4.7 + 2.85 = | 7.55 |
| 6.15 + 3.7 = | 9.85 |

> **解説**: 小数点の位置をそろえて計算します。けた数が違うときは、足りないところに0を補って考えましょう。例えば 3.25 + 1.5 は 3.25 + 1.50 = 4.75 です。

## 小数のひき算

| 問題 | 答え |
|------|------|
| 0.8 - 0.3 = | 0.5 |
| 3.5 - 1.2 = | 2.3 |
| 5.7 - 2.9 = | 2.8 |
| 4.6 - 1.75 = | 2.85 |
| 7.3 - 3.85 = | 3.45 |
| 8.5 - 4.25 = | 4.25 |

> **解説**: たし算と同じく小数点をそろえます。4.6 - 1.75 は 4.60 - 1.75 = 2.85 のように、0を補ってから計算します。

## 小数のかけ算

| 問題 | 答え |
|------|------|
| 0.5 × 2 = | 1.0 |
| 1.2 × 3 = | 3.6 |
| 2.5 × 4 = | 10.0 |
| 0.8 × 0.5 = | 0.40 |

> **解説**: 整数として計算してから、小数点以下のけた数の合計だけ小数点を左にずらします。0.8 × 0.5 は 8 × 5 = 40 で、小数点以下が合計2けたなので 0.40 です。
