# 極限 - 解答

**参考学年**: 高校3年

## 数列の極限

| 問題 | 答え |
|------|------|
| lim(n→∞) 1/n = | 0 |
| lim(n→∞) (2n+1)/n = | 2 |
| lim(n→∞) (3n²+1)/(n²+2) = | 3 |
| lim(n→∞) n/(n²+1) = | 0 |

## 関数の極限

| 問題 | 答え |
|------|------|
| lim(x→2) (x²-4)/(x-2) = | 4 |
| lim(x→0) sin(x)/x = | 1 |
| lim(x→1) (x²-1)/(x-1) = | 2 |
| lim(x→∞) (2x+1)/(3x-1) = | 2/3 |

💡 **ポイント**: 分数式は最高次の項で割ると極限が見えます。0/0の不定形は因数分解で解消
