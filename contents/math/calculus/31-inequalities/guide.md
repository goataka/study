# 不等式 - 解説

**参考学年**: 高校1年 | **学習目的**: 一次不等式を正確に解き、数直線上に表せるようにする

## ✏️ 不等式とは

- **>**（大きい）、**<**（小さい）、**≥**（以上）、**≤**（以下）を使った式
- 方程式と同じように移項して解く

## 📖 例題と解説

### 例① 2x + 3 > 7

```
2x + 3 > 7
2x > 7 - 3   ← 移項
2x > 4
x > 2 ✅

数直線: ──────(2)======→
        2は含まない（>）
```

### 例② -2x + 4 > 0

```
-2x + 4 > 0
-2x > -4
x < 2 ✅  ← ⚠️ 負の数でわると不等号が逆転！
```

## 📌 重要ルール ⚠️

- 負の数をかける・わるとき → **不等号の向きが逆** になる
- 正の数なら向きはそのまま
