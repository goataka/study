# 二次関数

## ✏️ 二次関数とは

- **y = ax² + bx + c** の形
- グラフは **放物線** 🪂

## 📖 例題と解説

### 例① y = x² - 4x + 5 の最小値

```text
平方完成:
y = x² - 4x + 5
  = (x² - 4x + 4) + 1
  = (x - 2)² + 1

→ 頂点 (2, 1)
→ a=1 > 0 → 下に凸 ∪
→ 最小値 1（x=2のとき）✅
```

### 例② y = -x² + 6x - 5 の最大値

```text
y = -(x² - 6x) - 5
  = -(x² - 6x + 9) + 9 - 5
  = -(x - 3)² + 4

→ 頂点 (3, 4)
→ a=-1 < 0 → 上に凸 ∩
→ 最大値 4（x=3のとき）✅
```

## 📌 標準形のポイント

| 項目 | 見方 |
|------|------|
| 頂点 | y = a(**x - p**)² + **q** → (**p**, **q**) |
| 軸 | x = **p** |
| a > 0 | 下に凸 ∪ → **最小値**あり |
| a < 0 | 上に凸 ∩ → **最大値**あり |

## 一覧

| 問題 | 頂点 | 軸 |
|------|------|------|
| y = (x - 2)² + 1 | (2, 1) | x = 2 |
| y = (x + 3)² - 2 | (-3, -2) | x = -3 |
| y = 2(x - 1)² + 3 | (1, 3) | x = 1 |
| y = -(x + 2)² + 4 | (-2, 4) | x = -2 |

💡 `y = a(x-p)² + q` → 頂点 **(p, q)**、軸 **x = p**。`(x+3)² = (x-(-3))²` なので p=-3

## 最大値・最小値

| 問題 | 最大値/最小値 | そのときのx |
|------|------|------|
| y = x² - 4x + 5 | 最小値 1 | x = 2 |
| y = -x² + 6x - 5 | 最大値 4 | x = 3 |
| y = 2x² - 8x + 10 | 最小値 2 | x = 2 |

💡 平方完成する。`x²-4x+5 = (x-2)²+1` → 下に凸なので最小値 **1**（x=2）
