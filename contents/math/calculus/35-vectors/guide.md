# ベクトル - 解説

**参考学年**: 高校2年 | **学習目的**: ベクトルの基本演算と内積の計算ができるようにする

## ✏️ ベクトルとは

- **大きさ** と **向き** を持つ量
- 矢印 →a で表す
- 成分表示: →a = (x, y)

## 📖 例題と解説

### 例① →a = (2, 3), →b = (4, -1) のたし算

```text
→a + →b = (2+4, 3+(-1)) = (6, 2) ✅
```

🎯 成分ごとにたす

### 例② 内積 →a · →b

```text
→a · →b = 2×4 + 3×(-1) = 8 - 3 = 5 ✅
```

🎯 対応する成分をかけてたす

### 例③ 大きさ |→a|

```text
|→a| = √(2² + 3²) = √(4 + 9) = √13 ✅
```

## 📌 公式まとめ

- たし算: (a₁, a₂) + (b₁, b₂) = **(a₁+b₁, a₂+b₂)**
- 内積: →a · →b = **a₁b₁ + a₂b₂**
- 大きさ: |→a| = **√(a₁² + a₂²)**
- 内積 = 0 → **直交（垂直）** ⊥

## 解答

| 問題 | 答え |
|------|------|
| →a + →b = | (6, 2) |
| →a - →b = | (-2, 4) |
| 3→a = | (6, 9) |
| →a · →b（内積）= | 2×4 + 3×(-1) = 5 |
| \|→a\|（大きさ）= | √(4+9) = √13 |

## 応用

| 問題 | 答え |
|------|------|
| →a = (1, 2), →b = (3, 4) のとき →a · →b = | 1×3 + 2×4 = 11 |
| →a = (3, 4) の大きさ \|→a\| = | √(9+16) = 5 |
| →a = (1, 0), →b = (0, 1) のとき →a · →b = | 0（直交） |

💡 **ポイント**: 内積が0のとき、2つのベクトルは直交（垂直）です
