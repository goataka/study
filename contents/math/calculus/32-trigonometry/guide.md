# 三角関数 - 解説

**参考学年**: 高校2年 | **学習目的**: 三角関数の基本値を覚え、相互関係と加法定理を使った計算ができるようにする

## ✏️ 三角関数とは

- 直角三角形の辺の比から定義

```
        /|
 斜辺 / |  対辺
     /  |
    /θ__|
     隣辺
```

- **sin θ** = 対辺 / 斜辺
- **cos θ** = 隣辺 / 斜辺
- **tan θ** = 対辺 / 隣辺

## 📖 基本角の値

| 角度 | sin | cos | tan |
|------|-----|-----|-----|
| 0° | 0 | 1 | 0 |
| 30° | 1/2 | √3/2 | 1/√3 |
| 45° | √2/2 | √2/2 | 1 |
| 60° | √3/2 | 1/2 | √3 |
| 90° | 1 | 0 | ─ |

## 📖 例題と解説

### 例① sin θ = 3/5 のとき cos θ = ?

```
sin²θ + cos²θ = 1
(3/5)² + cos²θ = 1
9/25 + cos²θ = 1
cos²θ = 16/25
cos θ = 4/5 ✅ (0°<θ<90°なので正)
```

## 📌 加法定理

- sin(α ± β) = sin α cos β **±** cos α sin β
- cos(α ± β) = cos α cos β **∓** sin α sin β
