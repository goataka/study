# 連立方程式 - 解答

> **参考学年**: 中学2年

## 連立方程式を解く

| 問題 | x = | y = |
|------|------|------|
| x + y = 5<br>x - y = 1 | 3 | 2 |
| 2x + y = 7<br>x + y = 4 | 3 | 1 |
| x + 2y = 8<br>x - y = 2 | 4 | 2 |
| 3x + y = 11<br>x + y = 5 | 3 | 2 |
| 2x + 3y = 13<br>x + 2y = 8 | 2 | 3 |

> **解説**: 加減法では2つの式をたす・ひくことで1つの文字を消します。例えば x + y = 5 と x - y = 1 を足すと 2x = 6、x = 3。これを元の式に戻すと y = 2 です。

## 文章題

| 問題 | 答え |
|------|------|
| りんご1個とみかん1個を買うと180円、<br>りんご2個とみかん1個を買うと260円です。<br>りんご1個の値段は？ | 80円 |
| りんご1個とみかん1個を買うと180円、<br>りんご2個とみかん1個を買うと260円です。<br>みかん1個の値段は？ | 100円 |

> **解説**: りんごを x 円、みかんを y 円とおくと、x + y = 180、2x + y = 260。2つ目から1つ目をひくと x = 80。y = 180 - 80 = 100 です。
