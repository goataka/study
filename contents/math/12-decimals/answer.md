# 小数 - 解答

> **参考学年**: 小学5年

## 小数のたし算

| 問題 | 答え |
|------|------|
| 0.3 + 0.4 = | 0.7 |
| 1.5 + 2.3 = | 3.8 |
| 2.6 + 1.8 = | 4.4 |
| 3.25 + 1.5 = | 4.75 |

> 💡 小数点をそろえて計算。桁が足りないときは0を補う → `3.25 + 1.50 = 4.75`

## 小数のひき算

| 問題 | 答え |
|------|------|
| 0.8 - 0.3 = | 0.5 |
| 3.5 - 1.2 = | 2.3 |
| 5.7 - 2.9 = | 2.8 |
| 4.6 - 1.75 = | 2.85 |

> 💡 たし算と同じく小数点をそろえる → `4.60 - 1.75 = 2.85`

## 小数のかけ算

| 問題 | 答え |
|------|------|
| 0.5 × 2 = | 1.0 |
| 1.2 × 3 = | 3.6 |
| 2.5 × 4 = | 10.0 |
| 0.8 × 0.5 = | 0.40 |

> 💡 整数で計算 → 小数点以下の桁数の合計だけ小数点を左にずらす。`8 × 5 = 40` → 小数2桁分 → **0.40**
