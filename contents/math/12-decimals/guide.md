# 小数 - 解説

> **参考学年**: 小学5年 | **学習目的**: 小数のたし算・ひき算・かけ算を正確に計算できるようにする

## ✏️ 小数とは

- 1より小さい数や、端数を表す
- 0.1 = 1を **10に分けた** 1つ分
- 0.01 = 1を **100に分けた** 1つ分

## 📖 例題と解説

### 例① 1.5 + 2.3 = ?（たし算）

```
  1.5
+ 2.3     ← 小数点をそろえる
-----
  3.8 ✅
```

### 例② 5.7 - 2.9 = ?（ひき算）

```
  5.7
- 2.9     ← くり下がりに注意
-----
  2.8 ✅
```

### 例③ 0.8 × 0.5 = ?（かけ算）

```
ステップ１: 整数として計算 → 8 × 5 = 40
ステップ２: 小数点以下の桁数を数える → 0.8(1桁) + 0.5(1桁) = 2桁
ステップ３: 40 → 小数点を2桁左へ → 0.40 ✅
```

## 📌 コツ

- たし算・ひき算 → 小数点をそろえる📏
- かけ算 → 整数で計算してから小数点を合わせる🎯
