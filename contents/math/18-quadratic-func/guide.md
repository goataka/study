# 二次関数 - 解説

> **参考学年**: 高校1年 | **学習目的**: 二次関数の頂点・軸を求め、最大値・最小値を計算できるようにする

## ✏️ 二次関数とは

- **y = ax² + bx + c** の形
- グラフは **放物線** 🪂

## 📖 例題と解説

### 例① y = x² - 4x + 5 の最小値

```
平方完成:
y = x² - 4x + 5
  = (x² - 4x + 4) + 1
  = (x - 2)² + 1

→ 頂点 (2, 1)
→ a=1 > 0 → 下に凸 ∪
→ 最小値 1（x=2のとき）✅
```

### 例② y = -x² + 6x - 5 の最大値

```
y = -(x² - 6x) - 5
  = -(x² - 6x + 9) + 9 - 5
  = -(x - 3)² + 4

→ 頂点 (3, 4)
→ a=-1 < 0 → 上に凸 ∩
→ 最大値 4（x=3のとき）✅
```

## 📌 標準形のポイント

| 項目 | 見方 |
|------|------|
| 頂点 | y = a(**x - p**)² + **q** → (**p**, **q**) |
| 軸 | x = **p** |
| a > 0 | 下に凸 ∪ → **最小値**あり |
| a < 0 | 上に凸 ∩ → **最大値**あり |
