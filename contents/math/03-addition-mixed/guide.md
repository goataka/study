# たし算（まとめ） - 解説

**参考学年**: 小学1年 | **学習目的**: くり上がりなし・ありを混ぜた計算で、たし算の力を確かめる

## 📌 見分け方

- 一の位を合わせて **9以下** → くり上がり **なし**
- 一の位を合わせて **10以上** → くり上がり **あり**

## 📖 例題と解説

### 例① 5 + 3 = ?（くり上がりなし）

```
5 + 3 = 8（一の位が 8 → 10未満なのでそのまま）✅
```

### 例② 7 + 8 = ?（くり上がりあり）

```
7 + 3 = 10 → 10 + 5 = 15 ✅
```

## 📌 まとめ

| パターン | 見分け方 | 例 |
|----------|----------|-----|
| くり上がりなし | 合計が9以下 | 3 + 4 = 7 |
| くり上がりあり | 合計が10以上 | 8 + 5 = 13 |
