# 二次関数

> **参考学年**: 高校1年

## 学習目的

二次関数のグラフの特徴（頂点・軸）を理解し、最大値・最小値を求められるようになる。

## 解説

### 二次関数とは

y = ax² + bx + c の形で表される関数です。グラフは放物線になります。

### 標準形

y = a(x - p)² + q の形を標準形といいます。

- **頂点**: (p, q)
- **軸**: x = p
- a > 0 → 下に凸（最小値をもつ）
- a < 0 → 上に凸（最大値をもつ）

### 平方完成

一般形 → 標準形に変形することを平方完成といいます。

- y = x² - 4x + 5
- = (x² - 4x + 4) + 1
- = (x - 2)² + 1
- → 頂点 (2, 1)、最小値 1

### 最大値・最小値の求め方

1. 平方完成して標準形にする
2. 頂点の y 座標が最大値または最小値
