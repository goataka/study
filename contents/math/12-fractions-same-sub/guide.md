# 分数のひき算（同分母） - 解説

**参考学年**: 小学4年 | **学習目的**: 分母が同じ分数のひき算を正確に計算できるようにする

## ✏️ 同じ分母のひき算

- 分母が同じなら、**分子だけをひく**
- 分母はそのまま変えない
- 約分できるときは約分する

## 📖 例題と解説

### 例① 3/4 - 1/4 = ?

```
🍕🍕🍕  −  🍕  ＝  🍕🍕
 3/4      1/4      2/4 → 約分して 1/2
```

🎯 分子をひく → 3 - 1 = 2 → 2/4 → 約分 **1/2**

### 例② 7/8 - 3/8 = ?

- 分子をひく → 7 - 3 = 4 → 4/8
- 約分 → 4÷4 / 8÷4 = **1/2** ✅

## 📌 おぼえておこう

- たし算と同じく分母はさわらない
- 答えが0になることもある → 3/5 - 3/5 = **0**
