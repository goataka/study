# 積分 - 解説

**参考学年**: 高校3年 | **学習目的**: 多項式関数の不定積分と定積分を正確に計算できるようにする

## ✏️ 積分とは

- 微分の **逆の操作**
- 導関数から元の関数を求める

## 📌 積分の公式

| 公式 | 意味 |
|------|------|
| ∫xⁿdx = xⁿ⁺¹/(n+1) + C | 指数に1を足して、新しい指数で割る |
| C は積分定数 | 微分すると消える定数 |
| ∫{f+g}dx = ∫fdx + ∫gdx | 各項を個別に積分 |

## 📖 例題と解説

### 例① ∫(3x² + 2x)dx = ?

```
∫3x²dx = 3 · x³/3 = x³
∫2x dx = 2 · x²/2 = x²

→ x³ + x² + C ✅
```

### 例② ∫[0→2] (3x² + 2x)dx = ?

```
ステップ１: 不定積分 = x³ + x²
ステップ２: [x³ + x²]₀²
         = (2³ + 2²) - (0³ + 0²)
         = (8 + 4) - (0 + 0)
         = 12 ✅
```

## 📌 定積分のポイント

- ∫[a→b] f(x)dx = **F(b) - F(a)**
- F(x) は f(x) の不定積分
- 定積分では **C は不要**（差を取るので消える）
