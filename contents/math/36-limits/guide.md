# 極限 - 解説

**参考学年**: 高校3年 | **学習目的**: 数列と関数の極限の基本的な求め方を身につける

## ✏️ 極限とは

- ある値に **限りなく近づく** ときの値
- lim（リミット）を使って表す

## 📖 例題と解説

### 例① lim(n→∞) (2n+1)/n

```
分子分母をnでわる:
  (2n+1)/n = 2 + 1/n

n→∞ のとき 1/n → 0

答え: 2 + 0 = 2 ✅
```

### 例② lim(x→2) (x²-4)/(x-2)

```
x=2 を代入すると 0/0（不定形）→ 因数分解！

(x²-4)/(x-2) = (x+2)(x-2)/(x-2) = x+2

x→2 を代入: 2+2 = 4 ✅
```

## 📌 極限の基本パターン

- **最高次で割る**: 分数式の極限は最高次の係数の比
- **因数分解**: 0/0の不定形は因数分解で約分
- **重要公式**: lim(x→0) sin(x)/x = **1**
