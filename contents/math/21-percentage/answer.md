# 割合 - 解答

**参考学年**: 小学6年

## 割合の計算

| 問題 | 答え |
|------|------|
| 200の30%はいくつですか | 60 |
| 150の20%はいくつですか | 30 |
| 80の25%はいくつですか | 20 |
| 300の15%はいくつですか | 45 |

💡 「もとの数 × 割合」で求めます。30% = 0.3 なので `200 × 0.3 = 60`。25% = 1/4 なので `80 ÷ 4 = 20` とも計算できます。

## 速さの計算

| 問題 | 答え |
|------|------|
| 時速60kmで2時間走った距離は？ | 120km |
| 120kmを3時間で走った速さは？ | 時速40km |
| 時速40kmで100km走る時間は？ | 2.5時間 |

💡 公式「**き・は・じ**」を使います → 距離=速さ×時間、速さ=距離÷時間、時間=距離÷速さ
