# 中学3年生 - 二次方程式（解答）

## 因数分解で解く

| 問題 | 解 |
|------|------|
| x² - 5x + 6 = 0 | x = 2, 3 |
| x² - 7x + 12 = 0 | x = 3, 4 |
| x² - x - 6 = 0 | x = -2, 3 |
| x² + 5x + 6 = 0 | x = -2, -3 |
| x² - 9 = 0 | x = -3, 3 |

**解説:**

- x² - 5x + 6 = 0 → (x - 2)(x - 3) = 0 → x = 2, 3
- x² - 7x + 12 = 0 → (x - 3)(x - 4) = 0 → x = 3, 4
- x² - x - 6 = 0 → (x + 2)(x - 3) = 0 → x = -2, 3
- x² + 5x + 6 = 0 → (x + 2)(x + 3) = 0 → x = -2, -3
- x² - 9 = 0 → (x + 3)(x - 3) = 0 → x = -3, 3

## 解の公式で解く

| 問題 | 解 |
|------|------|
| x² + 4x + 3 = 0 | x = -1, -3 |
| x² - 6x + 5 = 0 | x = 1, 5 |
| x² + 2x - 8 = 0 | x = -4, 2 |
| 2x² - 5x + 2 = 0 | x = 1/2, 2 |

**解説:**

解の公式を使用：x = (-b ± √(b²-4ac)) / 2a
