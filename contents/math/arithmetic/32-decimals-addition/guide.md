# 小数のたし算 - 解説

**参考学年**: 小学5年

**学習目的**: 小数点をそろえて、小数のたし算を正確に計算できるようにする

## ✏️ 小数のたし算のやり方

- 小数点の位置を **たてにそろえる** 📏
- 桁が足りないときは **0を補う**
- 整数のたし算と同じように計算する

## 📖 例題と解説

### 例① 1.5 + 2.3 = ?

```text
  1.5
+ 2.3     ← 小数点をそろえる
-----
  3.8 ✅
```

### 例② 3.25 + 1.5 = ?

```text
  3.25
+ 1.50    ← 桁が足りないので 0 を補う
------
  4.75 ✅
```

## 📌 コツ

- 小数点の位置をそろえれば、あとは整数と同じ ✅
- くり上がりにも注意 → `0.7 + 0.6 = 1.3`
