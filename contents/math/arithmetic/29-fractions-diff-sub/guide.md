# 分数のひき算（通分あり） - 解説

**参考学年**: 小学5年 | **学習目的**: 分母が異なる分数のひき算で通分を使って正確に計算できるようにする

## ✏️ 通分してひき算

- たし算と同じく **通分してから** ひく
- 分母を最小公倍数にそろえる

## 📖 例題と解説

### 例① 1/2 - 1/3 = ?

```text
ステップ１: 通分（最小公倍数 = 6）
  1/2 → 3/6
  1/3 → 2/6

ステップ２: 分子をひく
  3/6 - 2/6 = 1/6 ✅
```

### 例② 5/6 - 2/3 = ?

```text
ステップ１: 通分（6は3の倍数 → 6に合わせる）
  5/6 → そのまま
  2/3 → 4/6

ステップ２: 5/6 - 4/6 = 1/6 ✅
```

## 📌 おぼえておこう

- 通分の手順はたし算と同じ
- ひき算の答えが0になることもある

## 解答

| 問題 | 答え |
|------|------|
| 1/2 - 1/3 = | 3/6 - 2/6 = 1/6 |
| 3/4 - 1/6 = | 9/12 - 2/12 = 7/12 |
| 2/3 - 1/4 = | 8/12 - 3/12 = 5/12 |
| 4/5 - 1/3 = | 12/15 - 5/15 = 7/15 |
| 5/6 - 1/4 = | 10/12 - 3/12 = 7/12 |
| 3/4 - 2/5 = | 15/20 - 8/20 = 7/20 |
| 7/8 - 1/2 = | 7/8 - 4/8 = 3/8 |
| 2/3 - 1/5 = | 10/15 - 3/15 = 7/15 |
| 5/6 - 2/3 = | 5/6 - 4/6 = 1/6 |
| 3/4 - 1/3 = | 9/12 - 4/12 = 5/12 |

💡 **ポイント**: たし算と同じく、まず通分してからひきます
