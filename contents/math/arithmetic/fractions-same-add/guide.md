# たし算（同分母）

## ✏️ 同じ分母のたし算

- 分母が同じなら、**分子だけをたす**
- 分母はそのまま変えない
- 答えが約分できるときは約分する

## 📖 例題と解説

### 例① 1/4 + 2/4 = ?

```text
🍕  ＋  🍕🍕  ＝  🍕🍕🍕
1/4    2/4      3/4
```

🎯 分子をたす → 1 + 2 = 3 → **3/4**

### 例② 1/9 + 5/9 = ?

- 分子をたす → 1 + 5 = 6 → 6/9
- 約分 → 6÷3 / 9÷3 = **2/3** ✅

## 📌 約分のやり方

- 分子と分母を **同じ数でわる**
- 例: 6/9 → 6÷3 / 9÷3 = **2/3**

## 例

| 問題 | 答え |
|------|------|
| 1/4 + 2/4 = | 3/4 |
| 1/3 + 1/3 = | 2/3 |
| 2/5 + 1/5 = | 3/5 |
| 3/8 + 2/8 = | 5/8 |
| 1/6 + 4/6 = | 5/6 |
| 2/7 + 3/7 = | 5/7 |
| 1/9 + 5/9 = | 6/9 = 2/3 |
| 3/10 + 4/10 = | 7/10 |
| 2/5 + 2/5 = | 4/5 |
| 1/8 + 6/8 = | 7/8 |

💡 **ポイント**: 分母はそのまま、分子だけをたします。約分できるときは約分しましょう
