# 比 - 解説

**参考学年**: 小学6年 | **学習目的**: 比を最も簡単な整数の比にする方法を身につける

## ✏️ 比とは

- 2つの数の **関係** を「:」で表す
- 例: 男子4人、女子6人 → 男女の比は **4:6**

## 📖 例題と解説

### 例① 4:6 を簡単にする

```text
4 : 6
↓ ÷2  ↓ ÷2     ← 両方を最大公約数(2)でわる
2 : 3  ✅
```

🎯 **4:6 = 2:3**

### 例② 15:20 を簡単にする

```text
15 : 20
↓ ÷5   ↓ ÷5    ← 両方を最大公約数(5)でわる
 3 :  4  ✅
```

🎯 **15:20 = 3:4**

## 📌 手順

1. 2つの数の **最大公約数** を見つける
2. 両方をその数で **わる**
3. これ以上われなければ完成 ✅

## 解答

| 問題 | 答え |
|------|------|
| 4:6 = | 2:3 |
| 9:12 = | 3:4 |
| 15:20 = | 3:4 |
| 8:12 = | 2:3 |
| 6:10 = | 3:5 |
| 14:21 = | 2:3 |
| 12:18 = | 2:3 |
| 20:30 = | 2:3 |

💡 両方の数を **最大公約数** でわります。`4:6` → 両方を2でわる → **2:3**。`14:21` → 両方を7でわる → **2:3**
