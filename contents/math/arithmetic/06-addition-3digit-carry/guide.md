# 3桁のたし算（くり上がりあり） - 解説

**参考学年**: 小学3年 | **学習目的**: くり上がりのある3けたのたし算を正確に計算できるようにする

## ✏️ くり上がりが続く3けたのたし算

- 一の位でくり上がると十の位の計算がかわる
- 十の位でくり上がると百の位の計算がかわる
- 百の位でくり上がると4けたになる

## 📖 例題と解説

### 例① 357 + 246 = ?

```
  357
+ 246
-----
一の位: 7 + 6 = 13 → 3を書いて1くり上げ
十の位: 5 + 4 + 1 = 10 → 0を書いて1くり上げ
百の位: 3 + 2 + 1 = 6
→ 603
```

### 例② 765 + 489 = ?

```
  765
+ 489
-----
一の位: 5 + 9 = 14 → 4を書いて1くり上げ
十の位: 6 + 8 + 1 = 15 → 5を書いて1くり上げ
百の位: 7 + 4 + 1 = 12 → 2を書いて1くり上げ
→ 1254
```

## 📌 おぼえておこう

- くり上がりを小さく書いておくと間違いが少ない
- 右から順に計算することが大切
