# 1桁のかけ算（九九） - 解説

**参考学年**: 小学2年 | **学習目的**: かけ算の意味を理解し、九九を覚えて正確に計算できるようにする

## ✏️ かけ算とは

- 同じ数を **くり返したす** 計算
- 記号「×」を使う

## 📖 例題と解説

### 例① 3 × 4 = ?

```
🍎🍎🍎  🍎🍎🍎  🍎🍎🍎  🍎🍎🍎
 (3個)  (3個)  (3個)  (3個) → 3を4回 = 12
```

🎯 **3 × 4 = 3 + 3 + 3 + 3 = 12**

### 例② 5 × 3 = ?

```
🔵🔵🔵🔵🔵  🔵🔵🔵🔵🔵  🔵🔵🔵🔵🔵
  (5個)      (5個)      (5個) → 5を3回 = 15
```

🎯 **5 × 3 = 5 + 5 + 5 = 15**

## 📌 おぼえておこう

- 順番を変えても答えは同じ → `3 × 4 = 4 × 3 = 12`
- 0をかけると0 → `5 × 0 = 0`
- 1をかけてもそのまま → `5 × 1 = 5`
