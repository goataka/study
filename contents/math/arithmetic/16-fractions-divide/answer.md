# 分数のわり算 - 解答

**参考学年**: 小学6年

| 問題 | 答え |
|------|------|
| 1/2 ÷ 1/3 = | 1/2 × 3/1 = 3/2 = 1と1/2 |
| 3/4 ÷ 2/3 = | 3/4 × 3/2 = 9/8 = 1と1/8 |
| 2/5 ÷ 1/4 = | 2/5 × 4/1 = 8/5 = 1と3/5 |
| 5/6 ÷ 5/9 = | 5/6 × 9/5 = 45/30 = 3/2 |
| 3/8 ÷ 3/4 = | 3/8 × 4/3 = 12/24 = 1/2 |
| 4/7 ÷ 2/3 = | 4/7 × 3/2 = 12/14 = 6/7 |
| 1/3 ÷ 2/5 = | 1/3 × 5/2 = 5/6 |
| 2/3 ÷ 4 = | 2/3 × 1/4 = 2/12 = 1/6 |
| 7/8 ÷ 7/16 = | 7/8 × 16/7 = 112/56 = 2 |
| 5/6 ÷ 1/2 = | 5/6 × 2/1 = 10/6 = 5/3 |

💡 **ポイント**: わる数をひっくり返して（逆数）かけ算にします
