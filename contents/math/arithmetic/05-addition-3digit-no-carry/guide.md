# 3桁のたし算（くり上がりなし） - 解説

**参考学年**: 小学3年 | **学習目的**: くり上がりのない3けたのたし算を正確に計算できるようにする

## ✏️ 3けたのたし算

- 一の位・十の位・百の位を**それぞれ**足す
- くり上がりがないので各桁の和は9以下

## 📖 例題と解説

### 例① 213 + 145 = ?

```
  213
+ 145
-----
一の位: 3 + 5 = 8
十の位: 1 + 4 = 5
百の位: 2 + 1 = 3
→ 358
```

### 例② 412 + 384 = ?

```
  412
+ 384
-----
一の位: 2 + 4 = 6
十の位: 1 + 8 = 9
百の位: 4 + 3 = 7
→ 796
```

## 📌 おぼえておこう

- 右（一の位）から順に計算すると間違いが少ない
- 桁をそろえて書くと計算しやすい
