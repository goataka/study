# 2桁のひき算（くり下がりなし） - 解説

**参考学年**: 小学2年 | **学習目的**: くり下がりのない2けたのひき算を正確に計算できるようにする

## ✏️ 2けたのひき算

- 一の位どうし、十の位どうしを**それぞれ**ひく
- くり下がりがないので一の位の上が下以上になっている

## 📖 例題と解説

### 例① 57 - 23 = ?

```
  57
- 23
----
一の位: 7 - 3 = 4
十の位: 5 - 2 = 3
→ 34
```

### 例② 98 - 54 = ?

```
  98
- 54
----
一の位: 8 - 4 = 4
十の位: 9 - 5 = 4
→ 44
```

## 📌 おぼえておこう

- 大きい位から順に引いても、小さい位から順に引いても同じ
- それぞれの位で上の数が下の数より大きいか、たしかめよう
