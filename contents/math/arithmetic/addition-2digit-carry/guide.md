# 2桁のたし算（くり上がりあり）

## ✏️ くり上がりとは

- 一の位の和が **10以上** になるとき、十の位に1を「くり上げる」

## 📖 例題と解説

### 例① 37 + 46 = ?

```text
  37
+ 46
----
一の位: 7 + 6 = 13 → 3を書いて1をくり上げ
十の位: 3 + 4 + 1（くり上がり）= 8
→ 83
```

### 例② 69 + 35 = ?

```text
  69
+ 35
----
一の位: 9 + 5 = 14 → 4を書いて1をくり上げ
十の位: 6 + 3 + 1 = 10 → 0を書いて1をくり上げ
→ 104
```

## 📌 おぼえておこう

- 一の位が10以上 → 十の位に1くり上げ
- 十の位が10以上 → 百の位に1くり上げ（3けたになる）

## 例

| 問題 | 答え |
|------|------|
| 37 + 46 = | 83 |
| 58 + 27 = | 85 |
| 69 + 35 = | 104 |
| 47 + 53 = | 100 |
| 76 + 48 = | 124 |
| 84 + 39 = | 123 |
| 55 + 67 = | 122 |
| 28 + 74 = | 102 |
| 93 + 17 = | 110 |
| 46 + 85 = | 131 |

💡 **ポイント**: 一の位の和が10以上になったら、十の位に1くり上げます。答えが100を超えることもあります。
