# 2桁のたし算（くり上がりなし）

## ✏️ 2けたのたし算とは

- 十の位と一の位を**別々に**計算する
- くり上がりがないので、一の位と十の位の合計が9以下になる

## 📖 例題と解説

### 例① 21 + 13 = ?

```text
  21
+ 13
----
一の位: 1 + 3 = 4
十の位: 2 + 1 = 3
→ 34
```

### 例② 45 + 32 = ?

```text
  45
+ 32
----
一の位: 5 + 2 = 7
十の位: 4 + 3 = 7
→ 77
```

## 📌 おぼえておこう

- 一の位の合計が10をこえないか、たしかめよう
- 十の位と一の位を分けて考えると計算しやすい
