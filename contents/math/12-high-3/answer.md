# 高校3年生 - 微分と積分（解答）

## 微分

| 問題 | 導関数 f'(x) |
|------|------|
| f(x) = 3x² + 2x + 1 | 6x + 2 |
| f(x) = x³ - 4x² + 5x | 3x² - 8x + 5 |
| f(x) = 2x⁴ - 3x³ + x | 8x³ - 9x² + 1 |
| f(x) = 5x³ + 2x² - 7 | 15x² + 4x |

**解説:**

(xⁿ)' = n・xⁿ⁻¹ を使います。

## 極値

| 問題 | 極大値 | 極小値 |
|------|------|------|
| f(x) = x³ - 3x² + 4 | 4 (x=0) | 0 (x=2) |
| f(x) = -x³ + 3x² + 1 | 5 (x=2) | 1 (x=0) |

**解説:**

f(x) = x³ - 3x² + 4 の場合：

- f'(x) = 3x² - 6x = 3x(x - 2) = 0
- x = 0, 2
- f(0) = 4（極大値）、f(2) = 0（極小値）

## 積分

| 問題 | 答え |
|------|------|
| ∫(3x² + 2x)dx = | x³ + x² + C |
| ∫(2x³ - 4x)dx = | (1/2)x⁴ - 2x² + C |
| ∫(x² + 3x + 2)dx = | (1/3)x³ + (3/2)x² + 2x + C |

**解説:**

∫xⁿdx = (1/(n+1))xⁿ⁺¹ + C を使います。
