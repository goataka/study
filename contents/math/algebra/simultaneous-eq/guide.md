# 連立方程式

## ✏️ 連立方程式とは

- 2つの方程式を **同時に** 満たす解を求める
- 未知数が2つ → 方程式も **2つ必要**

## 📖 例題と解説

### 例① 加減法

```text
  x + y = 5  …①
  x - y = 1  …②

① + ②:  2x = 6  →  x = 3
x=3を①に代入: 3 + y = 5  →  y = 2
```

🎯 **x = 3, y = 2**

### 例② 代入法

```text
  x + y = 5  →  y = 5 - x  …①'
  ①'を②に代入:
  x - (5 - x) = 1
  2x - 5 = 1
  x = 3, y = 2
```

🎯 同じ答え！方法が違うだけ

## 📌 文章題の手順

1. 求めたいものを **x, y** とおく
2. 条件から **2つの式** をつくる
3. 連立方程式を **解く**

## 📝 例

| 問題 | x = | y = |
|------|------|------|
| x + y = 5, x - y = 1 | 3 | 2 |
| 2x + y = 7, x + y = 4 | 3 | 1 |
| x + 2y = 8, x - y = 2 | 4 | 2 |
| 3x + y = 11, x + y = 5 | 3 | 2 |
| 2x + 3y = 13, x + 2y = 8 | 2 | 3 |

💡 加減法: 2式をたす/ひくことで1つの文字を消す。`x+y=5` と `x-y=1` を足す → `2x=6` → `x=3`、`y=2`

## 📌 文章題

| 問題 | 答え |
|------|------|
| りんご1個とみかん1個で180円、りんご2個とみかん1個で260円。りんご1個は？ | 80円 |
| 同じ条件で、みかん1個は？ | 100円 |

💡 りんご=x、みかん=y → `x+y=180`, `2x+y=260`。②-① → `x=80`、`y=100`
