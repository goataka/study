# 文字式 - 解答

**参考学年**: 中学1年

## 文字式のルール

| 問題 | 答え |
|------|------|
| a × 3 = | 3a |
| x × y = | xy |
| a × a = | a² |
| 1 × x = | x |
| x ÷ 3 = | x/3 |

## 文字式の計算

| 問題 | 答え |
|------|------|
| 3a + 5a = | 8a |
| 7x - 2x = | 5x |
| 4a + 3b + 2a - b = | 6a + 2b |
| 2(3x + 1) = | 6x + 2 |
| 5(2a - 3) + 4a = | 14a - 15 |

💡 **ポイント**: 同じ文字の項どうしをまとめます。数字は数字で、aはaで、bはbで
