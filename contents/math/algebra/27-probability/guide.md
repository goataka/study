# 確率 - 解説

**参考学年**: 中学2年 | **学習目的**: 場合の数の数え方と確率の基本的な求め方を身につける

## ✏️ 確率とは

- ある事が起こる **可能性の度合い** を数で表したもの
- 確率 = **あてはまる場合の数** ÷ **すべての場合の数**
- 0（絶対起きない）〜 1（必ず起きる）

## 📖 例題と解説

### 例① サイコロで偶数が出る確率

```
すべての場合: {1, 2, 3, 4, 5, 6} → 6通り
偶数の場合:   {2, 4, 6}           → 3通り

確率 = 3/6 = 1/2 ✅
```

### 例② コイン2枚の場合

```
すべての場合（樹形図）:
  1枚目  2枚目  結果
  表 ── 表    → (表,表) ← これ！
  表 ── 裏    → (表,裏)
  裏 ── 表    → (裏,表)
  裏 ── 裏    → (裏,裏)

すべて: 4通り、2枚とも表: 1通り
確率 = 1/4 ✅
```

## 📌 おぼえておこう

- すべての確率の合計 = **1**
- 「起きない確率」= 1 - 「起きる確率」
