# 一次方程式 - 解答

**参考学年**: 中学1年

| 問題 | x = |
|------|------|
| x + 3 = 7 | x = 4 |
| x - 5 = 2 | x = 7 |
| 2x = 10 | x = 5 |
| 3x + 4 = 13 | x = 3 |
| 5x - 3 = 12 | x = 3 |
| 2x + 1 = x + 5 | x = 4 |
| 4x - 7 = 2x + 3 | x = 5 |
| 3(x + 2) = 15 | x = 3 |
| 2(x - 1) = 3x - 5 | x = 3 |
| (x + 3)/2 = 4 | x = 5 |

💡 **ポイント**: xの項を左辺に、数字を右辺に移項して整理します
