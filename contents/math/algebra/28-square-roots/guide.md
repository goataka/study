# 平方根 - 解説

**参考学年**: 中学3年 | **学習目的**: 平方根の意味を理解し、計算を正確にできるようにする

## ✏️ 平方根とは

- 2乗して a になる数を **√a**（ルートa）という
- √4 = 2（2² = 4 だから）
- √9 = 3（3² = 9 だから）

## 📖 例題と解説

### 例① √18 を簡単にする

```
ステップ１: 素因数分解
  18 = 2 × 3 × 3 = 2 × 3²

ステップ２: ペアを外に出す
  √18 = √(3² × 2) = 3√2 ✅
```

### 例② √12 + √27 = ?

```
ステップ１: それぞれ簡単にする
  √12 = √(4×3) = 2√3
  √27 = √(9×3) = 3√3

ステップ２: 同類項をまとめる
  2√3 + 3√3 = 5√3 ✅
```

## 📌 計算ルール

- √a × √b = **√(ab)**
- √a ÷ √b = **√(a/b)**
- n√a + m√a = **(n+m)√a**（同じルートの中身のみ）
