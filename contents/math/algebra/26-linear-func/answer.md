# 一次関数 - 解答

**参考学年**: 中学2年

## 一次関数の値

| x | y |
|---|---|
| 0 | 1 |
| 1 | 3 |
| 2 | 5 |
| -1 | -1 |
| -2 | -3 |

## 式を求める

| 条件 | y = |
|------|------|
| 傾き2、切片3 | y = 2x + 3 |
| 傾き-1、切片5 | y = -x + 5 |
| (1, 3)と(3, 7)を通る | y = 2x + 1 |
| (0, 4)と(2, 0)を通る | y = -2x + 4 |
| 傾き3、(1, 5)を通る | y = 3x + 2 |

💡 **ポイント**: 2点から傾き = (y₂-y₁)/(x₂-x₁) で求め、y = ax + b に代入して b を出します
