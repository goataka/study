# 二次方程式 - 練習問題

**参考学年**: 中学3年 | **学習目的**: 因数分解と解の公式を使い分けて、二次方程式を確実に解けるようにする

## 因数分解で解く

| 問題 | 解 |
|------|------|
| x² - 5x + 6 = 0 | 　　　　　　　　　　　　 |
| x² - 7x + 12 = 0 | 　　　　　　　　　　　　 |
| x² - x - 6 = 0 | 　　　　　　　　　　　　 |
| x² + 5x + 6 = 0 | 　　　　　　　　　　　　 |
| x² - 9 = 0 | 　　　　　　　　　　　　 |

## 解の公式で解く

📐 x = (-b ± √(b²-4ac)) / 2a

| 問題 | 解 |
|------|------|
| x² + 4x + 3 = 0 | 　　　　　　　　　　　　 |
| x² - 6x + 5 = 0 | 　　　　　　　　　　　　 |
| x² + 2x - 8 = 0 | 　　　　　　　　　　　　 |
| 2x² - 5x + 2 = 0 | 　　　　　　　　　　　　 |
