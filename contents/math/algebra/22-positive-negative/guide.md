# 正負の数 - 解説

**参考学年**: 中学1年 | **学習目的**: 正の数と負の数の四則計算を正確にできるようにする

## ✏️ 正負の数とは

- ➕ 正の数: 0より大きい（例: +3, +5）
- ➖ 負の数: 0より小さい（例: -2, -7）
- 0は正でも負でもない

```
← マイナス  0  プラス →
... -3  -2  -1  0  +1  +2  +3 ...
```

## 📖 例題と解説

### 例① (+7) + (-4) = ?（たし算）

```
符号がちがう → 絶対値の差、大きい方の符号
|7| - |4| = 3 → 7の方が大きいので ＋
```

🎯 **(+7) + (-4) = +3**

### 例② (+6) - (-2) = ?（ひき算）

```
ひき算 → 符号を変えてたし算にする
(+6) - (-2) = (+6) + (+2) = +8
```

🎯 **(+6) - (-2) = +8**

### 例③ (-4) × (-7) = ?（かけ算）

```
異符号 → 負(-)   ⚡ 同符号 → 正(+)
(-) × (-) = (+) ← 同符号！
4 × 7 = 28
```

🎯 **(-4) × (-7) = +28**

## 📌 符号のルール（かけ算・わり算）

| 組み合わせ | 結果 |
|------------|------|
| ＋ × ＋ | ＋ |
| ＋ × − | − |
| − × ＋ | − |
| − × − | ＋ |
