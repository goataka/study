# 一次方程式

## ✏️ 方程式とは

- **=（イコール）** でつながった式
- 未知の数 **x** の値を求める

## 📖 例題と解説

### 例① x + 3 = 7

```text
x + 3 = 7
x = 7 - 3  ← +3を移項すると-3
x = 4 ✅
```

### 例② 3x + 4 = 13

```text
3x + 4 = 13
3x = 13 - 4  ← +4を移項
3x = 9
x = 9 ÷ 3   ← ×3を移項すると÷3
x = 3 ✅
```

### 例③ 2x + 1 = x + 5

```text
2x + 1 = x + 5
2x - x = 5 - 1  ← xを左、数字を右に移項
x = 4 ✅
```

## 📌 移項のルール

- 左辺から右辺（またはその逆）に項を移すこと
- 移項すると **符号が逆** になる
  - +3 → **-3**
  - -5 → **+5**

## 例

| 問題 | x = |
|------|------|
| x + 3 = 7 | x = 4 |
| x - 5 = 2 | x = 7 |
| 2x = 10 | x = 5 |
| 3x + 4 = 13 | x = 3 |
| 5x - 3 = 12 | x = 3 |
| 2x + 1 = x + 5 | x = 4 |
| 4x - 7 = 2x + 3 | x = 5 |
| 3(x + 2) = 15 | x = 3 |
| 2(x - 1) = 3x - 5 | x = 3 |
| (x + 3)/2 = 4 | x = 5 |

💡 **ポイント**: xの項を左辺に、数字を右辺に移項して整理します
