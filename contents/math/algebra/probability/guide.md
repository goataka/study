# 確率

## ✏️ 確率とは

- ある事が起こる **可能性の度合い** を数で表したもの
- 確率 = **あてはまる場合の数** ÷ **すべての場合の数**
- 0（絶対起きない）〜 1（必ず起きる）

## 📖 例題と解説

### 例① サイコロで偶数が出る確率

```text
すべての場合: {1, 2, 3, 4, 5, 6} → 6通り
偶数の場合:   {2, 4, 6}           → 3通り

確率 = 3/6 = 1/2 ✅
```

### 例② コイン2枚の場合

```text
すべての場合（樹形図）:
  1枚目  2枚目  結果
  表 ── 表    → (表,表) ← これ！
  表 ── 裏    → (表,裏)
  裏 ── 表    → (裏,表)
  裏 ── 裏    → (裏,裏)

すべて: 4通り、2枚とも表: 1通り
確率 = 1/4 ✅
```

## 📌 おぼえておこう

- すべての確率の合計 = **1**
- 「起きない確率」= 1 - 「起きる確率」

## 解答

| 問題 | 答え |
|------|------|
| サイコロを1回振って3が出る確率 | 1/6 |
| サイコロを1回振って偶数が出る確率 | 3/6 = 1/2 |
| コイン1枚を投げて表が出る確率 | 1/2 |
| トランプ52枚からハートを引く確率 | 13/52 = 1/4 |
| 赤玉3個、白玉2個から赤玉を引く確率 | 3/5 |
| サイコロを1回振って5以上が出る確率 | 2/6 = 1/3 |
| コイン2枚を投げて2枚とも表が出る確率 | 1/4 |
| サイコロ2個を振って合計が7になる確率 | 6/36 = 1/6 |

💡 **ポイント**: 確率 = あてはまる場合の数 ÷ すべての場合の数
