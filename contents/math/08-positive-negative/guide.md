# 正負の数

> **参考学年**: 中学1年

## 学習目的

正の数と負の数の概念を理解し、四則計算（たし算・ひき算・かけ算・わり算）ができるようになる。

## 解説

### 正負の数とは

- 正の数：0より大きい数（例：+3, +5）
- 負の数：0より小さい数（例：-2, -7）
- 0は正でも負でもありません

### たし算のルール

- 同じ符号どうし：絶対値をたして、共通の符号をつける
  - (+5) + (+3) = +8
  - (-5) + (-3) = -8
- 違う符号どうし：絶対値の大きい方から小さい方をひき、大きい方の符号をつける
  - (+7) + (-4) = +3
  - (-6) + (+2) = -4

### ひき算のルール

ひき算は「符号を変えてたし算」に変えられます。

- (+8) - (+5) = (+8) + (-5) = +3
- (-4) - (+7) = (-4) + (-7) = -11

### かけ算・わり算のルール

- 同じ符号どうし → 答えは正（+）
- 違う符号どうし → 答えは負（-）
