# 正負の数 - 解答

> **参考学年**: 中学1年

## 正負の数のたし算・ひき算

| 問題 | 答え |
|------|------|
| (+5) + (+3) = | +8 |
| (+7) + (-4) = | +3 |
| (-6) + (+2) = | -4 |
| (-5) + (-3) = | -8 |
| (+8) - (+5) = | +3 |
| (+6) - (-2) = | +8 |
| (-4) - (+7) = | -11 |
| (-9) - (-3) = | -6 |

> **解説**: たし算は同符号なら絶対値をたし、異符号なら絶対値の差を求めます。ひき算は「符号を変えてたし算」にします。例えば (+6) - (-2) = (+6) + (+2) = +8 です。

## 正負の数のかけ算・わり算

| 問題 | 答え |
|------|------|
| (+3) × (+4) = | +12 |
| (+6) × (-2) = | -12 |
| (-5) × (+3) = | -15 |
| (-4) × (-7) = | +28 |
| (+12) ÷ (+3) = | +4 |
| (+15) ÷ (-5) = | -3 |
| (-18) ÷ (+6) = | -3 |
| (-20) ÷ (-4) = | +5 |

> **解説**: 同符号どうしの計算は答えが正（+）、異符号どうしの計算は答えが負（-）になります。(-4) × (-7) = +28 のように、マイナス × マイナス = プラスです。
