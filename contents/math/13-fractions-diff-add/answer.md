# 分数のたし算（通分あり） - 解答

**参考学年**: 小学5年

| 問題 | 答え |
|------|------|
| 1/2 + 1/3 = | 3/6 + 2/6 = 5/6 |
| 1/4 + 1/6 = | 3/12 + 2/12 = 5/12 |
| 2/3 + 1/4 = | 8/12 + 3/12 = 11/12 |
| 1/3 + 1/5 = | 5/15 + 3/15 = 8/15 |
| 3/4 + 1/6 = | 9/12 + 2/12 = 11/12 |
| 2/5 + 1/3 = | 6/15 + 5/15 = 11/15 |
| 1/2 + 2/5 = | 5/10 + 4/10 = 9/10 |
| 3/8 + 1/4 = | 3/8 + 2/8 = 5/8 |
| 1/6 + 1/4 = | 2/12 + 3/12 = 5/12 |
| 2/3 + 1/6 = | 4/6 + 1/6 = 5/6 |

💡 **ポイント**: 分母が違うときは、まず通分して分母をそろえてからたします
