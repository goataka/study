# 小数のかけ算 - 解説

**参考学年**: 小学5年

**学習目的**: 小数のかけ算を、整数の計算に置きかえて正確に計算できるようにする

## ✏️ 小数のかけ算のやり方

- 小数を **整数にして** 計算する
- 小数点以下の桁数を数えておく
- 最後に **桁数の合計だけ** 小数点を左にずらす

## 📖 例題と解説

### 例① 1.2 × 3 = ?

```
ステップ１: 整数として計算 → 12 × 3 = 36
ステップ２: 小数点以下の桁数 → 1.2 は 1桁
ステップ３: 36 → 小数点を1桁左へ → 3.6 ✅
```

### 例② 0.8 × 0.5 = ?

```
ステップ１: 整数として計算 → 8 × 5 = 40
ステップ２: 小数点以下の桁数 → 0.8(1桁) + 0.5(1桁) = 2桁
ステップ３: 40 → 小数点を2桁左へ → 0.40 ✅
```

## 📌 コツ

- まず整数で計算 🔢 → そのあと小数点を合わせる 🎯
- 桁数を数えまちがえないように注意 ⚠️
