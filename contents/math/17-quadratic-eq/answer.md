# 二次方程式 - 解答

> **参考学年**: 中学3年

## 因数分解で解く

| 問題 | 解 |
|------|------|
| x² - 5x + 6 = 0 | x = 2, 3 |
| x² - 7x + 12 = 0 | x = 3, 4 |
| x² - x - 6 = 0 | x = -2, 3 |
| x² + 5x + 6 = 0 | x = -2, -3 |
| x² - 9 = 0 | x = ±3 |

> 💡 「たして b、かけて c」の2数を探す。`x²-5x+6` → たして-5、かけて6 → **-2と-3** → `(x-2)(x-3)=0`

## 解の公式で解く

| 問題 | 解 |
|------|------|
| x² + 4x + 3 = 0 | x = -1, -3 |
| x² - 6x + 5 = 0 | x = 1, 5 |
| x² + 2x - 8 = 0 | x = -4, 2 |
| 2x² - 5x + 2 = 0 | x = 1/2, 2 |

> 💡 `x²+4x+3=0` → a=1,b=4,c=3 → `x = (-4±√(16-12))/2 = (-4±2)/2` → **x=-1,-3**
