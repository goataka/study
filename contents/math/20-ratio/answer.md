# 比 - 解答

**参考学年**: 小学6年

| 問題 | 答え |
|------|------|
| 4:6 = | 2:3 |
| 9:12 = | 3:4 |
| 15:20 = | 3:4 |
| 8:12 = | 2:3 |
| 6:10 = | 3:5 |
| 14:21 = | 2:3 |
| 12:18 = | 2:3 |
| 20:30 = | 2:3 |

💡 両方の数を **最大公約数** でわります。`4:6` → 両方を2でわる → **2:3**。`14:21` → 両方を7でわる → **2:3**
