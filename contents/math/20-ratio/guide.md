# 比 - 解説

**参考学年**: 小学6年 | **学習目的**: 比を最も簡単な整数の比にする方法を身につける

## ✏️ 比とは

- 2つの数の **関係** を「:」で表す
- 例: 男子4人、女子6人 → 男女の比は **4:6**

## 📖 例題と解説

### 例① 4:6 を簡単にする

```
4 : 6
↓ ÷2  ↓ ÷2     ← 両方を最大公約数(2)でわる
2 : 3  ✅
```

🎯 **4:6 = 2:3**

### 例② 15:20 を簡単にする

```
15 : 20
↓ ÷5   ↓ ÷5    ← 両方を最大公約数(5)でわる
 3 :  4  ✅
```

🎯 **15:20 = 3:4**

## 📌 手順

1. 2つの数の **最大公約数** を見つける
2. 両方をその数で **わる**
3. これ以上われなければ完成 ✅
