# 分数 - 解説

> **参考学年**: 小学4年 | **学習目的**: 同じ分母の分数のたし算・ひき算を正確に計算できるようにする

## ✏️ 分数とは

- 1つのものを等しく分けた数
- 上の数 → **分子（ぶんし）**
- 下の数 → **分母（ぶんぼ）**

```
  1   ← 分子
 ─── 
  4   ← 分母
```

> 🍕 ピザを4等分した1切れ → **1/4**

## 📖 例題と解説

### 例① 1/4 + 2/4 = ?（たし算）

```
🍕  ＋  🍕🍕  ＝  🍕🍕🍕
1/4    2/4      3/4
```

> 🎯 分母はそのまま、分子をたす → 1+2=3 → **3/4**

### 例② 3/4 - 1/4 = ?（ひき算）

> 🎯 分母はそのまま、分子をひく → 3-1=2 → 2/4 → 約分して **1/2**

## 📌 約分

- 分子と分母を **同じ数でわる** こと
- 例: 2/4 → 2÷2 / 4÷2 = **1/2**
- 例: 6/8 → 6÷2 / 8÷2 = **3/4**
