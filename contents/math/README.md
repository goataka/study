# 数学ドリル

[ドリル一覧へ戻る](../README.md)

## 学習の流れ

数学学習は「算数・基本計算」→「代数」→「解析・高校数学」の3つのカテゴリに分けて学習します。

### 📊 算数・基本計算

小学校で学ぶ算数の基本計算を扱います。「たし算・ひき算」→「かけ算・わり算」→「分数・小数」→「比・割合」の順に進みます。

[算数・基本計算ガイドを見る](arithmetic/README.md)

### 📐 代数

中学校で学ぶ代数を扱います。「正負の数・文字式・方程式」→「関数」→「確率・平方根」の順に進みます。

[代数学習ガイドを見る](algebra/README.md)

### 📈 解析・高校数学

高校で学ぶ数学を扱います。「二次関数・不等式」→「三角関数・指数対数・数列・ベクトル」→「極限・微分・積分」の順に進みます。

[解析・高校数学ガイドを見る](calculus/README.md)
