# 数学ドリル

## 学習の流れ

```
小学校（算数の基礎）
├─ たし算・ひき算（1年）
│   01 たし算（くり上がりなし）→ 02（くり上がりあり）→ 03（まとめ）
│   04 ひき算（くり下がりなし）→ 05（くり下がりあり）→ 06（まとめ）
│
├─ かけ算・わり算（2〜3年）
│   07 かけ算 → 08 わり算（あまりなし）→ 09（あまりあり）→ 10（まとめ）
│
├─ 分数（4〜6年）
│   11 たし算（同分母）→ 12 ひき算（同分母）
│   → 13 たし算（通分あり）→ 14 ひき算（通分あり）
│   → 15 かけ算 → 16 わり算
│
├─ 小数（4〜5年）
│   17 たし算 → 18 ひき算 → 19 かけ算
│
└─ 比・割合（5〜6年）
    20 比 → 21 割合

中学校（代数と確率）
├─ 正負の数・文字式（1年）
│   22 正負の数 → 23 文字式 → 24 一次方程式
│
├─ 連立方程式・関数・確率（2年）
│   25 連立方程式 → 26 一次関数 → 27 確率
│
└─ 平方根・二次方程式（3年）
    28 平方根 → 29 二次方程式

高校（関数・解析・幾何）
├─ 二次関数・不等式（1年）
│   30 二次関数 → 31 不等式
│
├─ 三角関数・指数対数・数列・ベクトル（2年）
│   32 三角関数 → 33 指数・対数
│   34 数列 → 35 ベクトル
│
└─ 極限・微分・積分（3年）
    36 極限 → 37 微分 → 38 積分
```

## 学習の前提関係

各トピックは以下の知識を前提としています。

- **かけ算(07)** ← たし算(01〜03)
- **わり算(08〜10)** ← かけ算(07)
- **分数(11〜16)** ← たし算・ひき算・かけ算・わり算(01〜10)
- **小数(17〜19)** ← たし算・ひき算・かけ算(01〜07)
- **割合(21)** ← 比(20)、かけ算・わり算
- **文字式(23)** ← 四則演算(01〜10)
- **一次方程式(24)** ← 文字式(23)、正負の数(22)
- **連立方程式(25)** ← 一次方程式(24)
- **一次関数(26)** ← 一次方程式(24)
- **確率(27)** ← 分数(11〜16)
- **平方根(28)** ← かけ算(07)
- **二次方程式(29)** ← 平方根(28)、一次方程式(24)
- **二次関数(30)** ← 二次方程式(29)、一次関数(26)
- **不等式(31)** ← 一次方程式(24)
- **三角関数(32)** ← 二次関数(30)
- **指数・対数(33)** ← 平方根(28)
- **数列(34)** ← 文字式(23)
- **ベクトル(35)** ← 三角関数(32)
- **極限(36)** ← 数列(34)、分数(11〜16)
- **微分(37)** ← 極限(36)、二次関数(30)
- **積分(38)** ← 微分(37)
