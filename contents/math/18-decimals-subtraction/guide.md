# 小数のひき算 - 解説

**参考学年**: 小学5年

**学習目的**: 小数点をそろえて、小数のひき算を正確に計算できるようにする

## ✏️ 小数のひき算のやり方

- 小数点の位置を **たてにそろえる** 📏
- 桁が足りないときは **0を補う**
- くり下がりに注意する

## 📖 例題と解説

### 例① 5.7 - 2.9 = ?

```
  5.7
- 2.9     ← くり下がりに注意
-----
  2.8 ✅
```

- 7 - 9 はひけない → 十の位から1くり下げる
- 17 - 9 = 8、4 - 2 = 2 → **2.8**

### 例② 4.6 - 1.75 = ?

```
  4.60    ← 桁が足りないので 0 を補う
- 1.75
------
  2.85 ✅
```

## 📌 コツ

- 小数点をそろえて、桁を合わせてから計算 📏
- くり下がりは整数のひき算と同じルール ✅
