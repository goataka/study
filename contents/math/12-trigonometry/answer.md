# 三角関数 - 解答

> **参考学年**: 高校2年

## 三角関数の値

| 問題 | 答え |
|------|------|
| sin 30° = | 1/2 |
| cos 60° = | 1/2 |
| tan 45° = | 1 |
| sin 90° = | 1 |
| cos 0° = | 1 |
| tan 60° = | √3 |

> **解説**: 30°, 45°, 60° の値は暗記が必要です。sin 30° = cos 60° = 1/2 のように、sin と cos は「90° - θ」の関係で値が入れ替わります。

## 三角関数の相互関係

| 問題 | 答え |
|------|------|
| sin²θ + cos²θ = | 1 |
| sin θ = 3/5のとき、cos θ = (0° < θ < 90°) | 4/5 |
| cos θ = 1/2のとき、sin θ = (0° < θ < 90°) | √3/2 |

> **解説**: sin²θ + cos²θ = 1 を変形して使います。sin θ = 3/5 のとき、cos²θ = 1 - (3/5)² = 1 - 9/25 = 16/25 なので cos θ = 4/5（0° < θ < 90° で正）です。

## 加法定理

| 問題 | 答え |
|------|------|
| sin(α + β) = | sin α cos β + cos α sin β |
| cos(α + β) = | cos α cos β - sin α sin β |
| sin(α - β) = | sin α cos β - cos α sin β |

> **解説**: 加法定理は三角関数の最も重要な公式です。sin は「サイン・コス・コス・サイン」、cos は「コス・コス・マイナス・サイン・サイン」と覚えます。
