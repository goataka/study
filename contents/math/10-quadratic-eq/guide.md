# 二次方程式

> **参考学年**: 中学3年

## 学習目的

因数分解や解の公式を使って二次方程式を解けるようになる。

## 解説

### 二次方程式とは

x² を含む方程式のことです。一般形は ax² + bx + c = 0 です。

### 因数分解で解く

式を (x - p)(x - q) = 0 の形にできれば、x = p, q が解です。

- 例：x² - 5x + 6 = 0
- (x - 2)(x - 3) = 0
- x = 2, 3

#### 因数分解のコツ

- 「たして b、かけて c」になる2つの数を探す
- x² - 5x + 6 → たして -5、かけて 6 → -2 と -3

### 解の公式

因数分解が難しいときは、解の公式を使います。

**x = (-b ± √(b² - 4ac)) / 2a**

- 例：x² + 4x + 3 = 0（a=1, b=4, c=3）
- x = (-4 ± √(16 - 12)) / 2 = (-4 ± 2) / 2
- x = -1, -3

### 判別式

b² - 4ac の値で解の個数がわかります。

- 正 → 異なる2つの実数解
- 0 → 重解（同じ解が2つ）
- 負 → 実数解なし
