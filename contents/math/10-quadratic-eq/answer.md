# 二次方程式 - 解答

> **参考学年**: 中学3年

## 因数分解で解く

| 問題 | 解 |
|------|------|
| x² - 5x + 6 = 0 | x = 2, 3 |
| x² - 7x + 12 = 0 | x = 3, 4 |
| x² - x - 6 = 0 | x = -2, 3 |
| x² + 5x + 6 = 0 | x = -2, -3 |
| x² - 9 = 0 | x = ±3 |

> **解説**: 「たして b の係数、かけて c」になる2つの数を探します。x² - 5x + 6 = (x-2)(x-3) = 0 なので x = 2, 3。x² - 9 = (x+3)(x-3) = 0 は「差の平方」の公式です。

## 解の公式で解く

| 問題 | 解 |
|------|------|
| x² + 4x + 3 = 0 | x = -1, -3 |
| x² - 6x + 5 = 0 | x = 1, 5 |
| x² + 2x - 8 = 0 | x = -4, 2 |
| 2x² - 5x + 2 = 0 | x = 1/2, 2 |

> **解説**: x = (-b ± √(b²-4ac)) / 2a に代入します。x² + 4x + 3 = 0 では a=1, b=4, c=3 なので x = (-4 ± √(16-12)) / 2 = (-4 ± 2) / 2 = -1, -3 です。
