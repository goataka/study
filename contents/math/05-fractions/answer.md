# 分数 - 解答

> **参考学年**: 小学4年

## 分数のたし算

| 問題 | 答え |
|------|------|
| 1/4 + 2/4 = | 3/4 |
| 1/3 + 1/3 = | 2/3 |
| 2/5 + 1/5 = | 3/5 |
| 3/8 + 2/8 = | 5/8 |
| 1/6 + 4/6 = | 5/6 |
| 2/7 + 3/7 = | 5/7 |

> **解説**: 分母が同じときは、分子だけをたします。分母はそのままです。例えば 1/4 + 2/4 は、分子の 1 + 2 = 3 なので 3/4 です。

## 分数のひき算

| 問題 | 答え |
|------|------|
| 3/4 - 1/4 = | 2/4 = 1/2 |
| 4/5 - 2/5 = | 2/5 |
| 5/6 - 1/6 = | 4/6 = 2/3 |
| 7/8 - 3/8 = | 4/8 = 1/2 |
| 4/7 - 2/7 = | 2/7 |
| 5/9 - 2/9 = | 3/9 = 1/3 |

> **解説**: 分母が同じときは、分子だけをひきます。答えが約分できるときは約分しましょう。例えば 3/4 - 1/4 = 2/4 は、分子と分母を2でわって 1/2 になります。
