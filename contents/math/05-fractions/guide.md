# 分数

> **参考学年**: 小学4年

## 学習目的

分数の意味を理解し、同じ分母の分数のたし算・ひき算ができるようになる。

## 解説

### 分数とは

分数は、1つのものをいくつかに等しく分けた数です。

- 1/4 は「1を4つに分けた1つ分」を表します
- 上の数を「分子（ぶんし）」、下の数を「分母（ぶんぼ）」といいます

### 分数のたし算

分母が同じ分数は、分子どうしをたします。

- 例：1/4 + 2/4 = 3/4（分母はそのまま、分子をたす）

### 分数のひき算

分母が同じ分数は、分子どうしをひきます。

- 例：3/4 - 1/4 = 2/4 = 1/2（分母はそのまま、分子をひく）

### 約分

分子と分母を同じ数でわって、簡単な分数にすることを約分といいます。

- 例：2/4 → 2÷2 / 4÷2 = 1/2
- 例：6/8 → 6÷2 / 8÷2 = 3/4
