# 音声変化練習ドリル - 解答

## リンキング、リダクション、フラッピング、アシミレーションの音を書きましょう

音声変化した部分の音をカタカナで書きましょう。

### リンキング（Linking）
音がつながる

| フレーズ | 音 |
|----------|------|
| check it out | チェキットアウト |
| turn it off | ターニットオフ |
| put it on | プティットオン |
| pick it up | ピキットアップ |
| take a look | テイカルック |
| not at all | ノタットール |
| an apple | アナップル |
| turn on | ターノン |

### リダクション（Reduction）
音が弱くなる・消える

| フレーズ | 音 |
|----------|------|
| want to | ワナ |
| going to | ゴナ / ガナ |
| have to | ハフタ |
| got to | ガタ |
| kind of | カインダ |
| sort of | ソータ |
| a lot of | アロダ / アロラ |
| out of | アウタ / アウラ |

### フラッピング（Flapping）
t/dがラ行になる

| フレーズ | 音 |
|----------|------|
| water | ワラー |
| better | ベラー |
| little | リロー |
| get it | ゲリット |
| what about | ワラバウト |
| put it | プリット |
| city | シリー |
| party | パーリー |

### アシミレーション（Assimilation）
音が混ざる

| フレーズ | 音 |
|----------|------|
| want you | ワンチュー |
| did you | ディジュー |
| would you | ウッジュー |
| miss you | ミッシュー |
| nice to meet you | ナイストゥミーチュー |
| let me | レミー |
| get you | ゲチュー |
| don't you | ドンチュー |
