# 音声変化練習ドリル

## リンキング、リダクション、フラッピング、アシミレーションの音を書きましょう

音声変化した部分の音をカタカナで書きましょう。

### リンキング（Linking）

音がつながる

| フレーズ | 音 |
|----------|------|
| check it out | 　　　　　　　　　　　　 |
| turn it off | 　　　　　　　　　　　　 |
| put it on | 　　　　　　　　　　　　 |
| pick it up | 　　　　　　　　　　　　 |
| take a look | 　　　　　　　　　　　　 |
| not at all | 　　　　　　　　　　　　 |
| an apple | 　　　　　　　　　　　　 |
| turn on | 　　　　　　　　　　　　 |

### リダクション（Reduction）

音が弱くなる・消える

| フレーズ | 音 |
|----------|------|
| want to | 　　　　　　　　　　　　 |
| going to | 　　　　　　　　　　　　 |
| have to | 　　　　　　　　　　　　 |
| got to | 　　　　　　　　　　　　 |
| kind of | 　　　　　　　　　　　　 |
| sort of | 　　　　　　　　　　　　 |
| a lot of | 　　　　　　　　　　　　 |
| out of | 　　　　　　　　　　　　 |

### アシミレーション（Assimilation）

となりの音に引っ張られて音が変わる

| フレーズ | 変化 | 音 |
|----------|------|------|
| meet you | t+y→チュ | 　　　　　　　　　　　　 |
| can't you | t+y→チュ | 　　　　　　　　　　　　 |
| would you | d+y→ジュ | 　　　　　　　　　　　　 |
| could you | d+y→ジュ | 　　　　　　　　　　　　 |
| miss you | s+y→シュ | 　　　　　　　　　　　　 |
| bless you | s+y→シュ | 　　　　　　　　　　　　 |
| ten people | n→m（唇音の前） | 　　　　　　　　　　　　 |
| in Paris | n→m（唇音の前） | 　　　　　　　　　　　　 |
