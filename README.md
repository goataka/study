# Goataka Study

子供向け学習アプリのリポジトリです。

## 学習アプリ

- [ドリル](support/README.md)
- [クイズ](https://goataka.github.io/study/app/) - 4択クイズ形式で学習
  - [英語クイズ](https://goataka.github.io/study/app/?subject=english) - 英語の4択クイズ
  - [数学クイズ](https://goataka.github.io/study/app/?subject=math) - 数学の4択クイズ

## GitHub Pages

GitHub Actions により、`support/` フォルダの内容と `app/` アプリが自動的に GitHub Pages へデプロイされます。

- **Pages URL**: <https://goataka.github.io/study/>
- **アプリ URL**: <https://goataka.github.io/study/app/>
- **サポート URL**: <https://goataka.github.io/study/support/>
- **ワークフロー**: `.github/workflows/jekyll-gh-pages.yml`
