# フラッピング練習ドリル - 解答

## フラッピングの音を書きましょう

t/dがラ行になる部分の音をカタカナで書きましょう。

| フレーズ | 音 | 変化した部分 |
|----------|------|--------------|
| water | ワ ラー | t → ラ |
| better | ベ ラー | tt → ラ |
| little | リ ロー | tt → ロ |
| get it | ゲ リッ ト | t → リ |
| what about | ワ ラ バウ ト | t → ラ |
| put it | プ リッ ト | t → リ |
| city | シ リー | t → リ |
| party | パー リー | t → リ |
