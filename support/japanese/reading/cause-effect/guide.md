# 因果関係

## 📖 因果関係とは

文章の中で、**原因（なぜそうなったか）と結果（どうなったか）のつながり**を示す関係のことです。論述文・説明文ではとくに重要です。

## 📝 代表的な接続語・表現

| 接続語・表現 | 使い方のポイント |
|-------------|----------------|
| だから | 前が原因、後が結果 |
| そのため | 前が原因、後が結果（やや書き言葉的） |
| したがって | 前が根拠、後が論理的な結論（書き言葉） |
| その結果 | 前の行為・出来事の結果を示す |
| なぜなら | 後が理由・原因の説明（順序が逆） |
| 〜のは〜からだ | 「〜のは」が結果、「〜からだ」が原因 |
| 〜ので | 前が原因・理由、後が結果 |
| 〜ため（に） | 前が原因・目的、後が結果 |

## 📊 例文で確認

> 「毎日練習を続けた。**そのため**、大会で優勝することができた。」

- 原因: 「毎日練習を続けた」
- 結果: 「大会で優勝することができた」

> 「彼が成功したのは、努力を惜しまなかった**からだ**。」

- 結果: 「彼が成功した」
- 原因: 「努力を惜しまなかった」
- 「〜のは〜からだ」の形に注意（結果→原因の順）

> 「**なぜなら**、土壌が乾燥していたからだ。」（前文: 「その地域では作物が育たなかった。」）

- 「なぜなら」が来たら、その後に**理由・原因**が書かれている

## 💡 読解でのポイント

- 「なぜ〜なのか」という問いに答えるとき、因果関係の接続語を手がかりにする
- **原因と結果を逆にしない**よう注意する（特に「なぜなら」は後が原因）
- 因果関係が複数連鎖することがある（A→B→C）
  - 例: 「気温が上がった→氷が溶けた→海面が上昇した」
- 原因を問う問題では「〜から」「〜ため」「〜ので」で終わる答えを作る
