# 対比関係

## 📖 対比関係とは

文章の中で、二つの事柄を**比べてその違いを明確にする**関係のことです。筆者の主張をより際立たせるために使われます。

## 📝 代表的な接続語

| 接続語 | 使い方のポイント |
|--------|----------------|
| しかし | 前の内容と反対・対立する内容を続ける |
| ところが | 予想に反する内容が続く（意外性がある） |
| 一方 | 一方の視点・立場を示す |
| それに対し | 前の内容と対照的な内容を示す |
| 逆に | 前の内容とは逆の内容を示す |
| 反対に | 前の内容の反対を示す |

## 📊 例文で確認

> 「都市では生活が便利で何でもすぐ手に入る。**しかし**、自然が少なく、静けさを感じる場所は限られている。」

- 「便利さ」と「自然の少なさ」を対比している
- 「しかし」の後に筆者が強調したい内容が来ることが多い

> 「兄は積極的で話し好きだ。**一方**、弟は内気で読書が好きだ。」

- 兄と弟の性格の違いを対比している
- 「一方」は二つの物事を並べて比較するときに使う

## 💡 読解でのポイント

- 対比の接続語が出たら、**A（前）とB（後）の違いは何か**を整理する
- 「しかし」「ところが」の後には筆者の**重要な主張**が来ることが多い
- 対比関係を整理するときは、**表にまとめる**と分かりやすい
  - 例: 「都市」vs「田舎」、「伝統」vs「革新」
- 対比されている二つの立場を**混同しない**ように注意する
