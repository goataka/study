# ことわざ（小学生）

## 📖 ことわざとは

- 昔から伝わる**生活の知恵や教え**を短い言葉で表したもの
- 教訓・経験・風刺が凝縮されている
- 実際の場面で使えると、表現が豊かになる

## 📝 例題

- 「**七転び八起き**」（ななころびやおき）
  - 意味: 何度失敗しても諦めずに立ち上がること
  - 使い方: 試験に落ちても七転び八起きの精神で頑張ろう

- 「**石の上にも三年**」（いしのうえにもさんねん）
  - 意味: 辛抱強く続ければいつか成果が出る
  - 使い方: 練習が辛くても、石の上にも三年——続けることが大切だ

## 📊 ことわざ一覧（小学生）

| ことわざ | 読み方 | 意味 |
|----------|--------|------|
| 七転び八起き | ななころびやおき | 何度失敗しても諦めずに立ち上がること |
| 猿も木から落ちる | さるもきからおちる | どんな名人でも失敗することがある |
| 石の上にも三年 | いしのうえにもさんねん | 辛抱強く続ければいつか成果が出る |
| 塵も積もれば山となる | ちりもつもればやまとなる | 少しずつの積み重ねが大きな結果につながる |
| 急いては事を仕損じる | せいてはことをしそんじる | 焦って急ぐと失敗しやすい |
| 善は急げ | ぜんはいそげ | 良いことはすぐに実行すべき |
| 笑う門には福来たる | わらうかどにはふくきたる | 明るく笑いが絶えない家には幸福が訪れる |
| 習うより慣れろ | ならうよりなれろ | 実際に経験することで上達する |
| 果報は寝て待て | かほうはねてまて | 幸運は焦らず気長に待つことが大事 |
| 備えあれば憂いなし | そなえあればうれいなし | 前もって準備しておけばいざというときに心配がない |

## 💡 学習のコツ

- ことわざの**場面・状況**をイメージして覚える
- 似た意味のことわざを比べて覚える
  - 「猿も木から落ちる」→「名人でも失敗する」という教え
- 反対の意味のことわざを比べると整理しやすい
  - 「善は急げ」⇔「急いては事を仕損じる」
- 実際の会話や作文で使ってみると定着しやすい
