# 🎨 形容詞（叙述用法）

形容詞は be 動詞の後に置いて、主語の状態や性質を表します。

## 基本パターン

The game is `fun`.

The game is `exciting`.

- be動詞（is / am / are / was / were）の**後**に置く
- 主語の状態・性質を説明する

## よく使う形容詞（叙述用法）

| 形容詞 | 意味 | 例文 |
|--------|------|------|
| `fun` | 楽しい | The game is `fun`. |
| `exciting` | わくわくする | The game is `exciting`. |
| `interesting` | 面白い | The stage is `interesting`. |
| `beautiful` | 美しい | The graphics are `beautiful`. |
| `difficult` | 難しい | The game is `difficult`. |
| `easy` | 簡単な | The level is `easy`. |
| `tired` | 疲れた | I am `tired`. |
| `happy` | 幸せな | She is `happy`. |

## 📝 例

| パターン | 文 |
|----------|------|
| be動詞 + 形容詞 | The game is fun. |
| be動詞 + 形容詞 | The game is (`exciting`). |
| be動詞 + 形容詞 | The stage is (`interesting`). |
| be動詞 + 形容詞 | The graphics are (`beautiful`). |
| be動詞 + 形容詞 | The weather is (`cold`). |
| be動詞 + 形容詞 | I am (`tired`). |
