# 定冠詞 `the`

定冠詞 `the` は「（その）特定の〜」を表します。

## 使う場面

I play `the` game every day.

- お互いにどれか分かるとき
- すでに話題に出たもの
- 世界に1つしかないもの（the sun, the moon, the earth）
- 序数・最上級の前（the first, the best）

## `a` / `an` と `the` の比較

| 最初の登場 | 2回目以降 |
|-----------|-----------|
| I saw `a` cat. | `The` cat was cute. |
| She bought `a` book. | `The` book is interesting. |

## よく使う表現

| 表現 | 意味 |
|------|------|
| `the` sun | 太陽 |
| `the` moon | 月 |
| `the` earth | 地球 |
| `the` morning | 朝（in the morning） |
| `the` first | 最初の |

## 📝 例

| パターン | 文 |
|----------|------|
| 特定のもの | I play the game every day. |
| 特定のもの | (`The`) game is fun. |
| 2回目の登場 | I saw a cat. (`The`) cat was cute. |
| 世界に1つ | I see (`the`) moon. |
| 序数の前 | She won (`the`) first prize. |
| 最上級の前 | It is (`the`) best game. |

## ✏️ 英作文解答

| 日本語 | 英語 |
|--------|------|
| そのゲームは楽しいです。 | The game is fun. |
| 私は毎日そのゲームをします。 | I play the game every day. |
| 私は月を見ます。 | I see the moon. |
| それは最高のゲームです。 | It is the best game. |
