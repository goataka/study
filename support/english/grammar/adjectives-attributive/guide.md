# 形容詞（限定用法）

形容詞は名詞の前に置いて、名詞を修飾します。

## 基本パターン

I play `fun` games.

I play a `new` game.

- 形容詞は名詞の**前**に置く
- `a` / `an` の後にも置ける: a `new` game

## よく使う形容詞（限定用法）

| 形容詞 | 意味 | 例文 |
|--------|------|------|
| `new` | 新しい | a `new` game |
| `old` | 古い | an `old` book |
| `big` | 大きい | a `big` cat |
| `small` | 小さい | a `small` dog |
| `long` | 長い | a `long` story |
| `fast` | 速い | a `fast` player |
| `tall` | 背が高い | a `tall` person |
| `interesting` | 面白い | an `interesting` game |

## 📝 例

| パターン | 文 |
|----------|------|
| 形容詞 + 名詞 | I play fun games. |
| a + 形容詞 + 名詞 | I play a (`new`) game. |
| 形容詞 + 名詞 | The game has (`long`) stages. |
| an + 形容詞 + 名詞 | He plays an (`interesting`) game. |
| 形容詞 + 名詞 | He is a (`tall`) player. |
| 形容詞 + 名詞 | This is a (`fast`) game. |

## ✏️ 英作文解答

| 日本語 | 英語 |
|--------|------|
| 私は面白いゲームをします。 | I play fun games. |
| 私は新しいゲームをします。 | I play a new game. |
| そのゲームには長いステージがあります。 | The game has long stages. |
| 彼は背の高いプレイヤーです。 | He is a tall player. |
