# ⚽ スポーツ・趣味

スポーツや趣味を表す単語を英語で覚えましょう。

## 単語一覧

| 英語 | 読み方 | 意味 |
|------|--------|------|
| `soccer` | サッカー | サッカー |
| `tennis` | テニス | テニス |
| `basketball` | バスケットボール | バスケットボール |
| `swimming` | スウィミング | 水泳 |
| `skiing` | スキーイング | スキー |
| `cycling` | サイクリング | サイクリング |
| `hiking` | ハイキング | ハイキング |
| `fishing` | フィッシング | 釣り |
| `painting` | ペインティング | 絵を描くこと |
| `cooking` | クッキング | 料理 |
| `reading` | リーディング | 読書 |
| `shopping` | ショッピング | 買い物 |
| `dancing` | ダンシング | 踊ること |
| `gardening` | ガーデニング | ガーデニング |
| `photography` | フォトグラフィー | 写真撮影 |

## 例文

- I play `soccer` every weekend. (毎週末サッカーをします。)
- She enjoys `cooking` with her mom. (お母さんと料理するのが好きです。)
- He is interested in `photography`. (写真撮影に興味があります。)
- My grandmother enjoys `gardening`. (祖母はガーデニングが好きです。)
