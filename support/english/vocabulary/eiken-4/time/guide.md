# 🕐 時間・日付

時間や日付に関する単語を英語で覚えましょう。

## 単語一覧

| 英語 | 読み方 | 意味 |
|------|--------|------|
| `Monday` | マンデイ | 月曜日 |
| `Tuesday` | チューズデイ | 火曜日 |
| `Wednesday` | ウェンズデイ | 水曜日 |
| `Thursday` | サーズデイ | 木曜日 |
| `Friday` | フライデイ | 金曜日 |
| `Saturday` | サタデイ | 土曜日 |
| `Sunday` | サンデイ | 日曜日 |
| `morning` | モーニング | 朝 |
| `afternoon` | アフタヌーン | 午後 |
| `evening` | イーブニング | 夕方 |
| `midnight` | ミッドナイト | 真夜中 |
| `yesterday` | イエスタデイ | 昨日 |
| `tomorrow` | トゥモロー | 明日 |
| `birthday` | バースデイ | 誕生日 |
| `holiday` | ホリデイ | 休日 |

## 例文

- I go to school on `Monday`. (月曜日に学校に行きます。)
- I eat breakfast in the `morning`. (朝に朝食を食べます。)
- My `birthday` is in March. (私の誕生日は3月です。)
- We have a `holiday` next week. (来週は休日があります。)
