# 💬 形容詞

気持ちや状態を表す形容詞を英語で覚えましょう。

## 単語一覧

| 英語 | 読み方 | 意味 |
|------|--------|------|
| `hungry` | ハングリー | 空腹な |
| `tired` | タイアード | 疲れた |
| `angry` | アングリー | 怒った |
| `sad` | サッド | 悲しい |
| `excited` | イクサイテッド | 興奮した |
| `scared` | スケアード | 怖がっている |
| `bored` | ボアード | 退屈した |
| `busy` | ビジー | 忙しい |
| `quiet` | クワイエット | 静かな |
| `loud` | ラウド | 騒がしい |
| `soft` | ソフト | 柔らかい |
| `hard` | ハード | 硬い |
| `clean` | クリーン | きれいな |
| `dirty` | ダーティー | 汚い |
| `fresh` | フレッシュ | 新鮮な |

## 例文

- I am `hungry`. (空腹です。)
- She looks `tired` today. (彼女は今日疲れて見えます。)
- He is `angry` about it. (彼はそのことに怒っています。)
- The vegetables are `fresh`. (野菜が新鮮です。)
