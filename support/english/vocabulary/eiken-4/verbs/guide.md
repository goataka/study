# 🏃 動作動詞

動作を表す動詞を英語で覚えましょう。

## 単語一覧

| 英語 | 読み方 | 意味 |
|------|--------|------|
| `cook` | クック | 料理する |
| `swim` | スウィム | 泳ぐ |
| `draw` | ドロー | 描く |
| `jump` | ジャンプ | 跳ぶ |
| `dance` | ダンス | 踊る |
| `sing` | シング | 歌う |
| `ride` | ライド | 乗る |
| `climb` | クライム | 登る |
| `carry` | キャリー | 運ぶ |
| `push` | プッシュ | 押す |
| `pull` | プル | 引く |
| `throw` | スロー | 投げる |
| `catch` | キャッチ | 捕まえる |
| `kick` | キック | 蹴る |
| `wash` | ウォッシュ | 洗う |

## 例文

- I like to `cook` food. (料理するのが好きです。)
- She can `swim` fast. (彼女は速く泳げます。)
- I like to `draw` pictures. (絵を描くのが好きです。)
- He can `throw` the ball far. (彼は遠くにボールを投げられます。)
