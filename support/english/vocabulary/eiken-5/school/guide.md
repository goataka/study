# 🏫 学校・文具

学校や文具に関する言葉を英語で覚えましょう。

## 単語一覧

| 英語 | 読み方 | 意味 |
|------|--------|------|
| `school` | スクール | がっこう |
| `classroom` | クラスルーム | きょうしつ |
| `desk` | デスク | つくえ |
| `chair` | チェア | いす |
| `book` | ブック | ほん |
| `pencil` | ペンシル | えんぴつ |
| `pen` | ペン | ペン |
| `eraser` | イレイザー | けしゴム |
| `bag` | バッグ | かばん |
| `notebook` | ノートブック | ノート |
| `teacher` | ティーチャー | せんせい |
| `student` | ステューデント | せいと |
| `library` | ライブラリー | としょかん |
| `door` | ドア | ドア |
| `window` | ウィンドー | まど |

## 例文

- I go to `school` every day. (まいにちがっこうへ行きます。)
- Write with a `pencil`. (えんぴつで書いてください。)
- I read books at the `library`. (わたしはとしょかんで本を読みます。)
- Open the `door`. (ドアをあけてください。)
