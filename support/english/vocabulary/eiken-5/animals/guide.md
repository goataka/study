# 🐾 動物

動物の名前を英語で覚えましょう。

## 単語一覧

| 英語 | 読み方 | 意味 |
|------|--------|------|
| `cat` | キャット | ねこ |
| `dog` | ドッグ | いぬ |
| `bird` | バード | とり |
| `fish` | フィッシュ | さかな |
| `rabbit` | ラビット | うさぎ |
| `horse` | ホース | うま |
| `cow` | カウ | うし |
| `pig` | ピッグ | ぶた |
| `bear` | ベア | くま |
| `monkey` | マンキー | さる |
| `elephant` | エレファント | ぞう |
| `lion` | ライオン | らいおん |
| `tiger` | タイガー | とら |
| `mouse` | マウス | ねずみ |
| `sheep` | シープ | ひつじ |

## 例文

- I have a `cat`. (わたしはねこを飼っています。)
- The `dog` runs fast. (そのいぬは速く走ります。)
- A `bird` can fly. (とりは空を飛べます。)
- The `elephant` has a long nose. (ぞうは長い鼻を持っています。)
