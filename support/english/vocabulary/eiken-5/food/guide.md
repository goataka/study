# 🍎 食べ物・飲み物

食べ物・飲み物の名前を英語で覚えましょう。

## 単語一覧

| 英語 | 読み方 | 意味 |
|------|--------|------|
| `apple` | アップル | りんご |
| `orange` | オレンジ | オレンジ |
| `banana` | バナナ | バナナ |
| `bread` | ブレッド | パン |
| `rice` | ライス | ごはん |
| `milk` | ミルク | ぎゅうにゅう |
| `water` | ウォーター | みず |
| `juice` | ジュース | ジュース |
| `tea` | ティー | おちゃ |
| `coffee` | コーヒー | コーヒー |
| `egg` | エッグ | たまご |
| `cake` | ケーキ | ケーキ |
| `fish` | フィッシュ | さかな |
| `meat` | ミート | にく |
| `salad` | サラダ | サラダ |

## 例文

- I eat an `apple` every day. (まいにちりんごを食べます。)
- She drinks `milk` in the morning. (かのじょは朝ぎゅうにゅうを飲みます。)
- I want some `water`. (みずがほしいです。)
- My birthday `cake` is sweet. (たんじょう日のケーキは甘いです。)
