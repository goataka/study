# 👨‍👩‍👧‍👦 家族・人

家族や人を表す言葉を英語で覚えましょう。

## 単語一覧

| 英語 | 読み方 | 意味 |
|------|--------|------|
| `father` | ファーザー | おとうさん |
| `mother` | マザー | おかあさん |
| `brother` | ブラザー | おにいさん・おとうとさん |
| `sister` | シスター | おねえさん・いもうとさん |
| `grandfather` | グランドファーザー | おじいさん |
| `grandmother` | グランドマザー | おばあさん |
| `friend` | フレンド | ともだち |
| `teacher` | ティーチャー | せんせい |
| `doctor` | ドクター | いしゃ |
| `child` | チャイルド | こども |
| `boy` | ボーイ | おとこのこ |
| `girl` | ガール | おんなのこ |
| `man` | マン | おとこのひと |
| `woman` | ウーマン | おんなのひと |
| `baby` | ベイビー | あかちゃん |

## 例文

- My `father` is a doctor. (わたしのおとうさんはいしゃです。)
- She is my best `friend`. (かのじょはわたしの親友です。)
- The `teacher` teaches English. (そのせんせいは英語をおしえます。)
- The `baby` is sleeping. (あかちゃんはねています。)
