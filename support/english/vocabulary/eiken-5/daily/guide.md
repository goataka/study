# 🌞 日常生活

日常生活に関する言葉を英語で覚えましょう。

## 単語一覧

| 英語 | 読み方 | 意味 |
|------|--------|------|
| `morning` | モーニング | あさ |
| `afternoon` | アフタヌーン | ひる・午後 |
| `evening` | イブニング | ゆうがた |
| `night` | ナイト | よる |
| `day` | デイ | ひ・日 |
| `week` | ウィーク | しゅう |
| `month` | マンス | つき・月 |
| `year` | イヤー | とし・年 |
| `time` | タイム | じかん |
| `breakfast` | ブレックファスト | あさごはん |
| `lunch` | ランチ | ひるごはん |
| `dinner` | ディナー | ゆうごはん |
| `sleep` | スリープ | ねる |
| `work` | ワーク | しごと・はたらく |
| `play` | プレイ | あそぶ・プレイ |

## 例文

- I get up in the `morning`. (わたしはあさ起きます。)
- I eat `breakfast` at seven. (7時にあさごはんを食べます。)
- My family eats `dinner` together. (かぞくはいっしょにゆうごはんを食べます。)
- Let's `play` soccer! (サッカーをあそびましょう！)
