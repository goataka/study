# 🏠 場所

場所を表す言葉を英語で覚えましょう。

## 単語一覧

| 英語 | 読み方 | 意味 |
|------|--------|------|
| `house` | ハウス | いえ |
| `park` | パーク | こうえん |
| `school` | スクール | がっこう |
| `hospital` | ホスピタル | びょういん |
| `store` | ストア | みせ |
| `library` | ライブラリー | としょかん |
| `station` | ステーション | えき |
| `city` | シティ | まち・都市 |
| `country` | カントリー | くに・いなか |
| `street` | ストリート | とおり |
| `room` | ルーム | へや |
| `garden` | ガーデン | にわ |
| `beach` | ビーチ | かいがん |
| `mountain` | マウンテン | やま |
| `river` | リバー | かわ |

## 例文

- I live in a `house`. (わたしはいえに住んでいます。)
- Children play in the `park`. (こどもたちはこうえんで遊びます。)
- I read books at the `library`. (わたしはとしょかんで本を読みます。)
- I swim at the `beach`. (わたしはかいがんで泳ぎます。)
