# 🎨 色・数

色と数の言葉を英語で覚えましょう。

## 単語一覧

| 英語 | 読み方 | 意味 |
|------|--------|------|
| `red` | レッド | あか |
| `blue` | ブルー | あお |
| `green` | グリーン | みどり |
| `yellow` | イエロー | きいろ |
| `white` | ホワイト | しろ |
| `black` | ブラック | くろ |
| `pink` | ピンク | ピンク |
| `purple` | パープル | むらさき |
| `orange` | オレンジ | オレンジ（色） |
| `brown` | ブラウン | ちゃいろ |
| `one` | ワン | 1（いち） |
| `two` | トゥー | 2（に） |
| `three` | スリー | 3（さん） |
| `four` | フォー | 4（し・よん） |
| `five` | ファイブ | 5（ご） |

## 例文

- The apple is `red`. (りんごはあかです。)
- The sky is `blue`. (そらはあおです。)
- Snow is `white`. (ゆきはしろです。)
- I have `one` cat. (ねこを1ぴき飼っています。)
