# 👔 職業

職業を表す単語を英語で覚えましょう。

## 単語一覧

| 英語 | 読み方 | 意味 |
|------|--------|------|
| `doctor` | ドクター | 医者 |
| `nurse` | ナース | 看護師 |
| `teacher` | ティーチャー | 先生 |
| `engineer` | エンジニア | エンジニア |
| `lawyer` | ロイヤー | 弁護士 |
| `pilot` | パイロット | パイロット |
| `chef` | シェフ | シェフ |
| `journalist` | ジャーナリスト | ジャーナリスト |
| `scientist` | サイエンティスト | 科学者 |
| `firefighter` | ファイアーファイター | 消防士 |
| `farmer` | ファーマー | 農家 |
| `artist` | アーティスト | 芸術家 |
| `musician` | ミュージシャン | 音楽家 |
| `accountant` | アカウンタント | 会計士 |
| `architect` | アーキテクト | 建築家 |

## 例文

- My father is a `doctor` at the hospital. (父は病院で医者をしています。)
- She is a `scientist` who studies climate. (気候を研究する科学者です。)
- The `firefighter` put out the fire. (消防士が火を消しました。)
- The `architect` designed a new building. (建築家が新しい建物を設計しました。)
