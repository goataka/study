# 🌿 自然・環境

自然や環境に関する単語を英語で覚えましょう。

## 単語一覧

| 英語 | 読み方 | 意味 |
|------|--------|------|
| `mountain` | マウンテン | 山 |
| `river` | リバー | 川 |
| `ocean` | オーシャン | 海 |
| `forest` | フォレスト | 森 |
| `valley` | バレー | 谷 |
| `desert` | デザート | 砂漠 |
| `island` | アイランド | 島 |
| `volcano` | ボルケーノ | 火山 |
| `earthquake` | アースクェイク | 地震 |
| `pollution` | ポリューション | 汚染 |
| `environment` | エンバイロンメント | 環境 |
| `climate` | クライメット | 気候 |
| `wildlife` | ワイルドライフ | 野生動物 |
| `plant` | プラント | 植物 |
| `soil` | ソイル | 土 |

## 例文

- We climbed the `mountain` last summer. (去年の夏、山に登りました。)
- We must protect the `environment`. (環境を守らなければなりません。)
- `Climate` change is affecting us all. (気候変動はみんなに影響しています。)
- We should protect `wildlife`. (野生動物を守るべきです。)
