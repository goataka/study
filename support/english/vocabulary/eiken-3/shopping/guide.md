# 🛒 買い物・お金

買い物やお金に関する単語を英語で覚えましょう。

## 単語一覧

| 英語 | 読み方 | 意味 |
|------|--------|------|
| `price` | プライス | 値段 |
| `discount` | ディスカウント | 割引 |
| `receipt` | リシート | 領収書 |
| `wallet` | ウォレット | 財布 |
| `product` | プロダクト | 商品 |
| `expensive` | イクスペンシブ | 高価な |
| `cheap` | チープ | 安い |
| `sale` | セイル | セール |
| `store` | ストア | お店 |
| `customer` | カスタマー | お客さん |
| `tax` | タックス | 税金 |
| `change` | チェンジ | お釣り |
| `advertisement` | アドバタイズメント | 広告 |
| `purchase` | パーチェス | 購入 |
| `refund` | リファンド | 返金 |

## 例文

- What is the `price` of this bag? (このバッグの値段はいくらですか？)
- I got a 20% `discount` on this item. (この商品は20%割引になりました。)
- Everything is 50% off in the `sale`. (セールで全品50%オフです。)
- I want a `refund` for this broken item. (この壊れた商品の返金を求めたいです。)
