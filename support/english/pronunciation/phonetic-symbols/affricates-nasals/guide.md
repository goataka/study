# 発音記号 — 破擦音・鼻音

## 📚 破擦音・鼻音とは

**破擦音**: 息を一度止めてから摩擦音として出す（破裂+摩擦）
**鼻音**: 鼻から息を出す音

## 📌 破擦音一覧

| 発音記号 | 音のイメージ | 例 |
|----------|-------------|-----|
| `/tʃ/` | 「チュ」 | `chat`（チャット） |
| `/dʒ/` | 「ジュ」（破裂あり） | `jam`（ジャム） |

## 📌 鼻音一覧

| 発音記号 | 音のイメージ | 例 |
|----------|-------------|-----|
| `/m/` | 唇を閉じた鼻音 | `man`（マン） |
| `/n/` | 舌先を歯茎に当てた鼻音 | `not`（ノット） |
| `/ŋ/` | 鼻にかかった「ング」 | `sing`（スィング） |

## 📝 例題

🔤 `chat` → `/tʃæt/` → `チュ ア ト`（止めてから出す「チュ」）

- 💡 `/tʃ/` と `/dʒ/` は破裂のある「チュ」と「ジュ」
- 💡 `/ŋ/` は語末に来ることが多く、後に「グ」の破裂は出ない（sing → スィン**グ**ではなく スィング）

## 📝 例

| 発音記号 | 音のイメージ | 代表的な単語 |
|----------|-------------|-------------|
| `/tʃ/` | チュ（無声） | chat, chair, teach |
| `/dʒ/` | ジュ（有声・破裂） | jam, jump, age |
| `/m/` | ム（唇・鼻音） | man, him, some |
| `/n/` | ン（歯茎・鼻音） | not, can, fun |
| `/ŋ/` | ング（奥・鼻音） | sing, ring, long |
