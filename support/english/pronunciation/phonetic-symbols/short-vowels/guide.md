# 発音記号 — 短母音

## 📚 短母音とは

- 短く発音する母音
- 日本語の「アイウエオ」より口の形が重要

## 📌 短母音一覧

| 発音記号 | 音のイメージ | 例 |
|----------|-------------|-----|
| `/ɪ/` | 短くゆるい「イ」 | `sit`（シット） |
| `/e/` | 日本語の「エ」に近い | `bed`（ベッド） |
| `/æ/` | 「ア」と「エ」の間（口を横に大きく開ける） | `cat`（キャット） |
| `/ɑ/` | 口を大きく開けた短い「ア」（米英語） | `hot`（ハット） |
| `/ʌ/` | 口を軽く縦に開けた短い「ア」 | `cut`（カット） |
| `/ʊ/` | 短くゆるい「ウ」 | `book`（ブック） |
| `/ɒ/` | 口を丸く開けた短い「オ」（英英語） | `not`（ノット） |
| `/ə/` | 力を抜いた最も弱い「ア」（シュワー） | `about`（アバウト） |

## 📝 例題

🔤 `cat` → `/kæt/` → `ク ア ト`（「ア」は口を横に大きく開ける）

- 💡 `/æ/` と `/ʌ/` は似ているが、`/æ/` は口を横に開ける、`/ʌ/` は軽く縦に開ける
- 💡 `/ə/` はシュワーと呼ばれ、英語で最も頻繁に使われる母音

## 📝 例

| 発音記号 | 音のイメージ | 代表的な単語 |
|----------|-------------|-------------|
| `/ɪ/` | イ（短い・ゆるい） | sit, hit, big |
| `/e/` | エ（日本語に近い） | bed, get, help |
| `/æ/` | ア（口を横に開ける） | cat, apple, hat |
| `/ɑ/` | ア（口を大きく開ける・米） | hot, top, stop |
| `/ʌ/` | ア（軽く縦に開ける） | cut, sun, cup |
| `/ʊ/` | ウ（短い・ゆるい） | book, cook, put |
| `/ɒ/` | オ（口を丸く開ける・英） | not, dog, hot |
| `/ə/` | ア（力を抜く・シュワー） | about, banana, sofa |
