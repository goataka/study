# 発音記号 — 長母音

## 📚 長母音とは

- 「ː」記号が付いた母音は長く伸ばして発音する
- 短母音と比べて長さだけでなく口の形も違うことが多い

## 📌 長母音一覧

| 発音記号 | 音のイメージ | 例 |
|----------|-------------|-----|
| `/iː/` | 長くはっきりした「イー」（唇を横に引く） | `see`（スィー） |
| `/uː/` | 長くはっきりした「ウー」（唇を丸める） | `too`（トゥー） |
| `/ɑː/` | 口を大きく開けた長い「アー」 | `far`（ファー） |
| `/ɔː/` | 口を丸めた長い「オー」 | `law`（ロー） |

## 📝 例題

🔤 `see` → `/siː/` → `ス イー`（唇を横に引いて長く「イー」）

- 💡 `/iː/` と `/ɪ/` は長さだけでなく口の形も違う：`/iː/` は口を横に引く
- 💡 `/uː/` は唇を丸めて前に突き出す（`/ʊ/` より長く強い）

## 📝 例

| 発音記号 | 音のイメージ | 代表的な単語 |
|----------|-------------|-------------|
| `/iː/` | イー（長い） | see, feel, team |
| `/uː/` | ウー（長い） | too, food, blue |
| `/ɑː/` | アー（口を大きく開ける） | far, car, bar |
| `/ɔː/` | オー（口を丸める） | law, talk, all |
