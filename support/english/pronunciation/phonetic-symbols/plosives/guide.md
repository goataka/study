# 発音記号 — 破裂音

## 📚 破裂音とは

- 息を一度止めてから開放する音（「爆発音」）
- 有声（声帯を振動）と無声（声帯を振動させない）のペアがある

## 📌 破裂音一覧

| 発音記号 | 音のイメージ | 例 |
|----------|-------------|-----|
| `/p/` | 唇を閉じて開く無声音 | `pen`（ペン） |
| `/b/` | 唇を閉じて開く有声音 | `bad`（バッド） |
| `/t/` | 舌先を歯茎に当てる無声音 | `tea`（ティー） |
| `/d/` | 舌先を歯茎に当てる有声音 | `did`（ディッド） |
| `/k/` | 舌の奥を使う無声音 | `cat`（キャット） |
| `/ɡ/` | 舌の奥を使う有声音 | `go`（ゴー） |
| `/ʔ/` | 声門閉鎖音（のどを閉じて開く） | `bottle`（ボ`ʔ`ル） |

## 📝 例題

🔤 `pen` → `/pen/` → `プ エ ヌ`（唇を閉じてから一気に開く）

- 💡 `/p/` と `/b/` は同じ唇の動き、違いは声帯の振動（有声・無声）
- 💡 `/ʔ/` は主にアメリカ英語で語中の `t` が弱化した場合に現れる（bottle, button）

## 📝 例

| 発音記号 | 音のイメージ | 代表的な単語 |
|----------|-------------|-------------|
| `/p/` | プ（唇・無声） | pen, cup, top |
| `/b/` | ブ（唇・有声） | bad, cab, job |
| `/t/` | ト（歯茎・無声） | tea, cat, hit |
| `/d/` | ド（歯茎・有声） | did, bad, red |
| `/k/` | ク（奥・無声） | cat, back, key |
| `/ɡ/` | グ（奥・有声） | go, bag, big |
| `/ʔ/` | 声門閉鎖（米のt弱化） | bottle, button |
