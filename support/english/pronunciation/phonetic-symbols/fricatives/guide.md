# 発音記号 — 摩擦音

## 📚 摩擦音とは

- 息を狭い隙間から出す「擦れる音」
- 日本語にない音が多く、英語学習で重要なポイント

## 📌 摩擦音一覧

| 発音記号 | 音のイメージ | 例 |
|----------|-------------|-----|
| `/f/` | 上の歯を下唇に当てる無声音 | `fan`（ファン） |
| `/v/` | 上の歯を下唇に当てる有声音 | `van`（ヴァン） |
| `/θ/` | 舌先を上の歯に当てる無声音（「ス」に近い） | `think`（シンク） |
| `/ð/` | 舌先を上の歯に当てる有声音（「ズ」に近い） | `this`（ジス） |
| `/s/` | 「ス」 | `see`（スィー） |
| `/z/` | 「ズ」 | `zoo`（ズー） |
| `/ʃ/` | 「シュ」 | `she`（シー） |
| `/ʒ/` | 「ジュ」（濁った `/ʃ/`） | `vision`（ヴィジョン） |
| `/h/` | 息だけを出す「ハ」行 | `he`（ヒー） |

## 📝 例題

🔤 `think` → `/θɪŋk/` → `ス イ ン ク`（舌先を歯に当てて出す音）

- 💡 `/θ/` と `/ð/` は同じ「舌を噛む動作」で、`/θ/` は無声、`/ð/` は有声
- 💡 `/f/` と `/v/` は上の歯を下唇に当てる：`/f/` は無声、`/v/` は有声

## 📝 例

| 発音記号 | 音のイメージ | 代表的な単語 |
|----------|-------------|-------------|
| `/f/` | フ（歯・唇・無声） | fan, off, leaf |
| `/v/` | ヴ（歯・唇・有声） | van, love, give |
| `/θ/` | ス（舌先を噛む・無声） | think, three, tooth |
| `/ð/` | ズ（舌先を噛む・有声） | this, that, the |
| `/s/` | ス（無声） | see, bus, pass |
| `/z/` | ズ（有声） | zoo, buzz, nose |
| `/ʃ/` | シュ（無声） | she, shoe, wish |
| `/ʒ/` | ジュ（有声） | vision, measure |
| `/h/` | ハ行（息） | he, hat, behind |
