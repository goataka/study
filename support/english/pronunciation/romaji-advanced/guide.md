# 🎌 拡張ローマ字（濁音・半濁音・拗音）

## 濁音

- 「゛」（濁点）がつくと有声音になる（のどが震える音）
- 例: か(ka) → が(ga)

| ひらがな | ローマ字 | 元の清音 |
|----------|----------|---------|
| が | `ga` | か(ka) |
| ぎ | `gi` | き(ki) |
| ぐ | `gu` | く(ku) |
| げ | `ge` | け(ke) |
| ご | `go` | こ(ko) |
| ざ | `za` | さ(sa) |
| じ | `ji` | し(shi) |
| ず | `zu` | す(su) |
| ぜ | `ze` | せ(se) |
| ぞ | `zo` | そ(so) |
| だ | `da` | た(ta) |
| ぢ | `ji` | ち(chi) |
| づ | `zu` | つ(tsu) |
| で | `de` | て(te) |
| ど | `do` | と(to) |
| ば | `ba` | は(ha) |
| び | `bi` | ひ(hi) |
| ぶ | `bu` | ふ(fu) |
| べ | `be` | へ(he) |
| ぼ | `bo` | ほ(ho) |

## 半濁音

- 「゜」（半濁点）がつくと `p` の音になる
- は行のみ: ぱ(pa)・ぴ(pi)・ぷ(pu)・ぺ(pe)・ぽ(po)

| ひらがな | ローマ字 | 元の清音 |
|----------|----------|---------|
| ぱ | `pa` | は(ha) |
| ぴ | `pi` | ひ(hi) |
| ぷ | `pu` | ふ(fu) |
| ぺ | `pe` | へ(he) |
| ぽ | `po` | ほ(ho) |

## 拗音

- 小さい「ゃ・ゅ・ょ」を組み合わせた音（2文字で1音）
- 例：`きゃく`（客） → `kya``ku`　`しゃしん`（写真） → `sha``shin`
- 💡 `じ` は `ji`（`zi` ではない）　`ぢ` も `ji`、`づ` も `zu`

### 清音の拗音

| ひらがな | ローマ字 |
|----------|----------|
| きゃ | `kya` |
| きゅ | `kyu` |
| きょ | `kyo` |
| しゃ | `sha` |
| しゅ | `shu` |
| しょ | `sho` |
| ちゃ | `cha` |
| ちゅ | `chu` |
| ちょ | `cho` |
| にゃ | `nya` |
| にゅ | `nyu` |
| にょ | `nyo` |
| ひゃ | `hya` |
| ひゅ | `hyu` |
| ひょ | `hyo` |
| みゃ | `mya` |
| みゅ | `myu` |
| みょ | `myo` |
| りゃ | `rya` |
| りゅ | `ryu` |
| りょ | `ryo` |

### 濁音・半濁音の拗音

| ひらがな | ローマ字 |
|----------|----------|

| ぎょ | `gyo` |

| じょ | `jo` |
| びゃ | `bya` |
| びゅ | `byu` |
| びょ | `byo` |

| ぴょ | `pyo` |
