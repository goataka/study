# 拡張ローマ字（濁音・半濁音・拗音）

## 濁音とは

- 「゛」（濁点）がついたひらがなのローマ字表記
- 清音（基本の音）に濁点がつくと、有声音（のどが震える音）になる

## 濁音の一覧

| ひらがな | ローマ字 | 元の清音 |
|----------|----------|---------|
| が | `ga` | か(ka) |
| ぎ | `gi` | き(ki) |
| ぐ | `gu` | く(ku) |
| げ | `ge` | け(ke) |
| ご | `go` | こ(ko) |
| ざ | `za` | さ(sa) |
| じ | `ji` | し(shi) |
| ず | `zu` | す(su) |
| ぜ | `ze` | せ(se) |
| ぞ | `zo` | そ(so) |
| だ | `da` | た(ta) |
| ぢ | `ji` | ち(chi) |
| づ | `zu` | つ(tsu) |
| で | `de` | て(te) |
| ど | `do` | と(to) |
| ば | `ba` | は(ha) |
| び | `bi` | ひ(hi) |
| ぶ | `bu` | ふ(fu) |
| べ | `be` | へ(he) |
| ぼ | `bo` | ほ(ho) |

## 半濁音とは

- 「゜」（半濁点）がついたひらがなのローマ字表記
- は行に半濁点がつくと `p` の音になる

## 半濁音の一覧

| ひらがな | ローマ字 | 元の清音 |
|----------|----------|---------|
| ぱ | `pa` | は(ha) |
| ぴ | `pi` | ひ(hi) |
| ぷ | `pu` | ふ(fu) |
| ぺ | `pe` | へ(he) |
| ぽ | `po` | ほ(ho) |

## 拗音とは

- 小さい「ゃ・ゅ・ょ」を組み合わせた音
- 2文字で1つの音を表す

## 例題

🇯🇵 `きゃく`（客） → `kya``ku`

🇯🇵 `しゃしん`（写真） → `sha``shi``n`

- 💡 `じ` は `ji` と書く（`zi` ではない）
- 💡 `ぢ` は `ji`、`づ` は `zu` と書く（`じ`・`ず` と同じ発音のため）

## 拗音の一覧

### 清音の拗音

| ひらがな | ローマ字 |
|----------|----------|
| きゃ | `kya` |
| きゅ | `kyu` |
| きょ | `kyo` |
| しゃ | `sha` |
| しゅ | `shu` |
| しょ | `sho` |
| ちゃ | `cha` |
| ちゅ | `chu` |
| ちょ | `cho` |
| にゃ | `nya` |
| にゅ | `nyu` |
| にょ | `nyo` |
| ひゃ | `hya` |
| ひゅ | `hyu` |
| ひょ | `hyo` |
| みゃ | `mya` |
| みゅ | `myu` |
| みょ | `myo` |
| りゃ | `rya` |
| りゅ | `ryu` |
| りょ | `ryo` |
