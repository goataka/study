# 五十音

## 📚 五十音とは

- 日本語の基本的な音節を一覧にしたもの
- ひらがな（日本語の音）を英字（ローマ字）で表す方法も学ぶ
- 英語とは発音が異なるので注意！

|--|--|--|--|--|--|
| あ行 | あ(a) | い(i) | う(u) | え(e) | お(o) |
| か行 | か(ka) | き(ki) | く(ku) | け(ke) | こ(ko) |
| さ行 | さ(sa) | し(shi) | す(su) | せ(se) | そ(so) |
| た行 | た(ta) | ち(chi) | つ(tsu) | て(te) | と(to) |
| な行 | な(na) | に(ni) | ぬ(nu) | ね(ne) | の(no) |
| は行 | は(ha) | ひ(hi) | ふ(fu) | へ(he) | ほ(ho) |
| ま行 | ま(ma) | み(mi) | む(mu) | め(me) | も(mo) |

## 📌 基本ルール

- 子音 + 母音（a/i/u/e/o）で1つの音になる
- 例: `k` + `a` = `ka`（か）

## 📌 特殊な書き方

| ひらがな | ローマ字 | ポイント |
|----------|----------|---------|
| し | `shi` | `si` ではない |
| ち | `chi` | `ti` ではない |
| つ | `tsu` | `tu` ではない |
| ふ | `fu` | `hu` ではない |
| を | `wo` | `o` ではない |
| ん | `n` | 単独の `n` |

## 📌 っ（促音）

- 小さい「っ」は、次の子音を2つ重ねて書く

| ひらがな例 | ローマ字 |
|------------|----------|
| きって | `kitte` |
| いっぱい | `ippai` |
| いっしょ | `issho` |
| きっぱり | `kippari` |

## 📌 ー（長音符）

- 「ー」や長くのばす音は、前の母音をのばして読む
- ローマ字では同じ母音を2つ並べることが多い
- ただし、`おう` は `ou`、`おお` は `oo` と書くことがある

| ひらがな例 | ローマ字 |
|------------|----------|
| おかあさん | `okaasan` |
| おにいさん | `oniisan` |
| くうき | `kuuki` |
| おねえさん | `oneesan` |
| おとうさん | `otousan` |
| コーヒー（カタカナ） | `koohii` |

## 📝 例題

🇯🇵 `さくら` → `s`a`k`u`r`a

🇯🇵 `ありがとう` → `a`ri`g`a`t`o`u`

- 💡 `n` の後に母音が続く場合は `n'` と区切ることがある（例: `ho-n'ya`）
