# 英語の記号の読み書き

## 📚 英語の記号とは

- 英語の文章でよく使われる句読点や特殊文字のこと
- 記号にはそれぞれ英語名（カタカナ読み）がある

## 📝 例題

🔣 `.` → `ピリオド` / `?` → `クエスチョンマーク` / `@` → `アットマーク`

## 📝 例

| 記号 | 読み方（カタカナ） | 使い方 |
|------|------------------|--------|
| `.` | ピリオド | 文の終わり（I like cats.） |
| `,` | カンマ | 句読点・列挙（cats, dogs, birds） |
| `?` | クエスチョンマーク | 疑問文の終わり（What is it?） |
| `!` | エクスクラメーションマーク | 感嘆・強調（Wow!） |
| `:` | コロン | 説明・列挙の前（Fruits: apples, oranges） |
| `;` | セミコロン | 文と文をつなぐ（I like cats; they are cute） |
| `'` | アポストロフィー | 短縮形・所有格（don't, Mike's） |
| `"` | ダブルクォーテーション | 会話文・引用（He said "hello"） |
| `@` | アットマーク | メールアドレス（<name@example.com>） |
| `#` | ハッシュ | ハッシュタグ（#study） |
| `*` | アスタリスク | 注釈・強調（*注意） |
| `&` | アンパサンド | and の意味（pen & paper） |
| `/` | スラッシュ | URL・パス（https://） |
| `-` | ハイフン | 単語結合（well-known） |
| `_` | アンダースコア | ファイル名など（my_file） |
