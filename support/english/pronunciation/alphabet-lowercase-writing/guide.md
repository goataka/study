# アルファベット小文字の書き

## 📚 小文字の書き方練習

- 読み方（名前）を聞いて、正しい小文字を書く練習
- アルファベット26文字すべての小文字（a〜z）を覚えよう

## 📝 例題

🔤 `えい` → a / `びー` → b / `ぜっと` → z

- 💡 形が似ている文字に注意: b/d、p/q、m/n、u/v

## 📝 例

| 読み方（ひらがな） | 小文字 |
|-------------------|--------|
| えい | a |
| びー | b |
| しー | c |
| でぃー | d |
| いー | e |
| えふ | f |
| じー | g |
| えいち | h |
| あい | i |
| じぇい | j |
| けー | k |
| える | l |
| えむ | m |
| えぬ | n |
| おー | o |
| ぴー | p |
| きゅー | q |
| あーる | r |
| えす | s |
| てぃー | t |
| ゆー | u |
| ぶい | v |
| だぶりゅー | w |
| えっくす | x |
| わい | y |
| ぜっと | z |
