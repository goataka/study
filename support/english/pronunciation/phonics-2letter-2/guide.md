# フォニックス（2文字）その２

## 2文字の組み合わせ（ダイグラフ）

2つの文字が合わさって1つの音になる。

### 子音ダイグラフ

| 文字 | 音 | 例 |
|------|----|----|
| `sh` | シュ | `sh`op, di`sh` |
| `ch` | チ | `ch`in, pea`ch` |
| `th` | ザ（有声）/ ス（無声） | `th`ey / wi`th` |
| `ph` | フ | `ph`oto |
| `wh` | ホウ/ウ | `wh`ip |

- 💡 `th` は2種類ある：舌を前歯に当てて出す音
  - 有声（ザ行）: `th`ey, `th`em, `th`ose
  - 無声（ス行）: wi`th`, tee`th`, `th`in

### 母音ダイグラフ

| 文字 | 音 | 例 |
|------|----|----|
| `oo`（長い） | ウー | f`oo`d |
| `oo`（短い） | ウ | f`oo`t |
| `ee` | イー | f`ee`t |
| `ai` | エイ | s`ai`l |
| `oa` | オウ | r`oa`d |

## 📌 マジックE のルール

語末の `e` は発音しないが、その前の母音を「長く（名前読み）」にする。

| 短い音 | マジックE | 音の変化 |
|--------|-----------|---------|
| lat | l`a`te | ア → エイ |
| pin | p`i`ne | イ → アイ |
| rop | r`o`pe | ア → オウ |
| rul | r`u`le | ア → ユー |

## 📝 例題

🔤 `sh`op → `シュ` / s`ai`l → `エイ` / p`i`ne → `アイ`

## 📝 例

### 2文字の組み合わせ

| 単語 | 単語全体 | 意味 |
|------|----------|------|
| `sh`op | `シュ` オ プ → シュオプ | みせ |
| di`sh` | ドゥ イ `シュ` → ドゥイシュ | さら |
| `ch`in | `チ` イ ヌ → チイヌ | あご |
| pea`ch` | プ イー `チ` → プイーチ | もも |
| `th`ey | `ザ` エイ → ザエイ | かれら |
| wi`th` | ウ イ `ス` → ウイス | ～といっしょに |
| `ph`oto | `フ` オウ トゥ → フォウトゥ | しゃしん |
| `wh`ip | `ホウ` イ プ → ホウイプ | むち |
| f`oo`d | フ `ウー` ドゥ → フウードゥ | たべもの |
| f`oo`t | フ `ウ` ト → フウト | あし（足） |
| f`ee`t | フ `イー` ト → フイート | あし（複数） |
| s`ai`l | ス `エイ` ル → スエイル | ほ（帆） |
| r`oa`d | ル `オウ` ドゥ → ルオウドゥ | みち |

### マジックE

| 単語 | 単語全体 | 意味 |
|------|----------|------|
| l`a`te | ル `エイ` ト → ルエイト | おそい |
| t`a`pe | トゥ `エイ` プ → トゥエイプ | テープ |
| p`i`ne | プ `アイ` ヌ → プアイヌ | まつ（松） |
| t`i`me | トゥ `アイ` ム → トゥアイム | じかん |
| r`o`pe | ル `オウ` プ → ルオウプ | なわ |
| c`o`de | ク `オウ` ドゥ → クオウドゥ | コード |
| r`u`le | ル `ユー` ル → ルユール | きそく |
| t`u`ne | トゥ `ユー` ヌ → トゥユーヌ | きょく |
