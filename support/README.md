# サポート

[学習アプリ](../)のサポートページです。

## ⚠️ ご利用の前に

- **無料**で利用できます
- **サーバー通信は不要**です（問題データの初回読み込みのみ通信が発生します）
- **原則サポートなし**です。不具合・要望への対応は保証できません
- 学習データはお使いのブラウザにのみ保存されます。**キャッシュをすべて破棄するとデータが失われます**
