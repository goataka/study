# サポート

## 🚀 はじめに

このページは、学習アプリの使い方と参考情報をまとめたガイドです。

## ✨ 特徴

- **無料**で使えます。
- **ログイン不要**です。

## ⚠️ 免責

- **原則サポートなしです。**
- 不具合・要望への対応は保証できません。
- 学習データはお使いのブラウザにのみ保存されます。
- **キャッシュをすべて破棄するとデータが失われます。**

## 📌 注意点

- 大切なデータは**定期的にバックアップ**することをおすすめします。
- バックアップは「管理」タブ → 「✅ 管理」→ 「📤 エクスポート」から実行できます。
- コンテンツ一覧は [ガイド（コンテンツ）](../#subject=support&supportMenu=contents) から開けます。
