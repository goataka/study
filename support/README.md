# サポート

## ✨ 特徴

- **無料**で使えます。
- **ログイン不要**です。

## ⚠️ 免責

- アプリの動作・コンテンツの正誤について保証はできません。
- 操作方法・データ復旧などへの対応はできません。
- 不具合・要望への回答は体制的に行えません。

## 📌 注意点

- 学習データはお使いのブラウザにのみ保存されます。
- キャッシュをすべて破棄するとデータが失われます。
- 大切なデータは定期的にバックアップすることをおすすめします。
- バックアップは「⚙️ 管理」タブ → 「📦 管理」→ 「📤 保存」から実行できます。
