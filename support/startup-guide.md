## 🚀 使い始め方

### ステップ1：アプリを開く

[学習アプリ](../../) にアクセスします。

### ステップ2：名前を設定する

ヘッダーのユーザー名ボタン（👤）をクリックして、自分の名前を入力します。

- 名前を入力して ✓ ボタンを押すと保存されます
- 名前は学習記録に表示されます

### ステップ3：教科を選ぶ

アプリを開くと、最初に「🌐 総合」タブが表示されます。ここから学習する教科・単元を選べます。

画面上部のタブで教科を切り替えられます：

- **🌐 総合** ─ 全教科のおすすめ単元を表示（初期表示）
- **📚 英語** ─ 発音・文法・語彙
- **🔢 数学** ─ 算数から高校数学まで
- **📖 国語** ─ 漢字・ことわざ・読解

### ステップ4：単元を選ぶ

教科タブによって単元の選び方が異なります：

#### 総合タブ（おすすめ単元から選ぶ）

「🌐 総合」タブでは、未学習・学習中の単元がおすすめとして表示されます。どこから始めたらよいか分からない場合はここから始めましょう。

#### 教科タブ（単元一覧から選ぶ）

「📚 英語」「🔢 数学」「📖 国語」などの教科タブを選ぶと、教科ごとの単元一覧が表示されます。学習したい単元を直接選べます。

### ステップ5：問題に挑戦

ランダムな10問の4択問題が出題されます。選択肢をタップ（クリック）して回答しましょう。

## 🎯 基本的な操作

| 操作 | 説明 |
|------|------|
| 教科・単元を選択 | トップ画面から学習内容を選ぶ |
| 選択肢をタップ | 4択から正しいと思う答えを選ぶ |
| 解説を見る | 回答後に解説パネルが開く |
| 次の問題へ | 解説を閉じると次の問題が表示される |
| 結果を確認 | 10問終了後にスコアが表示される |
| もう一度 | 結果画面から同じ単元を再挑戦できる |

## 📖 解説ページの活用

各コンテンツには解説ページがあります。問題の前後に解説を読むと理解が深まります。

- コンテンツ一覧は[コンテンツ一覧](../content-list/)を参照してください
- 解説ページは問題中の「解説を見る」からも開けます
