# スタートアップガイド

学習アプリの使い始め方と基本的な操作を説明します。

## 🚀 使い始め方

### ステップ1：名前を設定する

ヘッダーのユーザー名ボタン（👤）をクリックして、自分の名前を入力します。

- 名前を入力して ✓ ボタンを押すと保存されます
- 名前は学習記録に表示されます

### ステップ2：おすすめタブで今日の学習を始める

2回目以降は起動時に「🎯 おすすめ」タブが自動で表示されます。
初回は「🚀 ガイド」タブが表示されるので、タブをクリックして「🎯 おすすめ」を選んでください。

「🎯 おすすめ」タブには、今日学習すべきおすすめ単元が自動で表示されます：

- 画面上部に **学習状況の星（☆/⭐）** が表示されます
  - 目標数と今日の学習数（例: 学習数 2/5）が星の上に表示されます
  - 学習するごとに星が埋まっていきます
- 星の下に **次の単元** が表示されます
  - 教科名と単元名が太字で表示されます
- 「開始する」ボタンをクリックすると、おすすめ単元の詳細を開きます
  - 未学習の単元: `📖 解説`
  - 学習済みの単元: `✅ 確認`
- 「開始する」の下に、`取り組み中の単元` と `終わった単元` が一覧表示されます
- 単元カードをクリックして特定の単元を選ぶこともできます
- 今日の単元にはステージが表示されます
  - `📝` 学習済
  - `📜` 復習済
  - `🎓` 検定済

### ステップ3：問題にちょうせん

5問の4択問題が出題されます。選択肢をタップ（クリック）して回答しましょう。

- 答えを選ぶとすぐに正解・不正解がわかります
- 解説も表示されるので、間違えても大丈夫です
- 全問終わると結果が表示されます
- 「学習を終わる」を押すと次のおすすめ単元に進めます

## 🎯 基本的な操作

| 操作 | 説明 |
|------|------|
| 教科・単元を選択 | トップ画面から学習内容を選ぶ |
| 選択肢をタップ | 4択から正しいと思う答えを選ぶ |
| 解説を見る | 回答後に解説が表示される |
| 次の問題へ | 「次へ」ボタンで次の問題に進む |
| 結果を確認 | 全問終了後にスコアが表示される |
| もう一度 | 結果画面から同じ単元を再挑戦できる |
| 学習を終わる | 次のおすすめ単元に進む |

## 📚 （オプション）教科タブで単元を直接選ぶ

自分で学習する単元を指定したい場合は、「🌐 英語」「📐 数学」「📖 国語」などの教科タブを選んでください。教科ごとの単元一覧が表示されます。

- 学習したい単元を一覧から直接選べます
- 学習状況フィルター（すべて／未学習／学習中／学習済み）で絞り込めます
- 単元名の横にはステージ表示（`📝` / `📜` / `🎓`）が表示されます
- 各単元には **解説ページ** へのリンクがあります（解説ページはインターネット接続が必要です）
  - 問題の前後に解説を読むと理解が深まります
  - 単元詳細パネルの「📖 解説」タブから開けます
  - コンテンツ一覧は [ガイド（教科・単元）](../../#subject=support&supportMenu=contents) から一覧できます
