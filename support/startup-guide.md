# スタートアップガイド

学習アプリの使い始め方と基本的な操作を説明します。

## 🚀 使い始め方

### ステップ1：アプリを開く

[学習アプリ](../app/) にアクセスします。

![学習アプリ スクリーンショット](../images/quiz-app-screenshot.png)

### ステップ2：科目を選ぶ

アプリを開くと、学習する科目を選択できます。

- **英語** ─ 発音・文法・語彙
- **数学** ─ 算数から高校数学まで
- **国語** ─ 漢字・ことわざ・読解

### ステップ3：カテゴリを選ぶ

科目を選んだ後、学習するカテゴリ（単元）を選びます。

### ステップ4：クイズに挑戦

ランダムな10問の4択クイズが出題されます。選択肢をタップ（クリック）して回答しましょう。

## 🎯 基本的な操作

| 操作 | 説明 |
|------|------|
| 科目・カテゴリを選択 | トップ画面から学習内容を選ぶ |
| 選択肢をタップ | 4択から正しいと思う答えを選ぶ |
| 解説を見る | 回答後に解説パネルが開く |
| 次の問題へ | 解説を閉じると次の問題が表示される |
| 結果を確認 | 10問終了後にスコアが表示される |
| もう一度 | 結果画面から同じカテゴリを再挑戦できる |

## 📖 解説ページの活用

各コンテンツには解説ページがあります。クイズの前後に解説を読むと理解が深まります。

- コンテンツ一覧は[操作ガイド](operation-guide)を参照してください
- 解説ページはクイズ中の「解説を見る」からも開けます
