# 機能リファレンス

学習アプリの各画面と機能を説明します。

## ヘッダー

画面上部に常時表示されるバーです。

![ヘッダー](../images/01-header.png)

| パーツ | 説明 |
|--------|------|
| アプリ名（タイトル） | クリックするとスタート画面に戻る（クイズ中は確認メッセージを表示） |
| ユーザー名ボタン | クリックすると名前・アバターの設定画面を開く |
| アバター画像 | クリックすると名前・アバターの設定画面を開く |
| 今日の日付 | 現在の日付を表示する |
| 🔄（更新ボタン） | ページを再読み込みする（日付の右に表示） |
| 小/中/大（文字サイズ） | アプリ全体の文字サイズを切り替える（解説タブにも反映）。初回起動時のデフォルトは「中」とする |

### プロフィール設定画面

ヘッダーのユーザー名またはアバター画像をクリックすると表示されるポップアップです。以下の操作をまとめて行えます。

| パーツ | 説明 |
|--------|------|
| 表示名 | 現在のユーザー名を表示。クリックすると編集モードになり、名前を変更できる |
| プロフィール画像を変更する | クリックするとアバター画像のトリミング画面を開く |
| ユーザー切り替え一覧 | 登録済みユーザーの一覧を表示。「切り替え」ボタンで別ユーザーに切り替わる（ページ再読み込み） |
| ＋ 新しいユーザーを追加 | クリックするとユーザー名入力欄が表示される。名前を入力して「追加」ボタンで新しいユーザーを追加し切り替わる（ページ再読み込み） |
| ✕（閉じるボタン） | 設定画面を閉じる |
| 背景クリック / Escape | 設定画面を閉じる |

## スタート画面

アプリを開いたときに最初に表示される画面です。

### 教科タブ

![教科タブ（ガイド）](../images/02-start-guide-tab.png)

| タブ | 説明 |
|------|------|
| 🎯 おすすめ | 全教科のおすすめ単元と学習状況サマリを表示する（2回目以降の初期表示） |
| 🚀 ガイド | 使い方・教科・単元・サポート情報を表示する（初回表示） |
| 📈 進度 | 教科別の学習進度を分野別・学年別に確認できる |
| 🌐 英語 | 英語の単元一覧を表示する |
| 📐 数学 | 数学の単元一覧を表示する |
| 📖 国語 | 国語の単元一覧を表示する |
| ⚙️ 管理 | 学習データの読み込み・保存・初期化ができる |

### おすすめタブ

おすすめタブ選択時は、全教科横断のおすすめ単元が表示されます。

![おすすめタブ](../images/03-start-osusume-tab.png)

#### 学習状況パネル

| パーツ | 説明 |
|--------|------|
| 学習数 n/目標 | 今日の学習数と目標数を星の上に表示する |
| 星（☆/⭐） | 目標数に連動した星。学習済みは ⭐、未達成は ☆ |
| 次の単元 | 最初に取り組むおすすめ単元（教科名 › トップ › 親 › 単元名）を星の下に表示する |
| スタートボタン | おすすめ先頭単元を開く（未学習は `📖 解説`、学習済みは `✅ 確認`） |
| 取り組み中の単元 / 終わった単元 | スタートボタンの下に当日学習した単元を進行状態ごとに表示する（各リストのみスクロール） |

#### 今日の単元リスト

| パーツ | 説明 |
|--------|------|
| タイトル右の ℹ️ | クリックすると選び方の説明を表示する（未学習優先 / 学習済📝7日後・復習済📜14日後の復習対象 / 検定済🎓除外 / 国語→数学→英語優先） |
| 目標数コントロール | おすすめ表示の目標数（2/3/5/8/13）を切り替える。デフォルトは3とする |
| 単元カード | 教科絵文字・単元名・ステージ表示・進捗バーを表示。クリックで単元を選択する |
| 目標ライン | 「🎯 目標ここまで」で目標分との境界を表示する |
| 🚀 もっと追加 | クリックするとおすすめ単元を追加で表示する |

![結果画面](../images/10-result-screen.png)

| パーツ | 説明 |
|--------|------|

| メニュー | 説明 |
|----------|------|
| 📦 管理 | データの読み込み・保存・初期化を行う |
| 👁 データ参照 | 設定・履歴・学習済み問題・連続正解データを確認する |
| 📋 仕様 | アプリの仕様・データ構造の説明を表示する |

### 管理メニュー（📦 管理）

| タブ | 説明 |
|------|------|
| 📥 読み込み | 保存したJSONファイルを選択してデータを更新する |
| 📤 保存 | 学習データ（履歴・学習済み・連続正解）を個別または一括でJSONファイルとして保存する |

## URLフラグメントによる直接アクセス

URLのハッシュ（#以降の部分）を使って、特定の教科やメニューに直接アクセスできます。

| パラメータ | 説明 | 例 |
|-----------|------|-----|
| `subject` | 教科タブを切り替える | `#subject=english` |
| `supportMenu` | ガイドタブのメニューを指定する | `#subject=support&supportMenu=contents` |

- 不正な教科名が指定された場合は現在のタブが維持される
- リンクを共有して特定の教科やガイドページに直接アクセスできる

## 画面スケーリング

アプリは1920×1080を基準としたスケーリングを行い、さまざまな画面サイズに対応します。

| 条件 | 動作 |
|------|------|
| 横幅768px以上 | 画面サイズに応じたスケーリングを適用する |
| 横幅768px未満（スマートフォン） | スケーリングを無効にしてレスポンシブCSSで対応する |
| 高さが制約となるワイドスクリーン | コンテンツがビューポート内に収まるよう調整する |

## 仕様変更提案まとめ

小学3年生の視点でアプリを利用した際に気になった点を以下にまとめます。高校生も使うため、極端な変更は避け、全年齢で使いやすい改善を提案します。

### 1. 用語の平易化（マニュアル・UI両方）

| 現状の用語 | 提案 | 理由 |
|-----------|------|------|
| プロフィールダイアログ | プロフィール設定画面 | 「ダイアログ」は小学生に馴染みがない |
| インポート | 読み込み | 日本語で十分伝わる |
| エクスポート | 保存 | 日本語で十分伝わる |
| カテゴリ | 分野 | 「カテゴリ」は小学生に難しい |
| オーバーレイクリック | 背景クリック | 技術用語を避ける |
| フィルター | 絞り込み（UI上はそのまま可） | マニュアル内の説明で補足 |
| フラグメント | ハッシュ（#以降の部分） | 補足説明を追加 |
| ストローク | 書く | 平易な表現に |
| フォントサイズ | 文字サイズ | マニュアル上で統一 |
| ローカルストレージ | ブラウザ | 詳細な技術用語を省略 |
| 中止ボタン | やめるボタン | 「中止」より「やめる」が直感的 |
| クリアボタン | 全部消すボタン | 「クリア」は小学生に馴染みが薄い |

### 2. デフォルト値の変更提案

| 項目 | 現状 | 提案 | 理由 |
|------|------|------|------|
| 文字サイズの初期値 | 小 | 中 | 小学生には「小」は読みにくい。高校生でも「中」は問題ない |
| おすすめ目標数のデフォルト | 5 | 3 | 小学生の集中力を考慮。高校生は自分で増やせる |
| クイズのデフォルト問題数 | 10問 | 5問 | 小学生には10問は長い。高校生は自分で増やせる |

### 3. 操作フローの改善提案

| 項目 | 問題点 | 提案 |
|------|--------|------|
| 結果画面のボタン優先度 | 「スタート画面に戻る」がopacity-50で目立たず、「もう一度」が最も目立つ | 「学習を終わる」を最も目立つ色にする（次のおすすめに進む自然な流れを誘導） |
| 消しゴムモードの解除 | 消しゴムモードにした後の戻し方がわかりにくい | 消しゴムボタン再押下でペンに戻ることをUI上で示す（トグル表示） |
| データ初期化の誤操作防止 | 1回の確認ダイアログのみ | 2段階確認にする（1回目で「本当に消す」ボタンに変化） |
| 確認ダイアログの文言 | 「OK」「キャンセル」のみ | 何をOKするのかを明確にする（例：「やめる」「続ける」） |

### 4. スタートアップガイドの改善

| 項目 | 問題点 | 提案 |
|------|--------|------|
| ステップ3の説明 | 「ランダムな10問」と固定的 | デフォルト5問と明記し、問題を解いた後の流れも説明 |
| 基本操作表 | 「解説パネルが開く」等の表現 | 「解説が表示される」とシンプルに |
| 「学習を終わる」の説明なし | 結果画面の重要ボタンだが説明がない | 基本操作表に追加 |

### 5. トラブルシューティングの追加項目

| 追加FAQ | 理由 |
|---------|------|
| Q. 文字が小さくて読みにくい | 文字サイズ変更の方法を知らない利用者向け |
| Q. クイズ中に間違えて「やめる」を押した | 小学生がよく起こす操作ミスへの対応 |
| Q. メモ帳の消しゴムモードから戻れない | 消しゴムの戻し方がわからない問い合わせ想定 |
