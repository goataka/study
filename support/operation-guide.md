# 機能リファレンス

学習アプリの各画面と機能を説明します。

## ヘッダー

画面上部に常時表示されるバーです。

![ヘッダー](../images/01-header.png)

| パーツ | 説明 |
|--------|------|
| アプリ名（タイトル） | クリックするとハッシュなしのトップURLに戻る（クイズ中は確認ダイアログを表示） |
| ユーザー名ボタン | クリックするとプロフィールダイアログを開く |
| アバター画像 | クリックするとプロフィールダイアログを開く |
| 今日の日付 | 現在の日付を表示する |
| 🔄（更新ボタン） | ページを再読み込みする（日付の右に表示） |
| 小/中/大（文字サイズ） | アプリ全体の文字サイズを切り替える（解説タブにも反映）。初回起動時のデフォルトは「中」とする |

### プロフィール設定画面

ヘッダーのユーザー名またはアバター画像をクリックすると表示されるポップアップです。以下の操作をまとめて行えます。

| パーツ | 説明 |
|--------|------|
| 表示名 | 現在のユーザー名を表示。クリックすると編集モードになり、名前を変更できる |
| プロフィール画像を変更する | クリックするとアバター画像のトリミング画面を開く |
| ユーザー切り替え一覧 | 登録済みユーザーの一覧を表示。「切り替え」ボタンで別ユーザーに切り替わる（ページ再読み込み） |
| ＋ 新しいユーザーを追加 | クリックするとユーザー名入力欄が表示される。名前を入力して「追加」ボタンで新しいユーザーを追加し切り替わる（ページ再読み込み） |
| ✕（閉じるボタン） | 設定画面を閉じる |
| 背景クリック / Escape | 設定画面を閉じる |

## スタート画面

アプリを開いたときに最初に表示される画面です。

### 教科タブ

![教科タブ（ガイド）](../images/02-start-guide-tab.png)

| タブ | 説明 |
|------|------|
| 🎯 おすすめ | 全教科のおすすめ単元と学習状況サマリを表示する（2回目以降の初期表示） |
| 🚀 ガイド | 使い方・教科・単元・サポート情報を表示する（初回表示） |
| 📈 進度 | 教科別の学習進度を分野別・学年別に確認できる |
| 🌐 英語 | 英語の単元一覧を表示する |
| 📐 数学 | 数学の単元一覧を表示する |
| 📖 国語 | 国語の単元一覧を表示する |
| ⚙️ 管理 | 学習データの読み込み・保存・初期化ができる |

### おすすめタブ

おすすめタブ選択時は、全教科横断のおすすめ単元が表示されます。

![おすすめタブ](../images/03-start-osusume-tab.png)

#### 学習状況パネル

| パーツ | 説明 |
|--------|------|
| 学習数 n/目標 | 今日の学習数と目標数を星の上に表示する |
| 星（☆/⭐） | 目標数に連動した星。学習済みは ⭐、未達成は ☆ |
| 次の単元 | 最初に取り組むおすすめ単元（教科名 › トップ › 親 › 単元名）を星の下に表示する |
| スタートボタン | おすすめ先頭単元を開く（未学習は `📖 解説`、学習済みは `✅ 確認`） |
| 取り組み中の単元 / 終わった単元 | スタートボタンの下に当日学習した単元を進行状態ごとに表示する（各リストのみスクロール） |

#### 今日の単元リスト

| パーツ | 説明 |
|--------|------|
| タイトル右の ℹ️ | クリックすると選び方の説明を表示する（未学習優先 / 学習済📝7日後・復習済📜14日後の復習対象 / 検定済🎓除外 / 国語→数学→英語優先） |
| 目標数コントロール | おすすめ表示の目標数（2/3/5/8/13）を切り替える。デフォルトは3とする |
| 単元カード | 教科絵文字・単元名・ステージ表示・進捗バーを表示。クリックで単元を選択する |
| 目標ライン | 「🎯 目標ここまで」で目標分との境界を表示する |
| 🚀 もっと追加 | クリックするとおすすめ単元を追加で表示する |

### 単元一覧（教科タブ）

教科タブを選択すると、左パネルに単元一覧が表示されます。

![単元一覧（英語）](../images/04-start-english-unit-list.png)

| パーツ | 説明 |
|--------|------|
| 学習状況フィルター | すべて／未学習／学習中／学習済み で単元を絞り込む |
| 分野別／学年別ボタン | 単元を分野順または学年順に並べ替える |
| 学習済みを隠すボタン | 学習済みの単元を非表示にする |
| 分野／学年の解説ボタン | 各グループ見出しの `📖 解説` から、その分野や学年に対応した解説を表示する |
| 単元アイテム | クリックすると単元を選択し、右パネルに詳細が表示される。単元名の横に `📝` 学習済 / `📜` 復習済 / `🎓` 検定済 を表示する |
| 進捗バー | 各単元の正解率を棒グラフで表示する |
| 学年バッジ | 対象学年を色付きバッジで表示する（赤＝小学、青＝中学、緑＝高校） |

### 単元詳細（右パネル）

単元を選択すると右パネルに詳細が表示されます。

#### タブ

| タブ | 説明 |
|------|------|
| 📖 解説 | 選択した単元の解説ページを表示する |
| ✅ 確認 | クイズの設定と開始ボタン |
| 📋 問題 | 単元の問題一覧を表示する |
| 📊 履歴 | 過去の回答履歴を表示する |

#### 解説タブ

選択した単元の解説ページをアプリ内に表示します。文字サイズは「小/中/大」ボタンと連動します。

![単元詳細 - 解説タブ](../images/05-unit-detail-guide-tab.png)

#### 確認タブ（クイズ設定）

![単元詳細 - 確認タブ](../images/05b-unit-detail-kakunin-tab.png)

| パーツ | 説明 |
|--------|------|
| スタートボタン | 設定した条件でクイズを開始する |
| 前提単元 / 着手状況 / 完了ステータス | パネル上部に、前提単元名・着手可否・完了状態を表示する |
| 問題数 | 5問 / 10問 / 20問 / 全て から選択する。デフォルトは5問とする |

| 学習済 | 学習済みの問題を含めるかどうか |
| ✅ 履修済にするボタン | 選択した単元を履修済みとしてマークする |

#### 問題タブ

![単元詳細 - 問題タブ](../images/05c-unit-detail-mondai-tab.png)

| パーツ | 説明 |
|--------|------|

| 問題一覧 | 問題文と正解・不正解状況を一覧表示する |

#### 履歴タブ

![単元詳細 - 履歴タブ](../images/05d-unit-detail-history-tab.png)

## 問題画面

クイズ開始後に表示される画面です。

![問題画面](../images/06-quiz-screen.png)

### 問題エリア（左側）

| パーツ | 説明 |
|--------|------|
| プログレスバー | 全問題中の現在の進捗を表示する |
| 問題番号 | 「1/10」のように現在の問題番号を表示する |
| 単元名バッジ | 現在の問題の単元名を表示する |
| ✕（やめるボタン） | クイズをやめてスタート画面に戻る（確認メッセージあり） |
| 問題文 | 出題される問題を表示する |
| 選択肢ボタン | 4択の回答候補。タップ（クリック）して回答する |
| 前へ／次へボタン | 問題を前後に移動する |
| 採点するボタン | 全問回答後に表示。クリックすると結果画面へ進む |

### 回答フィードバック

選択肢を選んだ後に表示されます。

![回答フィードバック](../images/06b-quiz-feedback.png)

| パーツ | 説明 |
|--------|------|
| ⭕ 正解 / ✕ 不正解 | 正誤を表示する |
| 解説テキスト | 正解の理由や補足説明を表示する |

### メモエリア（右側）

問題を解くときに使えるメモ帳です。

| パーツ | 説明 |
|--------|------|
| 🗑️（全部消すボタン） | メモキャンバスをすべて消去する |
| 消しゴムボタン | 消しゴムモードに切り替える（もう一度押すとペンに戻る） |
| 線の太さ（ふつう/太い/極太） | ペンの太さを選択する |
| 色（黒/青/赤/緑） | ペンの色を選択する |
| メモキャンバス | 自由に書き込めるキャンバス |

#### 手書き文字入力（text-input問題）

記述式問題では、メモエリアが手書き文字入力に切り替わります。

| パーツ | 説明 |
|--------|------|
| 手書きキャンバス | 1文字ずつ書いて文字を入力する |
| 候補ボタン | 書いた文字の認識候補を表示する。クリックすると解答欄に入力される |
| 🗑️（全部消すボタン） | キャンバスをすべて消去する |

## URLフラグメントによる直接アクセス

URLのハッシュ（#以降の部分）を使って、特定の教科やメニューに直接アクセスできます。

| パラメータ | 説明 | 例 |
|-----------|------|-----|
| `subject` | 教科タブを切り替える | `#subject=english` |
| `supportMenu` | ガイドタブのメニューを指定する | `#subject=support&supportMenu=contents` |

- 不正な教科名が指定された場合は現在のタブが維持される
- リンクを共有して特定の教科やガイドページに直接アクセスできる

## 画面スケーリング

アプリは1920×1080を基準としたスケーリングを行い、さまざまな画面サイズに対応します。

| 条件 | 動作 |
|------|------|
| 横幅768px以上 | 画面サイズに応じたスケーリングを適用する |
| 横幅768px未満（スマートフォン） | スケーリングを無効にしてレスポンシブCSSで対応する |
| 高さが制約となるワイドスクリーン | コンテンツがビューポート内に収まるよう調整する |
