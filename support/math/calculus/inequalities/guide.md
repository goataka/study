# ⚖️ 不等式

## ✏️ 不等式とは

- **>**（大きい）、**<**（小さい）、**≥**（以上）、**≤**（以下）を使った式
- 方程式と同じように移項して解く

## 📖 例題と解説

### 例① 2x + 3 > 7

```text
2x + 3 > 7
2x > 7 - 3   ← 移項
2x > 4
x > 2 ✅

数直線: ──────(2)======→
        2は含まない（>）
```

### 例② -2x + 4 > 0

```text
-2x + 4 > 0
-2x > -4
x < 2 ✅  ← ⚠️ 負の数でわると不等号が逆転！
```

## 📌 重要ルール ⚠️

- 負の数をかける・わるとき → **不等号の向きが逆** になる
- 正の数なら向きはそのまま

## 📝 例

| 問題 | 答え |
|------|------|
| 2x + 3 > 7 | x > 2 |
| 3x - 1 ≤ 8 | x ≤ 3 |
| -2x + 4 > 0 | x < 2 |
| 5x + 2 ≥ 3x + 10 | x ≥ 4 |
| 4(x - 1) < 2x + 6 | x < 5 |
| -3x + 9 ≤ 0 | x ≥ 3 |
| 1 < 2x + 3 ≤ 9 | -1 < x ≤ 3 |
| x/2 + 1 > 3 | x > 4 |

💡 **ポイント**: 方程式と同じ解き方。ただし負の数でかける・わるときは不等号の向きが逆転します
