# 📝 文章問題（小学生）

文章から数式を立て、条件を整理して計算できるようにします。

## 💡 解き方のポイント

- **何を求めるか確認する**: 問題文の最後の「？」に注目する
- **わかっていることを書き出す**: 数量と単位を整理する
- **式を立てる**: たし算・ひき算・かけ算・わり算のどれを使うか考える

## 例題

リンゴが3個あって2個もらいました。全部で何個？

```text
3 + 2 = 5個
```

## 主な問題のタイプ

| タイプ             | 使う演算 | 例               |
|--------------------|----------|------------------|
| 合わせると・全部で | たし算   | 3個 + 2個 = 5個  |
| 残りは・減ると     | ひき算   | 8個 - 3個 = 5個  |
| 1つ分を求める      | わり算   | 12個 ÷ 4人 = 3個 |
| まとめていくつ     | かけ算   | 4個 × 3袋 = 12個 |
