# ➕ 小数のたし算

## ✏️ 小数のたし算のやり方

- 小数点の位置を **たてにそろえる** 📏
- 桁が足りないときは **0を補う**
- 整数のたし算と同じように計算する

## 📖 例題と解説

### 例① 1.5 + 2.3 = ?

```text
  1.5
+ 2.3     ← 小数点をそろえる
-----
  3.8 ✅
```

### 例② 3.25 + 1.5 = ?

```text
  3.25
+ 1.50    ← 桁が足りないので 0 を補う
------
  4.75 ✅
```

## 📌 コツ

- 小数点の位置をそろえれば、あとは整数と同じ ✅
- くり上がりにも注意 → `0.7 + 0.6 = 1.3`

## 📝 例

| 問題 | 答え |
|------|------|
| 0.3 + 0.4 = | 0.7 |
| 1.5 + 2.3 = | 3.8 |
| 2.6 + 1.8 = | 4.4 |
| 3.25 + 1.5 = | 4.75 |
| 0.7 + 0.6 = | 1.3 |
| 4.8 + 3.5 = | 8.3 |
| 1.75 + 2.25 = | 4.00 |
| 6.3 + 0.9 = | 7.2 |

💡 **ポイント**: 小数点をそろえて計算します。桁が足りないときは0を補います。

`3.25 + 1.50 = 4.75`
