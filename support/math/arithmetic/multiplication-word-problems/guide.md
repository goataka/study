# ✖️ 文章問題

## ✏️ かけ算の文章問題とは

- 「1つ分 × いくつ分」のときかけ算を使う
- 「倍」「まとまり」などのキーワードが目印

## 📖 例題と解説

### 例① 1箱に6個入ったお菓子が4箱あります。全部で何個ですか？

```text
ステップ１: 1箱の数 → 6個
ステップ２: 箱の数 → 4箱
ステップ３: 6 × 4 = 24個 ✅
```

🎯 **答え: 24個**

## 📌 コツ

- 「1つ分 × いくつ分」→ かけ算
- 「〇倍」→ かけ算のサイン
- 単位（個・人・円・本）を忘れずに書く

## 📝 例

| 問題 | 答え |
|------|------|
| 1箱に6個入ったお菓子が4箱あります。全部で何個ですか？ | 24個 |
| 1列に8人が3列並んでいます。全部で何人ですか？ | 24人 |
| 1本12円のえんぴつを5本買います。代金はいくらですか？ | 60円 |
| 毎日15分ずつ勉強します。7日間では何分ですか？ | 105分 |
| 1袋に24個入ったキャンディが3袋あります。全部で何個ですか？ | 72個 |

💡 **ポイント**: 「1つ分の数」と「いくつ分」を見つけてかけ算します。
