# ➗ 3桁以上のわり算（あまりなし）

## ✏️ 3けた以上のわり算

- 百の位から順に割る
- 答えに0が入る場合がある

## 📖 例題と解説

### 例① 369 ÷ 3 = ?

```text
   123
  -----
3 ) 369
   3     ← 3 ÷ 3 = 1
   --
    6    ← 6を下ろす
    6    ← 6 ÷ 3 = 2
    --
     9   ← 9を下ろす
     9   ← 9 ÷ 3 = 3
     -
     0
→ 123
```

### 例② 525 ÷ 5 = ?（答えに0が入る）

```text
   105
  -----
5 ) 525
   5     ← 5 ÷ 5 = 1
   --
    2    ← 2を下ろす（2 < 5 なので答えは0）
    0    ← 0を立てる
    --
    25   ← 25を下ろす
    25   ← 25 ÷ 5 = 5
    --
     0
→ 105
```

## 📌 おぼえておこう

- 割れないときは答えに0を書いて次の位を下ろす
- 答えは3けたになることが多い

💡 **ポイント**: 百の位・十の位・一の位の順に割ります。商に0が入る場合（525÷5=105など）に注意しましょう。
