# ➕ 1桁のたし算（くり上がりなし）

## ✏️ たし算とは

- 数を **合わせる** 計算
- 記号「＋」を使う

## 📖 例題と解説

### 例① 3 + 4 = ?

```text
🍎🍎🍎 ＋ 🍎🍎🍎🍎 → 🍎🍎🍎🍎🍎🍎🍎
  3    ＋    4     ＝     7
```

3個と4個を合わせると **7個** 🎯

### 例② 5 + 1 = ?

```text
🔵🔵🔵🔵🔵 ＋ 🔵 → 🔵🔵🔵🔵🔵🔵
     5     ＋  1  ＝      6
```

5個にもう1個たすと **6個** 🎯

## 📌 おぼえておこう

- 順番を入れかえても答えは同じ → `2 + 3 = 3 + 2 = 5`
- 0をたしても数は変わらない → `5 + 0 = 5`
