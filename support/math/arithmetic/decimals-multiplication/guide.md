# ✖️ 小数のかけ算

## ✏️ 小数のかけ算のやり方

- 小数を **整数にして** 計算する
- 小数点以下の桁数を数えておく
- 最後に **桁数の合計だけ** 小数点を左にずらす

## 📖 例題と解説

### 例① 1.2 × 3 = ?

```text
ステップ１: 整数として計算 → 12 × 3 = 36
ステップ２: 小数点以下の桁数 → 1.2 は 1桁
ステップ３: 36 → 小数点を1桁左へ → 3.6 ✅
```

### 例② 0.8 × 0.5 = ?

```text
ステップ１: 整数として計算 → 8 × 5 = 40
ステップ２: 小数点以下の桁数 → 0.8(1桁) + 0.5(1桁) = 2桁
ステップ３: 40 → 小数点を2桁左へ → 0.40 ✅
```

## 📌 コツ

- まず整数で計算 🔢 → そのあと小数点を合わせる 🎯
- 桁数を数えまちがえないように注意 ⚠️

## 📝 例

| 問題 | 答え |
|------|------|
| 0.5 × 2 = | 1.0 |
| 1.2 × 3 = | 3.6 |
| 2.5 × 4 = | 10.0 |
| 0.8 × 0.5 = | 0.40 |
| 3.6 × 5 = | 18.0 |
| 1.5 × 1.5 = | 2.25 |
| 0.3 × 0.7 = | 0.21 |
| 4.2 × 2.5 = | 10.50 |

💡 **ポイント**: 整数として計算してから、小数点以下の桁数の合計だけ小数点を左にずらします。

`8 × 5 = 40` → 小数2桁分 → **0.40**
