# ➖ 小数のひき算

## ✏️ 小数のひき算のやり方

- 小数点の位置を **たてにそろえる** 📏
- 桁が足りないときは **0を補う**
- くり下がりに注意する

## 📖 例題と解説

### 例① 5.7 - 2.9 = ?

```text
  5.7
- 2.9     ← くり下がりに注意
-----
  2.8 ✅
```

- 7 - 9 はひけない → 十の位から1くり下げる
- 17 - 9 = 8、4 - 2 = 2 → **2.8**

### 例② 4.6 - 1.75 = ?

```text
  4.60    ← 桁が足りないので 0 を補う
- 1.75
------
  2.85 ✅
```

## 📌 コツ

- 小数点をそろえて、桁を合わせてから計算 📏
- くり下がりは整数のひき算と同じルール ✅

## 📝 例

| 問題 | 答え |
|------|------|
| 0.8 - 0.3 = | 0.5 |
| 3.5 - 1.2 = | 2.3 |
| 5.7 - 2.9 = | 2.8 |
| 4.6 - 1.75 = | 2.85 |
| 7.3 - 4.8 = | 2.5 |
| 9.0 - 3.6 = | 5.4 |
| 6.25 - 2.5 = | 3.75 |
| 8.1 - 0.45 = | 7.65 |

💡 **ポイント**: たし算と同じく小数点をそろえます。くり下がりに注意しましょう。

`4.60 - 1.75 = 2.85`
