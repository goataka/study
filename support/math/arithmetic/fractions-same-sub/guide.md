# ひき算（同分母）

## ✏️ 同じ分母のひき算

- 分母が同じなら、**分子だけをひく**
- 分母はそのまま変えない
- 約分できるときは約分する

## 📖 例題と解説

### 例① 3/4 - 1/4 = ?

```text
🍕🍕🍕  −  🍕  ＝  🍕🍕
 3/4      1/4      2/4 → 約分して 1/2
```

🎯 分子をひく → 3 - 1 = 2 → 2/4 → 約分 **1/2**

### 例② 7/8 - 3/8 = ?

- 分子をひく → 7 - 3 = 4 → 4/8
- 約分 → 4÷4 / 8÷4 = **1/2** ✅

## 📌 おぼえておこう

- たし算と同じく分母はさわらない
- 答えが0になることもある → 3/5 - 3/5 = **0**

## 📝 例

| 問題 | 答え |
|------|------|
| 3/4 - 1/4 = | 2/4 = 1/2 |
| 4/5 - 2/5 = | 2/5 |
| 5/6 - 1/6 = | 4/6 = 2/3 |
| 7/8 - 3/8 = | 4/8 = 1/2 |
| 4/7 - 2/7 = | 2/7 |
| 5/9 - 2/9 = | 3/9 = 1/3 |
| 6/10 - 4/10 = | 2/10 = 1/5 |
| 3/5 - 1/5 = | 2/5 |
| 8/9 - 5/9 = | 3/9 = 1/3 |
| 7/10 - 3/10 = | 4/10 = 2/5 |

💡 **ポイント**: 分母はそのまま、分子だけをひきます。約分も忘れずに
