# ➕ 文章問題

## ✏️ たし算の文章問題とは

- 「合わせると」「全部で」「増えると」などのキーワードでたし算を使う
- 問題をよく読んで、何を求めるかを確認する

## 📖 例題と解説

### 例① りんごが3個あります。4個もらいました。全部で何個ですか？

```text
ステップ１: 最初の数 → 3個
ステップ２: 増えた数 → 4個
ステップ３: 3 + 4 = 7個 ✅
```

🎯 **答え: 7個**

## 📌 コツ

- 「合わせて」「全部で」「増えた」→ たし算のサイン
- 単位（個・人・円・本）を忘れずに書く

## 📝 例

| 問題 | 答え |
|------|------|
| りんごが3個、みかんが4個あります。合わせて何個ですか？ | 7個 |
| バスに15人乗っています。次の停留所で8人乗りました。今何人いますか？ | 23人 |
| 本が24冊あります。12冊増えました。全部で何冊ですか？ | 36冊 |
| 赤いボールが17個、青いボールが25個あります。合わせて何個ですか？ | 42個 |
| お金が135円あります。48円もらいました。全部でいくらですか？ | 183円 |

💡 **ポイント**: 問題文の「何を求めるか」を確認してから計算しましょう。
