# ➗ 文章問題

## ✏️ わり算の文章問題とは

- 「同じ数ずつ分ける」「1つ分はいくつ」のときわり算を使う
- あまりが出る場合は問題の状況に合わせて解釈する

## 📖 例題と解説

### 例① 12個のりんごを4人で同じ数ずつ分けます。1人何個ですか？

```text
ステップ１: 全体の数 → 12個
ステップ２: 分ける人数 → 4人
ステップ３: 12 ÷ 4 = 3個 ✅
```

🎯 **答え: 3個**

## 📌 コツ

- 「等しく分ける」「1人分は」→ わり算
- あまりが出たら「何個余る」と書く
- 単位（個・人・円・本）を忘れずに書く

## 📝 例

| 問題 | 答え |
|------|------|
| 12個のりんごを4人で分けます。1人何個ですか？ | 3個 |
| 30本の花を6本ずつ花瓶に入れます。花瓶は何本必要ですか？ | 5本 |
| 25個のあめを3人で分けます。1人何個でいくつ余りますか？ | 8個、1個余り |
| 48枚の色紙を8人で同じ数ずつ分けます。1人何枚ですか？ | 6枚 |
| 100円で7円のシールを買います。何枚買えてお釣りはいくらですか？ | 14枚、2円余り |

💡 **ポイント**: 「何を1とするか」を確認してからわり算します。
