# ➖ 文章問題

## ✏️ ひき算の文章問題とは

- 「残りは」「減ると」「どちらが多い」などのキーワードでひき算を使う
- 問題をよく読んで、何を求めるかを確認する

## 📖 例題と解説

### 例① クッキーが12枚あります。5枚食べました。残りは何枚ですか？

```text
ステップ１: 最初の数 → 12枚
ステップ２: 減った数 → 5枚
ステップ３: 12 - 5 = 7枚 ✅
```

🎯 **答え: 7枚**

## 📌 コツ

- 「残り」「減った」「少ない」「差は」→ ひき算のサイン
- 単位（個・人・円・本）を忘れずに書く

## 📝 例

| 問題 | 答え |
|------|------|
| クッキーが12枚あります。5枚食べました。残りは何枚ですか？ | 7枚 |
| バスに23人乗っています。8人降りました。残りは何人ですか？ | 15人 |
| 本が48冊あります。17冊貸しました。残りは何冊ですか？ | 31冊 |
| 赤い花が35本、白い花が19本あります。どちらが何本多いですか？ | 赤い花が16本多い |
| 250円持っています。75円のお菓子を買いました。残りはいくらですか？ | 175円 |

💡 **ポイント**: 「元の数」から「減った数」を引きます。問題文をよく読みましょう。
